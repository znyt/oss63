　　　　慕清霄微微点头，道童给他指了个方向便转身离去，此处可是青云门禁地，他可不敢多做停留。
<br/>　　待道童离去后，慕清霄推开院门，走进四合院中，前脚刚踏入四合院中，便感觉到一道凌厉的目光锁定在了他身上。
<br/>　　秋风瑟瑟，院中黄叶遍地，抬头望去，只见院中央站着一名白衣如雪的中年男子，手中握着一柄扫帚，目光惊疑的打量着慕清霄。
<br/>　　慕清霄也打量着他，中年男子满头白发，容颜常驻，一眼看去，给人的感觉就是不俗，但手中的扫帚却让他与此景格格不入。
<br/>　　而且全身气息内敛，宛若普通一名普通人，看不出他的修为，但慕清霄却能从他身上感受到极强的剑意。
<br/>　　剧情中，祖先祠堂由万剑一把守，此人的身份不言明了。
<br/>　　“前辈，晚辈慕清霄，奉道玄真人之命，前来祖先祠堂面壁思过。”
<br/>　　慕清霄微微一笑，丝毫没有行礼的意思，喊一声前辈，是因为万剑一的年龄比他大。
<br/>　　而且，他也没必要行晚辈之礼，他能感受到万剑一身上极强的剑意，想必万剑一也是如此，在同等层次的对手面前，剑独有的气息，想藏都藏不住。
<br/>　　他收敛全身气息，凭永恒万花筒写轮眼制造错觉，能够瞒住道玄真人他们，却瞒不住身为强大剑修的万剑一。
<br/>　　“咻咻......!”
<br/>　　陡然，两道凌厉至极的剑气，以迅雷不及掩耳之势射来，慕清霄目光一凝，两道恐怖的剑气凝聚而成，向袭来的剑气迎面射去。
<br/>　　“铮...!”
<br/>　　两人的剑气相撞，同时在空中泯灭，无形的波动向四周扩散而出，掀起一片秋叶。
<br/>　　剑修！
<br/>　　万剑一剑眉紧皱在一起，他完全看不出慕清霄的深浅，也不敢在继续试探，于是出声问道：“好恐怖的剑意，阁下来青云门有何目的？”
<br/>　　根据刚刚的试探，万剑一心中凝重无比，面前这位神骏青年修为深不可测，剑道上的领悟更是远高于他，青云门恐怕无人能与之抗衡。
<br/>　　“前辈不必担忧，我来青云门只是想取一些东西，决无歹意，否则我也不会来祖先祠堂面壁思过，哪怕是道玄掌门的命令，还请前辈行个方便。”
<br/>　　万剑一察觉到慕清霄眉宇间的傲然，微不可查的点了点头，真是长江后浪推前浪，他能看出，慕清霄并非老怪物驻颜，年龄估计也高不到哪里去。
<br/>　　如此年轻就有一身恐怖的修为，当真是妖孽，而且慕清霄前脚刚踏进大院，他便知道来者不是敌人，否则也不会心平气和的交谈，之所以试探，也是出于好奇。
<br/>　　只要慕清霄不做出危害青云门的事情，他都不会询问，至于取什么东西，也不是他能够左右的。
<br/>　　“也罢，前面便是祖先祠堂，左边有一间空房，此处清静，阁下请自便。”
<br/>　　话语刚落，万剑一就不再握着扫帚清扫院落中的秋叶，慕清霄微微一笑，转身向左侧的空房走去，至于面壁思过，开什么国际玩笑？
<br/>　　青云门除了灵尊，就属他的修为最高，手握诛仙剑的道玄估计能跟他一较高下，可让他面壁思过还是免了吧，虽然他拜入青云，但对这里可没有什么归属感。
<br/>　　除了陆雪琪，青云门其余弟子，终究是他道路上的过客。
<br/>　　客房不大也不小，但其中一尘不染，空气中散发着一股淡淡的清香，想必是万剑一每天都在打扫。
<br/>　　目光在客房中环视一圈，随后就盘坐在床榻上，闭上双眼，心中思绪万千。
<br/>　　来到青云志位面近六年了，他的心性沉稳许多，炼化嗜魂魔棒，配上浓郁的天地灵气，修为也提升了不少。
<br/>　　仙侠位面的灵气浓郁程度，真不是低武位面能比的，若是低武位面，光靠灵气想突破一重，绝对是比登天还难。
<br/>　　让他最喜悦的是，诞生了‘领域’神通，往后战斗开启领域，绝对占据了主动权。
<br/>　　半个月后便是下山历练的日子，看来得加快速度了，先获取青云门的天书，在去万蝠古窟，鬼王宗也必须去一趟，还有天音寺，以及天帝宝库。
<br/>　　...
<br/>　　...
<br/>　　时间过的很快，转眼夜幕降临。
<br/>　　慕清霄睁开双眼，最终吐出一口浊气，退出修炼状态，嘴角不禁勾起一丝笑意，体内的阴阳灵气又浑厚了一分。
<br/>　　推开房门来到院落，皎洁的月光倾洒在身上，慕清霄身心一闪，顺着白天的路线原路返回，巧妙的避开路上的障眼法，几分钟后就来到后山入口。
<br/>　　不再犹豫，慕清霄眼中浮现三颗黑色勾玉，在血轮眼下，任何障眼法都起不了效果，直接踏上右侧的道路，神识开路，往深处走去。
<br/>　　幻月洞府是中青云门禁地，也是诛仙剑的存放处，洞府外有幻月结界，可如真实般再现闯入者的生平，迷惑闯入者的心智，心智不坚定者，便有去无回。
<br/>　　忽然，慕清霄感觉身体接触到了什么，宛如坠入湖面，进入另一个不同的世界，心神守一，想必自己已经进入幻月结界的范围了。
<br/>　　前脚刚踏进幻月结界，一股强大的精神力量袭来，慕清霄并没有因此而停留，血轮眼开启直接将精神力量吸收。
<br/>　　幻月结界中的精神力量确实能够迷惑闯入者的心智，但那仅限于弱者，对于元婴境界的修士来说，根本毫无作用，更何况慕清霄的精神力量，根本不是普通元婴修士能比拟的。
<br/>　　几分钟后，慕清霄就畅通无阻的来到目的地。
<br/>　　幻月洞府的洞口比常人高出一半的洞口，宽七尺左右，出现在一个平缓的山坡上，旁边都是绿色的藤蔓与荆棘，有几枝垂下了洞口。
<br/>　　洞门口有一石壁，平整的石壁之中，镶嵌着一块石板太极图案，这便是这个山洞之中唯一能与青云门有关系的事物了。
<br/>　　第五卷天书就在其中，慕清霄嘴角勾起一丝笑容，瞥了一眼身后，道：“前辈，何必躲躲藏藏？”
