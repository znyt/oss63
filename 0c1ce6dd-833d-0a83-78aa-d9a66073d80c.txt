<br/>    妖城之上，阳光照耀着整个大地，一个个妖王行走在妖城之上，天空之中，还有着一个个妖王飞行在空中，侦查燕山的动向。
<br/>    
<br/>        “报！燕山在原地驻扎，依旧是没有任何动向。”
<br/>    
<br/>        妖族大君身披白狼大麾，眸中雪白，如同没有眼瞳，一双眸子甚是瘆人。他走到妖城之上，看向百里之外的燕山驻军，正在这时，他忽然看到在妖城之前一条剑龙席卷腾空，而在剑龙的头顶，站着一男一女。
<br/>    
<br/>        若不是早就知道陈明的装扮，他还有可能将陈明和紫霞仙子认成仙门正道。
<br/>    
<br/>        毕竟陈明身着道袍，手握拂尘，看起来怎么都像是个玉树临风的道士。
<br/>    
<br/>        陈明来到城门之下，剑龙腾空，让他和城头平齐，陈明看向妖城之上的妖族大君，“那个白眼狼，对，不用左右看了，就你最显眼，那个啥，你们家大君不是要见我吗？我现在来了，你去叫他来见我吧。”
<br/>    
<br/>        妖族大君按在城头上的手，咔嚓一声将一块铁砖抓的粉碎，他的话语有些微怒，“我就是妖族大君。”
<br/>    
<br/>        陈明愣了愣。
<br/>    
<br/>        场面一度十分尴尬。
<br/>    
<br/>        陈明这才仔细看向妖族大君：
<br/>    
<br/>        危险指数：2050，危险评级：灾难。
<br/>    
<br/>        光环：，lv2，
<br/>    
<br/>        
<br/>    
<br/>        介绍：披风挂雪，唯我白狼。

<br/>        效果：狼族掌控。

<br/>        嗯，这个妖族大君，个人实力似乎并不是特别强的样子啊，不过也对，他要是拥有的光环是关于个人实力的，早把那几个大妖王扫平了。

<br/>        这么一想，陈明就明白了过来，看来有机会施行斩首战术啊。

<br/>        妖族大君看向陈明道：“陈明，你可知修行不易？”
<br/>    
<br/>        陈明道：“我自然是知道，你既然这么问了，想必也是知道的，所以我要劝你一句，苦海无边回头是岸。”
<br/>    
<br/>        妖族大君愣了愣，我是这么开头来劝说你的啊，怎么被你劝说了？
<br/>    
<br/>        妖族大君道：“我何须回头？陈明，我问你，你真的想要与我一战不成？”
<br/>    
<br/>        陈明道：“这妖族的乱世，总要有一个终结，这个终结，你做不到，那就由我来！”
<br/>    
<br/>        妖族大君根本就是有恃无恐，“陈明，你自崛起之间，根本没有多少岁月，想必是一位新晋大君，成为大君不容易，还是爱惜自己的命比较好。作为一位新晋大君，你不知道很多事，我是能够理解的，所以我决定告诉你，我背后到底有着什么.....”
<br/>    
<br/>        陈明直接打断了妖族大君问道：“你是说你投靠了鬼仙的事？”
<br/>    
<br/>        妖族大君再次愣了愣，问道：“你既然知道我的背后是一位仙，为何还要来寻死，我等求仙问道，便是为了变得更强，而今，鬼仙不辞辛苦，想要降临大陆，赐予我等大道，指引我等的仙路，你为何不愿助我一臂之力！”
<br/>    
<br/>        紫霞仙子的眼中已经是杀意凛然，她背后的古仙门，可是在封印鬼仙这件事上出过大力的，要是鬼仙真的降临大陆了，后果不堪设想，紫霞仙子怒道：“这妖族大君已经被鬼仙迷惑了心智了，宗师，就让我斩杀了他，以证天地正道！”
<br/>    
<br/>        陈明看妖族大君这般模样，知道劝说妖族大君是没有任何机会了，当然，实际上，陈明压根没有准备劝说妖族大君，只是这对白也太俗套了，目前遇到的大君，都是上来就是一句，你修行不易。
<br/>    
<br/>        不是，你们就不能换句话开始对白吗？

<br/>        陈明朝着妖族大君露出一个微笑，拱手道：“我，燕山之主，陈明，此时此刻此地，正式邀君一战，看看这妖族的天下，到底鹿死谁手！”

<br/>        紫霞仙子微笑道：“宗师不愧是我们仙门正道魁首之一，听闻妖族大君背后有仙人都不惧！”

<br/>        陈明懒得理这些热血的仙门正道思想，你陈仙师会告诉你，我当年干了一票大的，把鬼仙的分身给宰了，让他没有说完对白？

<br/>        即便是妖王，在这星辰之下，也是瞬间被炸成齑粉！
