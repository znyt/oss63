　　　　“老板，这是一位鼠类掌控修武者”
<br/>　　黑衣保镖顿时对着苏羽严峻的开口。
<br/>　　此话一出，苏羽一下子明白了，明白自己的工地为什么会出现鼠灾了。
<br/>　　原来工地上的鼠灾不是什么鼠精霍乱，而是人为的。
<br/>　　这也是觉醒修武者的可怕之处，觉醒能力，元素为最，其次就是生物掌控者。
<br/>　　涂小安饶有兴趣的看着特殊出场的黑袍男子，眼睛浑然一眯。
<br/>　　“叮...林凡，等级二十一，天才型觉醒者，觉醒后，拥有控制鼠类的掌控能力”
<br/>　　又是一位生物掌控者，比自己之前在白镇遇到了那位蚊虫掌控者要强的多。
<br/>　　“你是谁，我苏羽有什么地方得罪你了，你要在我的工地闹鼠灾，还杀死了几个工人”
<br/>　　苏羽一脸的愤怒，他没想到既然能是人为，这更过分。
<br/>　　林凡无视他，反而打量工地上的一层冻结的冰，看着里面栩栩如生的老鼠，一阵心疼。
<br/>　　然而，他看向涂小安，冷冷的说：“朋友，你可知道你这一手，杀了我多少只老鼠吗”
<br/>　　“我不杀这些老鼠，难道让你这些老鼠杀人吗”涂小安淡淡的道。
<br/>　　林凡冷冷的注视涂小安：“这件事情于你无关，希望你别插手”
<br/>　　“我是苏老板青睐解决麻烦的，现在麻烦显然还没解决”
<br/>　　“你...”
<br/>　　林凡黑袍下的脸阴郁起来，看着苏羽道：“姓苏的，有点手段呀，连元素掌控者都被你请来帮忙了，看来老天都帮你”
<br/>　　“你到底是谁，你这样做有什么目的”苏羽确认自己跟对方根本不认识，况且他是做生意的只求财，不可能得罪这种修武者。
<br/>　　“目的？”林凡黑袍下，流露一张俊逸的脸庞，意味深长的说：“目的那么明显，你还看不出来吗”
<br/>　　苏羽蹙眉，一下子领悟了：“你是要我的工地荒废，不想它完工是吧”
<br/>　　“你到还不算太笨，虽然我做的已经很明显了”
<br/>　　“那你为何要杀人，我的几个工人白死了”
<br/>　　林凡平淡的回：“工地不死几个人，你岂会知道害怕”
<br/>　　“是谁让你这样做的”
<br/>　　这很显然是商业纠纷了，是他的工地建成的豪华小区挡了某些人的财路，所以才使阴招，请来鼠类掌控者，让他的工地无法按期建成。
<br/>　　“你无需知道是谁让我这样做，知道我是拿人钱财与人消灾便可，所以你不怪我”
<br/>　　苏羽现在虽然有涂小安护着，但他也不想跟对方闹的不死不休：“你背后的人给你多少钱，我给你双倍，只要你离开我的工地”
<br/>　　“一单子的买卖，岂能顾左右而言他，你当我是什么人，三姓家奴？”苏羽的建议，非但没有让对方开心，反而有点惹怒了对方。
<br/>　　林凡忽然看了涂小安一眼，又对苏羽说：“我给你一个建议，你若是答应了，我立马就走”
<br/>　　“你先说”
<br/>　　“你这个工地停工一年，一年之后随便你，但在这一年，工地连一片瓦都不能加上去”
<br/>　　苏羽闻言，气愤的一口拒绝：“这不可能，我无法答应你”
<br/>　　这片区域是新建开发，附近大批的小区都会在近几年完成，而且会成为温城土豪们的新宠。
<br/>　　他这个工地小区是三十层以上的大楼盘，加上其他的娱乐建筑，起码要三年完工。
<br/>　　本来时间都在争分夺秒，要是再晚一年在建筑，那么黄花菜都凉了。
<br/>　　等他这小区完工，土豪们，都已经入住了其他开发商的公寓小区了。
<br/>　　隐隐间，苏羽大致知道这黑袍男子是谁派来的了，这片地带，有两块最好的地皮。
<br/>　　一块被他投标下来，另一块被温城有头有脸的杨家投标下来。
<br/>　　大家都要紧锣密鼓的赶工，就怕落在人后。
<br/>　　显然是杨家的人不想他的小区完工抢在前头，而耍的卑鄙手段。
<br/>　　“你既然不答应，那么就别怪我不客气了”林凡被一口拒绝，冷笑了起来。
<br/>　　苏羽气道：“你的老鼠大军都被消灭了，你还有什么手段”
<br/>　　“哈哈....”林凡像是听到了什么天大的笑话：“这个天底下的老鼠能全被人消灭吗”
<br/>　　鼠类掌控者不是开玩笑的，他若想，可以在召唤大量的老鼠过来。
<br/>　　“小安兄弟，你说这如何是好”
<br/>　　苏羽只能求助涂小安，要是没有他在，对方恐怕都拦的跟自己废话。
<br/>　　涂小安听明白了，这人不过是收了别人的钱，不让苏羽的工地完工。
<br/>　　他耸了耸肩膀，提醒：“苏哥，你不是请来了余大师吗，让余大师将这人打走不就行了”
<br/>　　余子童差点一个踉跄，对方可是鼠类觉醒者，着实不好对付的。
<br/>　　但是白家的声望不能败，余子童一步踏前，冲着林凡道：“年轻人，这件事情我们白家的人管了，我是白家的捉妖师，我劝你还是离开吧”
<br/>　　“白家？”林凡脸色一变，但是旋即淡定下来：“你是白家的捉妖师，但这里那里有妖？”
<br/>　　“你要觉得我的老鼠是妖，那尽管捉了去”
<br/>　　这满地被冰封的老鼠，你随便抓，林凡压根不在乎了。
<br/>　　“你...”余子童被呛的说不出话。
<br/>　　涂小安提醒道：“余大师，你看到他肩膀上那只小白鼠没有，那可是觉醒精怪”
<br/>　　这黑袍男子不但脚踏群鼠出来，他的肩膀上更趴着一只与众不同的小白鼠，此鼠晶莹可爱，浑身毛发纯白胜雪，散发淡淡光泽，一看就不是凡鼠。
<br/>　　涂小安已经洞察过，这可是一只二阶的鼠精，别看那么迷你可爱，但千万不要被它的外表迷惑了。
<br/>　　“对，是妖，我察觉到了妖气”余子童惊喜了起来。
<br/>　　林凡笑了，神情自若道：“你想抓我身上的这只老鼠，可以，尽管来”
<br/>　　“黄金瞳”余子童自然不会亲自出手，一声呐喊。
<br/>　　猫抓老鼠，他的猫最好不过。
<br/>　　嗷....
<br/>　　工地声呼啸声冲天而起，黄金瞳化为一道金光，骤然朝着黑袍男子林凡爆射了过去，前肢上锋利的爪子在空中延长了一寸，见血封喉。
<br/>　　“哼...”林凡见到，冷哼了一声：“谁说猫便克老鼠，谁是谁的天敌，可没有绝对，实力才决定一切。”
<br/>　　“包子，那你去会会一只不断怪叫的讨厌猫吧”
<br/>　　此话一出，他肩膀上的小白鼠发出了一道戏谑的声音：“好呀，好呀，我也最讨厌这些该死的猫，见一只杀一只”
<br/>　　说罢，这只小白鼠朝着金光骤射出去。
<br/>　　砰...
<br/>　　空中猛然迸发一股爆炸性的力量，一猫一鼠最后落到了一片空旷的工地上，剑拔弩张了起来。
