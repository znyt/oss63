　　“一剑万重？这楚风眠居然也足以用的出一剑万重？”
<br/>　　“这楚风眠，怎么会如此厉害？他的一身力量肉身，都足以是跟圣王宗的人抗衡，现在他的剑道，居然还不在剑苍穹之下？”
<br/>　　“肉身媲美圣王宗，剑道足以跟天剑宗平起平坐，此人到底是什么妖孽？”
<br/>　　“这小子才不过二十岁的年纪，哪里有可能修炼如此多种武道！”
<br/>　　天羽城中的无数武者，一个个震惊连连，纷纷议论起来。
<br/>　　楚风眠不单单是实力强大，在剑道上，足以是足以跟剑苍穹平起平坐，这简直是一种骇人听闻的事情。
<br/>　　大部分武者，穷其一生，都无法将一种武道，达到大成。
<br/>　　像是圣王宗弟子，所修炼的便就是圣王宗的锻体之术，而天剑宗弟子所学的，就是天剑宗之中的剑道。
<br/>　　可眼前楚风眠的肉身足以媲美圣王宗，剑道甚至都不亚于剑苍穹，这种一心二用，究竟是如何做到的？
<br/>　　天才？妖孽？这种词都不能够形容现在的楚风眠。
<br/>　　“九域年轻一代，谁还是此人的对手？”
<br/>　　“此人的天资，若是让他真的成长下去，将来很有可能会位于九域巅峰！”
<br/>　　无数武者看向楚风眠，眼神中都是出现了杀机。
<br/>　　楚风眠的天资，实在是太过恐怖，以至于现在，没有任何一人还期待楚风眠崛起。
<br/>　　一旦是楚风眠强势崛起，他们一个个都要被踩在脚下，没有任何一人可以翻身。
<br/>　　在众人纷纷议论之际。
<br/>　　天空之上无数剑光，已经是经历过无数次的交锋，数万道剑光，在空中不断厮杀的，这是一种什么样的场景？
<br/>　　这一道道剑光厮杀，甚至是要将这天羽城的废墟，都给硬生生的打破，周围的地面都被掀了起来，一片混乱。
<br/>　　“在我的面前动剑，看来你是不想活了。”
<br/>　　楚风眠看着剑苍穹，嘴角淡然的露出了几分不屑笑容，一双眼睛中，似乎是在嘲笑。
<br/>　　在整个九域中，能够有资格对楚风眠动剑的人，都不过一手之数。
<br/>　　但是现在，这剑苍穹，显然并不在其中。
<br/>　　“破！”
<br/>　　楚风眠灵力运转，喊出这一个字。
<br/>　　在空中的万道剑锋，突然之间便是如同被唤醒了一般，威力猛然暴涨，足足是达到了之前的十倍的样子。
<br/>　　每一道剑气，上面的剑锋，都是锋利无比，一阵厮杀，这剑苍穹的剑气，被全部摧毁。
<br/>　　“轰隆！”
<br/>　　一声惊天的爆炸声，这剑苍穹的剑气，被全部剿灭。
<br/>　　恐怖的剑势，更是丝毫不停，直接冲着剑苍穹，斩杀而去。
<br/>　　“不好！退！”
<br/>　　剑苍穹脸色大变，他的内心一万个想不到，居然会是这样一个结局。
<br/>　　在同境界武者之中，他的剑道，向来都是无敌，所向睥睨。
<br/>　　谁能想到，居然真的会被人正面击破。
<br/>　　一尊剑修，剑道一破，他的实力几乎十不足三，面对这袭来的剑光，他的脸色已经是苍白无比，身躯连连后退。
<br/>　　可是他退后的速度再快，也远远无法逃出剑光的锁定，这漫天之上的万道剑气，顿时联合起来，一下子便是化为了一道足有万米之长的恐怖剑光。。
<br/>　　向着剑苍穹，轰然落下。
<br/>　　“天魔罗伞！”
<br/>　　凌空之中，一道漆黑无比的身影，突然现身，一眼望去才足以看清，在这漆黑无比的遁光之中，居然是一座马车。
<br/>　　这马车，庞大无比，背后所拉的，似乎是一座极为庞大的宫殿，而这拉车的，也并而是马，而是一种魔头，一种天魔。
<br/>　　天魔拉车。
<br/>　　在这马车背后的宫殿之上，便是一座巨大无比的紫色罗伞，异常闪耀，显现出了一种耀眼的紫光，直接化为了一道屏障，向着楚风眠的剑光，抵挡而去。
<br/>　　“天魔驾车！魔主降临！”
<br/>　　“这是九魔城的无上宝物之一，天魔宝车！”
<br/>　　一道道惊呼声，突然响起。
<br/>　　眼前这马车，可并非是寻常之物，乃是一件名震九域的至宝。
<br/>　　在这马车之中，似乎是坐着几个人，每个人都是一袭黑衣，魔意滔天。
<br/>　　“九魔城的人也来了，看来不用楚某一个个的去找了，如今一个个都已经是送上门来了。”
<br/>　　楚风眠看着眼前这马车，平静的开口道。
<br/>　　“宝车以出，小魔主该现身了吧。”
<br/>　　天魔宝车，可并非单单是一件威力强大无比的灵器，更是九魔城的象征，代表着九魔城的无上城主，魔主的身份。
<br/>　　不过在这归墟天中，魔主却是绝不可能出现，以魔主的身份，一旦出世，便是真正的惊天动地，不可能没有任何动静。
<br/>　　而除了魔主之位，也就只有九魔城的太子，小魔主，才有资格驾驶这宝车。
<br/>　　“哦？看来楚兄对我们九魔城，倒是知道不少啊。”
<br/>　　听到楚风眠的话，从那宝车中，一声轻笑，突然响起，只见在其中所坐的，正是一名年轻少年。
<br/>　　这少年看起来不过十五六岁的模样，脸庞稚嫩但是他的身边，却是魔气翻腾，他一个人坐在这宝车中的王座上。
<br/>　　在他旁边，还站有四名武者，在一旁恭敬的站着，众星捧月一般。
<br/>　　这年轻少年，便是这宝车如今的主人，九魔城的小魔主。
<br/>　　“小魔主！”
<br/>　　“九魔城的小魔主，居然也来了。”
<br/>　　“七大宗门之三，已经是全部到来了。”
<br/>　　在这天羽城中，如今便是越发的热闹了起来。
<br/>　　除了圣王宗，天剑宗的人，如今九魔城的人，也到来了。
<br/>　　看来这云羽天宫，如今倒是足以吸引人，七大宗门的弟子，也都纷纷来到了这天羽城中。
<br/>　　“小魔主，你来做什么！”
<br/>　　剑苍穹看到小魔主出现，脸色有些冰冷的开口道。
<br/>　　“我们天剑宗的事，不用你来管。”
<br/>　　“你们天剑宗的事，我九魔城，自然管不着。”
<br/>　　那十五六岁的小魔主，从这宝车中突然走出，脸上露出这几分自信微笑，令人根本无法看清，他的想法。
<br/>　　“不过这小子，之前嚣张，说是要杀了我们所有七大宗门弟子，我们九魔城，也是位列七大宗门之一，自然是要来跟这小子，讨个说法来。”
