　　海明江被惊到了，被苏莫的实力惊到了。
<br/>　　苏莫的实力，出乎了他的预料。
<br/>　　天殷城中。
<br/>　　因为苏莫的攻击，整个天殷城都被震动了。
<br/>　　“敌袭！”
<br/>　　“苏莫来了！”
<br/>　　“快快加持阵法！”
<br/>　　无数的殷家之人从住处飞了出来，嘶吼不断，人影闪烁。
<br/>　　整个天殷城足有数百万人口，尽是殷家族人，其中武皇境强者数百，武王境强者更是不计其数。
<br/>　　对于天殷城遭到攻击，殷家族人并没有多少意外，毕竟，他们殷家和黄泉魔宗这么大动作，苏莫不可能不反击。
<br/>　　所以，他们早已做好了应对攻击的准备。
<br/>　　几乎是瞬息之间，便有数千名殷家强者出手，加持阵法之力，让得防御大阵更加的稳固。
<br/>　　而这个时候，苏莫依旧在继续攻击，一道道剑气不要命的向阵法上斩击。
<br/>　　苏莫没有留手，体内五枚神丹催动到了极致，玄力澎湃，气势滔天。
<br/>　　浩大的三色剑气，连天接地，不断的从天空上倾泄而下，铺天盖地的斩向防御大阵。
<br/>　　爆响不断，巨响震天。
<br/>　　崩碎的剑气，如同天地浪涛，横扫八方。
<br/>　　苏莫的攻击强大到了极致，但这大阵的防御力堪称无敌，轻易的便挡住了他的所有攻击。
<br/>　　而海明江也没有闲着，同样开始出手攻击防御大阵。
<br/>　　“碎！”
<br/>　　海明江的手中出现了一杆长矛，长矛锋锐无比，蕴含着惊人的杀伐之气。
<br/>　　咻！咻！咻！
<br/>　　海明江手持长矛，连连刺出，长矛如蛟龙出海，洞穿天地，裹挟无边锋芒，向阵法不断的攻击。
<br/>　　轰隆隆！！
<br/>　　爆响连连，巨大的防御阵法，在苏莫和海明江的双重攻击之中，不断的抖动，剧烈的震颤。
<br/>　　不过，防御阵法虽然剧烈的抖动，但却是丝毫没有破损的迹象，显然还远没有到承受的极限。
<br/>　　任苏莫和海明江攻击如何狂暴，威力如何强大，大阵全部轻松挡住了。
<br/>　　少倾之后，苏莫不再远攻，而是改为了近攻。
<br/>　　他直接落在了防御阵法之上，双手握剑，全身金光闪耀，修为和肉身的力量催动到了极致。
<br/>　　轰！轰！轰！
<br/>　　苏莫疯狂的斩击防御大阵，每一剑都惊天动地，每一剑都势大力沉。
<br/>　　然而，防御大阵虽然抖动的越发强烈，却还是完好无损，没有一丝一毫的破裂。
<br/>　　相反，巨大的反震之力，却是让得苏莫双臂一阵酸麻，全身气血上涌。
<br/>　　就这样，苏莫和海明江不停的攻击，两人基本上都动用了全力，但一连攻击了小半个时辰的时间，还是没有破开防御大阵。
<br/>　　巨大的反震之力，几乎都将苏莫的双臂震麻木了！
<br/>　　“哈哈哈！！”
<br/>　　天殷城内响起一阵大笑之声，一名浓眉大眼的高大老者仰天狂笑，满脸讥讽的说道：“海明江、苏莫，你们想攻破我殷家的防御大阵，简直是痴心妄想！”
<br/>　　“殷林德，你休要嚣张，老朽若是破开了阵法，就是你的死期！”海明江怒喝道。
<br/>　　这名浓眉大眼的老者，名为殷林德，乃是殷家的大长老，武皇境九重巅峰修为。
<br/>　　“破开阵法？”
<br/>　　殷林德闻言面带不屑之色，道：“家主早有防备，防御大阵经过了重新布置，威力大增，现在又有我们这么多人加持阵法之力，就凭你们两人，还想要破开阵法？”
<br/>　　殷林德有绝对的自信，苏莫和海明江两人，绝对不可能破开大阵。
<br/>　　苏莫和海明江闻言，又攻击了几招之后，均是暂时停止了攻击，两人的脸色均是沉了下来。
<br/>　　他们攻击了小半个时辰，这阵法的防御力，他们心中已经非常清楚了。
<br/>　　以他们的实力，的确是不可能破开。
<br/>　　这殷家真是做足了准备啊！
<br/>　　“苏宫主，我们怎么办？”海明江阴沉着脸向苏莫问道，其实他已经有了离去之心。
<br/>　　此阵之强，出乎了他的预料，破不开阵法，耗在此地也是白费力气。
<br/>　　苏莫闻言沉默不语，就此离去他也有些不甘心。
<br/>　　若是能破开大阵，整个殷家的人和财富，都是将是他的修炼资源。
<br/>　　苏莫心中在苦思破阵之策，少倾之后，他眼眸一亮，心中有了一个非常可行的方法。
<br/>　　“海阁主，你退后！”苏莫对海明江道。
<br/>　　“苏宫主，你这是？”海明江不明所以，面带疑惑之色。
<br/>　　“我有办法破开此阵！”苏莫微眯着双眼，沉声说道。
<br/>　　“什么办法？”海明江闻言惊讶，满脸好奇的问道。
<br/>　　他们攻击了这么久都无果，苏莫能有什么办法破阵？
<br/>　　“等会海阁主自然会知晓！”苏莫嘴角勾起了一丝笑容，这个方法绝对可行，破开阵法应该问题不大。
<br/>　　海明江心中疑惑不已，不知道苏莫要如何破阵，但他还是稍稍后退了一段距离。
<br/>　　“苏莫，就凭你一人居然还想破阵？”
<br/>　　天殷城内的殷林德，自然也听到了苏莫的话语，顿时满脸讥讽之色。
<br/>　　土灵阵强大无比，没有什么破绽，殷林德可不相信苏莫能破开。
<br/>　　“苏莫，你在痴人说梦吗？”
<br/>　　“哈哈哈！简直是自不量力啊！”
<br/>　　“苏莫，我们承认你是个妖孽，但以你的实力想破阵，还差的远呢！赶紧滚回去吧！”
<br/>　　“……”
<br/>　　不少殷家的强者纷纷开口，均是目露嘲讽，满脸鄙夷之色。
<br/>　　没有人相信苏莫能破开土灵阵，一众殷家的强者，都很清楚土灵阵的强大。
<br/>　　就算是有三名准帝强者，一起攻击大阵，土灵阵都能坚持十天以上的时间。
<br/>　　而现在，苏莫居然说他自己就能破阵，这怎么可能？
<br/>　　就算苏莫是顶级阵法大师，都没有这个可能！
<br/>　　众人根本不信苏莫的话，俱是在准备看笑话！
<br/>　　“希望等会你们还能笑得出来！”苏莫冷冷一笑，随即身形缓缓上升，飞到了防御大阵之上百丈高处。
<br/>　　“哼！老夫倒要看看，你一个黄毛小儿如何破阵！”殷林德冷笑这说道。
<br/>　　苏莫没有理会殷林德，他伫立在防御阵法上空，眸光如利剑，凌厉万分。
<br/>　　“成败在此一举了！”苏莫深吸口气，体内玄力急速涌动，帝皇剑再次的高举了起来。
