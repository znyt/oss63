　　林峰看了眼一咬牙还是冲了过去，没办法，这C级空投太调皮了就掉在这里面。
<br/>　　林峰拿出斩灵刀就是冲了进去。
<br/>　　吼吼。
<br/>　　顿时。
<br/>　　几百头丧尸全部怒吼而来。
<br/>　　林峰放出千年僵尸然后又是几刀斩杀而去。
<br/>　　千年僵尸如今已经是七阶的存在，大手一挥，就好像是镰刀一样，足足十几个丧尸被割断。
<br/>　　狂龙出海！
<br/>　　青莲剑歌！
<br/>　　林峰数刀斩杀而去几十个丧尸也笼罩在犀利的刀芒之下，全部毙命。
<br/>　　吼吼吼
<br/>　　然而这其中不乏一些三阶四阶甚至五阶的丧尸犹如潮水一样涌来，一眼居然还望不到头。
<br/>　　林峰当机立断，知道现在不是去恋战的时候，一切以空投为重中之重，立即跳到千年僵尸背上趴着。
<br/>　　“跑！”
<br/>　　千年僵尸一跳，直接跳出个十几米高，几次跳跃就直接从丧尸群里飞出来了，然后稳稳落在一颗三十多米的大叔树干上。
<br/>　　几百只丧尸聚集在树下，抬起脑袋，张着嘴巴，双手在那里抓空气。
<br/>　　一些三阶四阶五阶丧尸都要爬上树，但他们爬个七八米的高度就要掉下去，根本怕不上来。
<br/>　　林峰瞧了眼远处，还有几千只丧尸在漫无目的的闲逛。
<br/>　　“恐怖！这个地方的确是墓地的存在了。”
<br/>　　林峰若是真的想要消灭这群丧尸自然还是可以做到的。
<br/>　　只是这个地方太危险，林峰也不确保里面有什么，所以尽量保存最大战斗力。
<br/>　　进入巨树之森，这个地方，已经是遮天蔽日了，阳光都照射不进来，时不时耳畔响起一些古怪的鸟叫声。
<br/>　　刷。
<br/>　　一只五阶的变异猴子忽然出现在林峰面前，龇牙咧嘴的就是在那里要攻击。
<br/>　　刷。
<br/>　　但下一刻。
<br/>　　一个巨大的白色变异鸟一口把他给叼走了，那五阶变异猴子根本没几乎反抗就被吞了下去。
<br/>　　“八阶的变异鸟？”
<br/>　　林峰震惊了，看着远处栖息在树枝上的巨大变异鸟林峰立即让千年僵尸背着自己离开。
<br/>　　因为出色的弹跳力，千年僵尸不只是战斗伙伴，还是坐骑，当然了，趴在一个僵尸后背上是需要一定的勇气的。
<br/>　　实际上千年僵尸根本没丧尸恐怖，千年僵尸身上的皮肤已经风干了，基本看不出什么，而丧尸不一样，烂肉血淋淋的，非常恶心倒胃口。
<br/>　　所以林峰还是可以接受的。
<br/>　　朝C级空投靠近的过程之中。
<br/>　　林峰不断看到有怪物。
<br/>　　不过八阶的再也没有看到过，大部分还是五阶上下。
<br/>　　有的怪物在杀丧尸，有的丧尸也在啃怪物。甚至林峰还看到了犹如猴子一样，在树林之中自有跳跃的丧尸，这群赫然是变异丧尸了。
<br/>　　半个小时后。
<br/>　　林峰就看到了挂在树杈上的两个空投，好在这两个空投是在一起的，不需要额外多跑一次。
<br/>　　“你原地警戒。”
<br/>　　林峰拍了拍千年僵尸然后就去收起空投。
<br/>　　两个C级空投顺利收回物品栏，林峰吐了口气，一切还算是有惊无险，倒也是老天爷眷顾了，正准备走时。
<br/>　　忽然传来了人类的交谈声。
<br/>　　林峰可以确认自己没听错，这的确是人的，说的还是林峰听得懂的中文。
<br/>　　“这地方居然还有人？”
<br/>　　林峰第一时间就知道，这群人肯定不简单，搞不好就是战门甚至第一邪教的，他立即潜伏起来。
<br/>　　“啧啧，目前曙光基地内足足两万人了吧。人数比沙长基地的还要多。”
<br/>　　“若能够灭了这个曙光基地，那我们战门又可以多一具血魔吧。”
<br/>　　“不过这曙光基地不好对付，其中有个古家不知道坏了我们多少好事，不然曙光基地早就被我们灭了。”
<br/>　　“别管古家了，星陨阁之前可以和我们战门抗衡，如今虽然落败了，但是瘦死的骆驼比马大，而且最近，尸傀门和血尸堂的人也在找我们麻烦，都已经死了好几个外门成员。”
<br/>　　“这群家伙还真是找死啊，莫名其妙的，怎么就对我们出手了，不过这两个组织都是第一邪教的下属实力，联手起来也不好对付，还真是麻烦。”
<br/>　　“不说他们了，这群家伙估计是以为有第一邪教罩着，寻思着来挑战我们的地位，到时候有钟师兄去对付他们，倒是曙光基地的那一部分先遣军，他们似乎发现了白骨山那边的秘密，最近有一大批战斗部队去那边集结了。”
<br/>　　“嘿嘿，既然如此，那就让他们有来无回。”
<br/>　　旋即就是几道猥琐的冷笑。
<br/>　　短短几句话，林峰却听得心中一沉。
<br/>　　“这群家伙居然是战门的人，而且听口气，似乎沙场基地被灭了，就是他们在背后推波助澜的，而且，似乎他们要对基地内派出去的战斗部队有想法。”
<br/>　　林峰暗道不好，自从朱民侦查到白骨山有怪物聚集，基地就高度重视，派出去了一股大部队去歼灭，若这群大部队全军覆没了，那曙光基地可以说元气大伤。
<br/>　　“握草，这群王八犊子，还是人吗？真的要毁灭全人类？？”
<br/>　　若是之前林峰还对古月的话将信将疑，但是现在他是百分百全部相信了。
<br/>　　因为事实就在眼前。
<br/>　　这群战门的人为达目的简直不择手段。
<br/>　　“必须要阻止他们！不然老子刚买的房子岂不是泡汤了！！”
<br/>　　林峰忽然觉得自己好沉重的使命，眼下他知道了这群家伙别有用心，怎么可能还会漠视不理，若是真的被这群战门之人得逞了，那他林峰等人也许要跟着这个曙光基地还有两万多的老百姓一起灰飞烟灭了。
<br/>　　没了基地，天下之大，何处是家。
<br/>　　等战门之人离去后，林峰才是立即离去。
<br/>　　第一时间他就是回到了基地找到了朱民。
<br/>　　朱民本来在忙军务，看到是林峰打来的电话他第一时间接了。
<br/>　　“朱民，先遣部队的路线图你知不知道，还有他们现在已经到了什么地方了。”
<br/>　　“你问这个干嘛、”朱民还是有些警惕的，毕竟军部内可不是上下一条心有些奸细，所以这种敏感问题他会本能留一个底线。
<br/>　　“我刚得到消息，有人要对先遣部队下手！而且这伙人准备许久了！我猜测，也许白骨山那边的丧尸和怪物忽然聚集起来，也是他们干的！这群人能耐非常大！”
<br/>　　林峰道。
<br/>　　“什么？林峰你没胡说八道吧？这件事情可不能开玩笑啊。”
<br/>　　朱民认真道。
<br/>　　“战门你知不知道！”
<br/>　　林峰道。
<br/>　　“战门？黑社会吗？？”
<br/>　　朱民一句话是把林峰给雷的不轻，黑社会？这也太瞧得起黑社会了吧。
