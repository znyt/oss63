　　这次的战斗，给萧逸最大的收获，莫过于武魂的进阶。
<br/>　　进阶黄色武魂后，控火兽的控火能力更为强悍了。最重要的是，操控的火焰温度飙升了一大截。
<br/>　　武魂增强后，萧逸的整体实力提升了一大层次。
<br/>　　噬火百刃发出的火刃，温度和爆炸力变得更强。
<br/>　　形意五绝与武魂力量的结合，也将远超从前。火属性豹形的速度，虎形的力度，也上升了一个档次。
<br/>　　说到底，武魂才是武者最根本的东西。而无论是丹药或者武技，都只是外在的因素。
<br/>　　这也是为什么武魂被认为是决定武者未来的东西，更是因武魂而将武者区分出天才抑或庸才。
<br/>　　当然了，别人的武魂觉醒后一生都不会变化。萧逸却因为太阴太阳之眼而有了进化武魂的能力。
<br/>　　整个紫云城，最强大的武魂只是黄色。
<br/>　　像萧家萧若寒，玄火门萧若狂，慕容家慕容千军，乃至各大家族和势力的长老，都不过是黄色武魂罢了。
<br/>　　从武魂这一角度，萧逸已经站在了紫云城的顶尖行列。
<br/>　　嗖的一声，萧逸忽然从马上跳下。
<br/>　　脚上猛地迸发出一股火焰，火焰再生出一股爆炸气流，萧逸的身子猛然间前进了数十米。
<br/>　　嘭、嘭、嘭、、、
<br/>　　一阵阵气流爆响声从萧逸脚下响过。
<br/>　　而萧逸的身影，则在数个呼吸间，突进了百多米的距离。
<br/>　　“好快的速度。”萧逸吃了一惊，万万没想到黄阶武魂的力量给予了自己这么大的增幅。
<br/>　　其他人的武魂，大多给予武者多重的增强。速度、力量、防御、缠绕等等。
<br/>　　唯独控火兽，只有控火的能力，也被所有人誉为最垃圾的武魂。
<br/>　　但如今在萧逸看来，控火兽反倒是比其他武魂更适合自己。那单一，却无与伦比的控火能力，往往能给予自己更大的实力提升。
<br/>　　像如今将火焰凝聚在脚下，产生气爆从而提升速度，就是其他武魂所无法给予的。
<br/>　　再看那匹千里骏马，竟已被萧逸远远地甩在了后面。
<br/>　　“呵。”萧逸看了马儿一眼，笑道，“马儿，你自己赶去陨星山脉吧，我先走一步了。”
<br/>　　......
<br/>　　陨星山脉，是一个危险却又充满了无限精彩的地方。
<br/>　　珍贵的紫云铁矿脉，被三大家族把持。但这里还存在着无数小型矿脉，不受任何人掌控，故而吸引了大批商人前来谋取利润。
<br/>　　同时，山脉内妖兽无数，没有实力的商人，只能聘请武者保护。
<br/>　　这便衍生出了一个职业——猎妖师。
<br/>　　猎妖师，大多是没有任何势力背景，却又需要金钱去购买各种修炼资源的独行武者。
<br/>　　有的靠保护商人赚取钱财，有的则是通过猎杀妖兽，得到妖兽尸体和内丹卖出去来赚钱。
<br/>　　总之，这里集合了形形色色、各色各样不同身份的人物。
<br/>　　.....
<br/>　　萧家大型矿脉附近数里之外，一座小型矿脉中，此刻正发生着一场战斗。
<br/>　　看两方武者的华丽服饰，显然不是普通人，而是大家族中的子弟。
<br/>　　“慕容家的，不要太过分了，这里是萧家的矿脉，轮不到你们乱来。”
<br/>　　很显然，开战双方是萧家和慕容家的子弟。
<br/>　　而萧家子弟，正以一种被碾压的状态，勉强抵挡着。
<br/>　　“过分？”慕容家走出一子弟，冷声道，“再过分也比不上你们萧家的那个小废物，哼，竟对我们慕容家的大小姐做出那等为人所唾弃的不耻之事。”
<br/>　　“慕容轩，你不要强词夺理，那种谣言，任谁都知道不可能，根本就是你们慕容家在故作刁难。”
<br/>　　说话的，正是萧壮。赤眼红牛武魂出现，怒视着慕容轩。
<br/>　　可惜，萧壮不过是凡境九重武者，而慕容轩乃是后天一重，他的怒视，起不了任何作用。
<br/>　　当初那批凡境六重的子弟，从紫云洞出来后已是凡境八重，接近九重。来到陨星山脉历练一番后，十人中，有七八人都突破了凡境九重。
<br/>　　“萧家的杂碎，识相地便自己滚，否则我不介意废了你们。”慕容轩冷笑道。
<br/>　　慕容轩，是慕容家有名的天才之一。在慕容家的身份，与萧若寒在萧家差不多。
<br/>　　萧家子弟顿时个个都露出了极为难看的脸色。
<br/>　　萧壮身旁的萧子木低声道，“萧壮，好汉不吃眼前亏，要不我们还是先退吧。”
<br/>　　“不行。”萧壮立刻道，“慕容家借着那莫须有的谣言，已经强行抢夺了我们七八座小型矿脉，不能再让他们得逞了。”
<br/>　　其他萧家子弟也道，“不错，再者矿脉事小，我们萧家的名声事大，岂可让慕容家这群小人骑在我们头上作威作福。”
<br/>　　“誓死捍卫萧家的尊严。”萧家子弟们顿时露出了强大的战意，摆明了死战不退。
<br/>　　“哼，一群蠢货。”慕容轩冷笑道，“既然你们自己不走，那便别怪我打断你们的腿。”
<br/>　　“我们上。”
<br/>　　慕容轩一声令下，一个个慕容家子弟如狼似虎般发起了攻击。
<br/>　　“跟他们拼了。”萧壮一马当先。
<br/>　　两方武者再次交战，只可惜，不出数分钟，在慕容轩的带领下，萧家子弟毫无还手之力，完全就是在被打。
<br/>　　“要怪，就怪你们萧家那个小废物吧。”
<br/>　　慕容家的子弟个个冷笑着，下手也越发狠辣。
<br/>　　“怪萧逸那个小废物也没用，他现在肯定还躲在萧家瑟瑟发抖吧。”
<br/>　　“谁让他得罪了慕容千军大哥呢，他敢走出萧家，就等死吧。”
<br/>　　慕容轩得意地看了眼同族之人，道，“那个小废物，哪里需要千军大哥去收拾。他敢出现，我第一个废了他。
<br/>　　恰在此时，远处一阵阵气爆声传来。
<br/>　　嘭、嘭、嘭、、
<br/>　　所有人闻声看去，只见一团火红色的身影，将空气炸开一阵阵巨响，快速奔袭而来。
<br/>　　人影转瞬即至，而后来到慕容轩身旁，一拳轰出。
<br/>　　轰的一声，慕容轩直接被吐血击飞。
<br/>　　“什么人？”慕容家子弟顿时大惊。
<br/>　　嘭，人影身上的火焰散开，露出面貌，正是萧逸。
<br/>　　“呵。”萧逸冷笑道，“还能是什么人，自然是你们口中要收拾的那个萧家小废物。”
<br/>　　“萧逸....萧逸....”
<br/>　　萧壮等人见来人是萧逸，且一拳轰飞了慕容轩，顿时大喜。
<br/>　　慕容轩站起身来，擦了擦嘴角的鲜血，冷冷地看着萧逸，“小废物，你竟然真的敢出现？”
<br/>　　嗖，萧逸瞬间在原地消失。
<br/>　　当他再次出现时，已在慕容轩身前。
<br/>　　“轰。”萧逸一拳打出。
<br/>　　慕容轩再度吐出一口腥血。
<br/>　　“好强，怎么可能？”慕容轩大吃一惊。
<br/>　　还未等他反应过来，萧逸一把扣住了他的脖子。
<br/>　　“额...”慕容轩顿时呼吸困难，他发现，自己后天一重的实力，竟完全挣脱不开萧逸的虎爪。
<br/>　　萧逸玩味地说道，“我刚才好像听到，有人说要废了我。“
<br/>　　这时，萧逸看了眼其他萧家子弟，发现他们个个带伤，严重的，甚至满身伤痕，要不是有其他子弟搀扶着，肯定已经倒下了。
<br/>　　萧逸冷眼看向慕容家子弟，暗道，“这群混蛋，完全是在往死里打，想杀了萧壮他们。”
<br/>　　想到这，萧逸露出了浓郁的杀意。
<br/>　　“快救慕容轩，杀了那小废物。”慕容家子弟此刻反应了过来，仗着人多，打算对付萧逸。
<br/>　　“哼。”萧逸冷哼一声，右手虚握。
<br/>　　霎那间，无数团火焰凭空而现，如同一条条火蛇，缠绕上了所有慕容家子弟。
<br/>　　这不是武技，只是单纯的控制火焰产生的攻击。
<br/>　　这里的慕容家子弟，除了慕容轩外，其他都是凡境武者。对付他们，萧逸单纯使用控火能力足矣。
<br/>　　“爆。”萧逸低喝一声。
<br/>　　包裹慕容家子弟的火焰瞬间爆炸，数十人全部被炸伤，个个口吐鲜血。
<br/>　　“好强，这小废物怎么回事，竟然一人挡住了我们所有人。”慕容家子弟不可置信地看着萧逸，但再也不敢有任何动作。
<br/>　　这时，萧逸扣住慕容轩的手加大了力度，“慕容轩，你不是说要废了我吗？现在看看是谁废了谁。”
<br/>　　萧逸明显起了杀意，要杀掉慕容轩。
<br/>　　恰在此时，萧壮惊呼一声，“萧浩，你怎么了？”
<br/>　　萧逸转过头去，竟有一位萧家子弟伤势过重，倒了下去。
<br/>　　“嗯？”萧逸一惊，连忙放下了手中的慕容轩，一个瞬步来到这位子弟面前。
<br/>　　萧逸看了一眼，顿时皱起了眉头，道，“伤势很严重，必须马上医治，否则会有生命之危。”
<br/>　　“阿？”萧壮等人顿时大急。
<br/>　　“别慌。”萧逸沉声道，“你们忘了我是炼药师吗，等我为他包扎一番，服些疗伤药，可保无虞。”
<br/>　　另一边，慕容轩趁着萧逸在救人，立马转头逃跑。其他慕容家子弟也如丧家之犬般，跟着离开。
<br/>　　“慕容家的，休想逃。”萧子木等人想去追。
<br/>　　“不必追。”萧逸喝了一声，道，“等我先帮你们稳下伤势，再去找他们算账不迟。”
<br/>　　萧逸为各个子弟一一包扎，并拿出疗伤的丹药给他们。
<br/>　　半个时辰后，当所有子弟都稳下伤势后，萧逸看向萧壮，问道，“我们有多少座小型矿脉被慕容家抢了？”
<br/>　　“足足八座。”萧壮回答道，“现在他们还霸占着那里呢。”
<br/>　　萧逸继续问道，“知道慕容家的小型矿脉位置吗？”
<br/>　　“当然知道。”萧壮道。
<br/>　　“那就好，带路吧。”萧逸站了起来。
<br/>　　“你想做什么？”萧壮皱起了眉头，问道。
<br/>　　萧逸淡淡一笑，道，“从今天起，慕容家的小型矿脉，归我们了。”
