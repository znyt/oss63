<br/>　　“不知道是何人？”
<br/>　　终极剑域之中，无数的人遥望神剑山的方向，因为这种异像，便是表明，有人进入了剑山之巅三丈之内。

<br/>　　轻轻的叹了口气，绫裳便继续修炼了起来，她没有抱多大的希望。
<br/>　　毕竟，进入了剑山之巅三丈之内，也不太可能登上剑山之巅，希望很是渺小。
<br/>　　神剑山顶部，一间石室之中，身形矮壮、长有短短胡须的老者，眼眸微睁。
<br/>　　老者自然便是终极神王殿的殿主，终极神王。

<br/>　　看似只有三丈的距离，但这最后的三丈，却是天与地的距离，无法衡量。
<br/>　　与此同时，紫霄剑宫之中，一声紫袍的紫霄神王，静静的伫立在大殿之前，气度非凡。
<br/>　　“不会是那个秦陨吧！”紫霄神王低声自喃，在他的印象之中，貌似除了秦陨之外，其他人不太有能力登上剑山之巅三丈之处。
<br/>　　因为上次，秦陨曾向他出手，所以，他心中清楚，单论战斗力而言，秦陨的战斗力，轻易便可以越三级战斗，甚至更高。
<br/>　　这种战斗力，便代表了天赋，非常的罕见。
<br/>　　“你若是能登上剑山之巅，获取终极剑道的秘密，倒也是好事！”
<br/>　　少倾之后，紫霄神王微微一笑，秦陨若是得到了终极剑道的秘密，那就是为他做了嫁衣。
<br/>　　毕竟，他手中可是有那个几乎已经死去的女人，这是重要的筹码。
<br/>　　当然，紫霄神王也就是随意一想，并未放在心上，因为，他并不认为苏莫能登上剑山之巅。

<br/>　　一阵惊天动地的爆响，苏莫浑厚无比的护体源力，几乎是瞬间崩溃开来。

<br/>　　就在这一刻，苏莫全身的气势再次大涨，万化神剑元神、不败战气元神、乃至千幻元神，全部释放而出，全部催动到了极致。
