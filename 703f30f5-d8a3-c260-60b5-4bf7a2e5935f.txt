　　第610章
<br/>　　草原上一场酣战，胜败将决定这片草原的归属，决定谁是草原上的王。
<br/>　　在另外一地，同样也有一场战斗。虽然没有太大意义，不过凶险程度有过之而无不及。程大雷和燕不归的战斗已经持续一刻钟时间，这的确是很少见的情形。程大雷与人动手，要么一剑杀之，要么看打不过就逃了，寻找一剑杀之的机会。
<br/>　　但燕不归绝对不是他能一剑杀之的角色，但程大雷也并不想逃。好不容易有今天这单挑决战的机会，放过了，可能就再也寻不到。
<br/>　　今天这场战斗非比寻常，程大雷当真是拿出了看家的本领。他有杀掉燕不归的机会，同样，也可能被燕不归杀死。生死不过就是在一线之间。而燕不归很显然没有程大雷的担忧，疯子么，毕竟和正常人想得不太一样。
<br/>　　但程大雷毕竟没有拖太久的时间，如果引得大兵过来，自己今天还是杀不掉燕不归。要想及早解决战斗，必须冒险，拼那一线之上的生机。
<br/>　　程大雷与燕不归交手数招，卖了个破绽，佯装不敌，没命往前逃。
<br/>　　“想走。”
<br/>　　燕不归怎么能容程大雷脱身，在后面紧紧追赶，一步也不肯放松。小白狼看得心惊胆战，没料到燕不归竟然如此了得，竟然逼得程大雷落荒而逃。
<br/>　　二人的速度都是极快，燕不归越追越急，手中一柄重剑随时准备将程大雷砍成两段。他出手残暴，这种事也并不是没有做过。
<br/>　　程大雷回头看了一眼。
<br/>　　就是这一眼，令燕不归汗毛倒竖，冷汗唰唰落地。这不像一个失败者的眼神，出于对危险的直觉，他立刻觉察到这件事情不对。至于哪里不对，他暂时还没反应过来。
<br/>　　没等他反应过来，程大雷已经出手。
<br/>　　回头一杀，百步飞剑。
<br/>　　匹夫剑在这一刻脱手，势若劲弩，燕不归有种被锁定的感觉，他想躲，已经来不及了。
<br/>　　这飞来的一剑，洞穿了他的心脏。燕不归低下头，看到一柄剑刺在自己的胸口。
<br/>　　最后，他究竟还是败了。
<br/>　　他的意识在飘散，体力在流逝，眼前模模糊糊，像是又出现那一身黑衣的俊俏女子。
<br/>　　“贞儿，我来陪你了。”
<br/>　　仰面倒地，脸上的愤怒和狰狞在此刻烟消云散，嘴角竟浮现一抹心安理得的笑容。
<br/>　　程大雷走过来，拔出匹夫剑，血就窜了出来。他担心燕不归不死，又在燕不归喉咙补了一剑，燕不归脸上的笑容彻底僵固住。
<br/>　　程大雷瘫倒在地上，呼哧呼哧喘气，这场胜利来得当真不易。从琅琊城一直追到草原，又野人般的在草原上生活这么久，最后总算是将想杀之人杀掉。
<br/>　　小白狼牵着马走过来，道：“大当家，我们该走了。”
<br/>　　程大雷看向蛤蟆城方向，点点头：“是呐，该走了。”
<br/>　　嘟，寻找华氏一族任务完成，奖励抽奖次数1
<br/>　　在这个时候，程大雷收到一条系统提示。如此看来，刘发财已经带着华正则抵达蛤蟆城。
<br/>　　程大雷和小白狼骑上座骑，踩着荒草，登上了返回蛤蟆城的路程。
<br/>　　这里的战斗结束时，野原火那边的战斗差不多也接近尾声。在生死关头，呼延力骨子里的好战因子被激发，他挥舞着弯刀，同野原火战在一起。
<br/>　　他也并非弱手，只是多年酒色掏空身体。如果当年，他眼里自然不会搁野原火，如这种戎族小子，草原上不知有多少。但今天不同，今天的野原火已经威胁到他的存在。
<br/>　　他愤怒，恐惧，想不透……所有这些都化作杀气，要将野原火斩杀。
<br/>　　好好好，你想杀我，我便先杀掉你。
<br/>　　铛铛铛！
<br/>　　空气中钢刀碰撞，发出一声声脆响。野原火一招铁板桥，避过呼延力的刀锋，随着一刀挥过去，砍中呼延力的手腕。
<br/>　　他的刀被击落，口中爆发出一声惨叫，野原火再进，一刀砍在他肩膀上。
<br/>　　呼延力被击落马下，以手撑地后退，看到野原火骑着马靠近自己。他瞳孔放大，看着野原火渐渐靠近自己。
<br/>　　“为什么，为什么，为什么……”
<br/>　　呼延力双手撑地后退，口中不停重复着这三个字。
<br/>　　野原火骑在马上，居高临下的看着他，微微摇了摇头。
<br/>　　“事到如今，你还问这个问题，看来你的确不适合在这个世上活下去。”
<br/>　　他座下马扬起前蹄，然后重重落在，砸在呼延力身上。
<br/>　　野原火抬起头，见呼延力的手下此刻已差不多被杀死，身边围绕的，都是自己手下战士。
<br/>　　他们触及野原火目光时，纷纷举起手中兵器，高呼着同一个字——王。
<br/>　　这一战过后，呼延力被乱马踏为肉酱，尸骨无存。白无双领着野原火入城，谎称鹰图谋造反，杀了呼延力，而鹰已经被野原火诛杀。
<br/>　　在白无双的操作下，野原火兵不血刃占了王城，又昭告诸部落，野原火取呼延力而代之。
<br/>　　其中有些小部落虽然明知其中有古怪，但也无力和野原火对抗，只能选择臣服。而有些稍大的部落，则打出为呼延力复仇的旗号，选择讨伐野原火。
<br/>　　草原上刮起腥风血雨，战斗一场接着一场，野原火带着手下人，很是打了几场硬仗，出手干净利落，凡有反抗着，一概斩草除根。草原上，人头滚滚而落。
<br/>　　大约一年后，野原火已经确定自己统治者的地位，草原上再无人敢反抗他。
<br/>　　他统一草原后，取消了原先各部落的名号，将所有戎族，统一分为八个部落。
<br/>　　分别时：龙部，狼部，犬部，鱼部，鹿部，熊部，虎部，狐部。
<br/>　　戎族设立八部统治，结束了之前乱战的情形。此时，野原火已经是当之无愧的戎族王，在白无双的暗中操纵下，无数人将他视作神明。
<br/>　　他们以烈火为旗，手下铁骑二十王，良将谋士不计其数。
<br/>　　当然，这还是一年之后才会发生的事情，程大雷是事后很久才得到的情报。此刻，他还和小白狼一起，走在返回蛤蟆寨的路上。
<br/>　　咔嚓一声，头顶一道惊雷闪过，漫长的冬天终于结束，春天来了。
