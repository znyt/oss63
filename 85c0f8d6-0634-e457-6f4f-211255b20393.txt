　　以段氏基地为中枢的南方边境防线上，接受了新型训练的大批队伍渐渐赶到，文奇胜也将这批队伍统称为联盟军，当下华夏内乱已经得到解决，虽说还有一定危险的因素存在，可至少在普通人这一方已经彻底统一，尤其当后勤和管理全部由段江伟等人统一操办后，内部腐朽的问题更是不复存在，队伍之中引进的狼牙式训练和管制方案，也让联盟军的战力大幅度提升。
<br/>　　只是出乎文奇胜等人意料的是，猎者联盟并没有任何动作，甚至猎王本人也消失的一干二净，好像对不久前的那次针对性毁灭事件，完全不在意一般。
<br/>　　不过当下的境况也不允许文奇胜将重点放在猎者联盟那里，边境上有太多的事情需要他注意，最重要的当属联盟军集体服用狼牙药剂一事。
<br/>　　大批的新人类觉醒剂和等阶提升剂，在这半个月的时间里被大批的赶制而出，就在段氏基地内的那个小小狼牙研究部，只是几个研究人员而已，却偏偏给予了这数量庞大的联盟军队伍最可靠的资源。
<br/>　　同时在两名老人的支持下，所有联盟基地都往段氏基地送来了特殊晶体，数量可观的解毒剂也制造完毕，这一研究部的生产能力惊人，似乎只要能给得起材料，他们就能生产出相对应数量的药剂。
<br/>　　当然在这一被严禁踏入的研究部，其内的生产机器也同样来自于白忧的私人岛屿，经过了和上改造后成为了狼牙最可怕的生产系统。
<br/>　　卢宏盛所在的锻造部同样给力，一批又一批的武器打造而出，将联盟军手中的那些残破剑刃逐渐换下。
<br/>　　“我现在觉得，我这柄佩剑就像是一把废铜烂铁。”一名军官望着自己统领的一批队伍感叹道。
<br/>　　这批队伍正是刚拿到新武器的一批，所有人都握着名匠卢宏盛设计的武器，战力当真提升不止一个档次。
<br/>　　“我也有同感。”旁边的一名少将嘴角抽了抽，道：“原以为狼牙提供的武器，不过是在原有基础上的提高韧性和锋利而已，没想到这些武器竟然这么超概念，亏我还把我这柄佩剑当宝贝一样。”
<br/>　　“啥时候才能轮到咱们军官换啊？”另一名军官有些羡慕的问道。
<br/>　　哪知此话一出，好几人当即苦了脸，他们提前打探过这批武器并非真正卢宏盛亲自锻造，而是来自狼牙锻造部的流水线产品。
<br/>　　“之前问我要不要换武器的时候，我想都没想就拒绝了。”
<br/>　　“我也是……”
<br/>　　段氏基地这一中转站并不能让联盟军停留太久，甚至很多部队都来不及经过这里，就直接赶往了分散在各处的防守点。
<br/>　　整个由东到西的这边境长线，已经以惊人的速度建立起一条防守线，没过一段距离就是一处小型基地，那里早已投放准备好大批的物资，除却武器还有可供一万人食用十天以上的食物。
<br/>　　准备非常浩大，段江伟在这段时间里光是资源调配就忙的焦头烂额，所以他更能够深刻体会到狼牙内部系统的可怕。
<br/>　　因为这条边境线的建造，是狼牙工程部的功劳，并且建造还在持续，苏行已经带着那批队伍不知道走到了多远的地方，一处处小型基地像是凭空出现一样，建造的速度让段江伟的资源调配都跟不上。
<br/>　　更别说那些可怕的各种新武器，以及锋利的刀剑兵刃，甚至药剂……
<br/>　　终于在最后一批联盟军队伍到达制定地点后，这条边境线的防守总算初步完成，所有人都待在自己该在的位置，时刻准备着与对面南方冲来的丧尸来一场生死搏斗。
<br/>　　而在这半个月以来的准备期间，那些丧尸自然不会坐以待毙，早在猎者联盟被针对的事件爆发后，一股尸潮就率先朝着段氏基地旁边的另一处基地发起了进攻。
<br/>　　它们并没有选择直接攻打段氏基地，而是选了一处并不起眼的防守基地，并且出动的丧尸数量高达了好几百万。
<br/>　　在接到消息的第一时间，防守线上的各方队伍都开始朝着那处地带转移，那一基地同样是被苏行工程部改造过的地方，放着数量可观的新武器，以及大批的枪支弹药，只是联盟军还未全数到达所以防守的人员不够。
<br/>　　于是在最初的两天时间里，驻扎在这一基地的人拼命消耗着新武器，不断用远程防御，基地之外躺着大·片的丧尸尸体，整个平地都呈现出了一片黑色海洋。
<br/>　　好在这条防守线的设计一开始就从支援为先出发，两旁的队伍用最快时间赶到，然后展开了与这批丧尸群的一次大战。
<br/>　　战役持续了整整一周的时间，参战人员数量高达好几万，热武器消耗量惊人，甚至还用上了和上最新发明的那台音波音响。
<br/>　　参与这次大战的带队将领是金泽宇，战斗中最紧张的就是时间，联盟军的人一边战斗一边不断用实战消化着学来的战术技巧，那些药剂也在关键时刻派上了用途。
<br/>　　终于在苦战了一周之后，金泽宇斩下了这批尸潮的丧尸小头领脑袋，宣告了第一场防守战的胜利。
<br/>　　虽然相对比狼牙战团的那场惊世骇俗的战役，这一次的胜利毫不起眼，甚至伤亡率都让金泽宇没脸说出口，但后方赶到的大批联盟军却兴奋异常，因为这是他们真正意义上的一次逆袭胜利。
<br/>　　无论从数量还是从战力上来看，这场战役都值得被大肆宣扬。
<br/>　　文奇胜的要求更简单，他一开始就明白想跟狼牙战团一较高下根本不可能，他们要做的就是守住这条防线，让对面的那批丧尸，哪怕数量再多，多到十几亿一起进攻都无济于事，都不能踏入华夏国土一步！
<br/>　　段江伟也在这一周内忙的手脚抽筋，于是在战役结束后当即把文职卸下，全数交给了副官鲁初雪来做，自己则是带着一批联盟军队伍跑到另一处防守点驻扎。
<br/>　　哪怕条件艰苦，哪怕长时间的只能吃干涩食物，他也再也不要去管什么后勤，上将就应该待在战场。
