　　一群战门成员眼中都是露出狞笑。
<br/>　　王长老目光微微眯起，他知道，这几天战门内的人和第一邪教的人走的很近，双方算是同盟关系，但是他没有想的自己组织内的成员似乎也都已经不知不觉被洗脑了。
<br/>　　“这个第一邪教还真是够恐怖的，据说他们教内有许多异能者没有任何战斗能力，纯粹的就是进行精神上的洗脑，类似于传销蛊惑一样的异能。”
<br/>　　王长老心中隐隐有些担忧。
<br/>　　“长老，不如让一些成员去杀着练练手吧。”
<br/>　　一个战门成员冷笑道。
<br/>　　“没错啊长老，反正都是些垃圾，他们眼下最得力的武器也没了，让我们去杀他们玩玩也不错。”
<br/>　　一群战门成员都开口了。
<br/>　　“嗯，既然如此，那你们就去玩玩吧。”
<br/>　　王长老点了点头。
<br/>　　“走咯，杀猪去咯！”
<br/>　　“猪？这些人可没资格当猪，他们只是蛀虫罢了！！”
<br/>　　一群战门成员哈哈大笑着离开了此处。
<br/>　　而此时此刻的林峰遇到了麻烦。
<br/>　　他们的直升机被一群变异鸟给困住了。
<br/>　　“我们这个直升机没有火力系统，无法靠近了，若是强制突破，会被变异鸟击落的！”
<br/>　　飞行员道。
<br/>　　“那就就地降落！我已经看到了先遣大队！”
<br/>　　林峰眯着眼道。
<br/>　　一群人都看到了前方出现的一个巨大光罩。
<br/>　　每个人都是一脸的诧然和震惊，显然不知道这是什么东西。
<br/>　　直升机就地降落后。
<br/>　　林峰就带着陈勾冲了过去。
<br/>　　“狗子，肉弹战车你学了没？”
<br/>　　林峰问道。
<br/>　　“放心，我早就学会了。”
<br/>　　陈勾嘿嘿笑道。
<br/>　　“很好。”
<br/>　　林峰两人一起朝前冲。
<br/>　　在直升机上因为视野的问题可以看到那个光罩，然而两条腿跑起来还有点远。
<br/>　　“林峰，那个像奶罩一样的东西到底是什么。”陈勾问道。
<br/>　　“不知道，不过肯定是战门的手段，先遣大队肯定已经中招了。”
<br/>　　林峰说道。
<br/>　　刷刷刷。
<br/>　　也就在这时忽然出现了三四个人。
<br/>　　林峰下意识的拔出大刀。
<br/>　　不过一看发现居然是罗风。
<br/>　　罗风身后跟着三个二十多岁的年轻人。
<br/>　　“罗风？”
<br/>　　林峰道。
<br/>　　“我们来晚了，这是战门组织内拥有封锁异能的异能者施展的手段，他们牺牲自己的生命，把自己的异能给发挥到极致，这才把先遣部队给困在其中，所有的坦克火力都没有办法使用了，所以要想救他们出来，只能灭掉这些异能者。”
<br/>　　罗风道。
<br/>　　“林峰，这是几个异能者的呆着的地方，把这几个异能者给灭了，他们的计划就不攻自破，当然了，事情没有这么容易，每个异能者身边肯定有不少战门之人保护。”
<br/>　　罗风说，
<br/>　　“牺牲异能者？这什么仇什么怨。”
<br/>　　林峰震惊了，异能者那可是宝贝一样的存在，可是这个战门居然舍得拿来牺牲。
<br/>　　“这个战门已经到了丧心病狂的地步，哪怕是要牺牲异能者也要灭掉先遣部队，看来他们已经被第一邪教给控制了，说是同盟，实际上，战门已经成为了第一邪教的一把利剑了。这是最糟糕的消息，但愿是我多想了，事不宜迟，我们先走一步。”罗风说完后就带着他的人离开了。
<br/>　　林峰有些糊涂，不过现在很显然不是去关心这的问题了，事不宜迟，每多拖一秒钟，可能先遣部队就要多死一个人。
<br/>　　“狗子，我们也走！”
<br/>　　“林峰，我怎么感觉我们变成奥特曼了，这拯救人类的重要使命就在我们肩膀上了。”陈勾道。
<br/>　　“差不多这个意思，若是这一股精锐死在这里，那曙光基地要不了多久也要葬送，所以还真的是拯救人类。”
<br/>　　林峰道。
<br/>　　两人一路狂奔来到了距离他们最近的一个据点。视线之中，有三个战门之人在来回巡逻警惕。在他们之中，有个人坐着就像是观音菩萨一样的闭着眼睛，在那里催动自己的异能。
<br/>　　“有三个人把守。”
<br/>　　林峰扫了眼。
<br/>　　“都是五阶的。”
<br/>　　“上！”
<br/>　　林峰一马当先冲了过去。
<br/>　　“谁？”
<br/>　　三个战门成员立即察觉纷纷朝林峰这边扫来。
<br/>　　“肉弹战车！”
<br/>　　下一刻他们就看到了一个大肉球滚了过来。
<br/>　　“靠，什么玩意。”
<br/>　　三个战门成员立即出手。
<br/>　　砰砰砰。
<br/>　　下一刻就被直接给撞飞了。
<br/>　　林峰冲过去直接给他们一刀一个给秒杀了。
<br/>　　林峰目光落在这个活人祭坛上面，此时此刻这个活人祭坛升起了一抹光束落在那所谓的天照大阵上。
<br/>　　在这个活人祭坛之中的确盘膝坐着一个人。
<br/>　　这个人也就二十多岁，但是此时此刻他的皮肤开始逐渐的萎缩。哪怕林峰来到近前了他都没有任何的察觉，就好像入定了一样。
<br/>　　“活死人？”
<br/>　　林峰直接是一拳给他砸爆了。这个人歪倒在一边，而这个光束也消失不见了。
<br/>　　与此同时。
<br/>　　“不好，我们一个阵眼消失了！”
<br/>　　“什么？怎么会这样？”
<br/>　　王长老皱着眉头道。
<br/>　　“还不知道发生什么，不过看守这个阵眼的三个战门成员都已经失去了联系！”
<br/>　　王长老面色难看，“没想到居然有人知道了，而且还摸到这里来了。”
<br/>　　“长老，我们该怎么办，若是失去阵眼，那这个天照大阵可就不攻自破了。”
<br/>　　“你们所有人不用呆在这里了，都立即去查看情况。注意阵眼的防卫。”
<br/>　　“是！”
<br/>　　瞬间留在这里的几十个战门成员全部奔了过去。
<br/>　　王长老面色阴沉。
<br/>　　“难道是星陨阁，那群坏事的家伙又来了？”
<br/>　　“王长老，我也去吧，此地有你镇守就足够了。”
<br/>　　一个长老道。
<br/>　　“嗯。”
<br/>　　王长老点了点头。
<br/>　　瞬间这个长老也飞奔而去。
<br/>　　而此时此刻。
<br/>　　林峰和陈勾已经朝下一个阵眼飞奔而去。
<br/>　　不过这一次，因为有了前车之鉴，这群看守的成员都格外的警惕了。
<br/>　　不过警惕也没用，都是些五阶的，最强也不过六阶，直接被林峰碾压。
<br/>　　“第二个了！林峰，比我们想象之中的要容易很多啊。”
<br/>　　陈勾道。
<br/>　　“别大意，说不定危险在后面，继续，第三个！”
<br/>　　林峰道。
<br/>　　此时此刻。
<br/>　　先遣部队一群人还一头雾水，他们根本不知道发生了什么。
<br/>　　“哈哈，瞧瞧这群可怜虫。”
<br/>　　一道冷笑传来。
<br/>　　“谁？”
<br/>　　陆勇立即回头瞧去就看到了几十个人出现了。
