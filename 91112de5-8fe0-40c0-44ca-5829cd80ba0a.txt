　　“吾观二弟走的是阴阳一道，我此处有一道神通，就在此传给二弟吧，希望这对你能有所帮助。”
<br/>　　元始说完，心念一动，阴阳印直接在虚空形成一个黑白阴阳太极图，阴阳图文缓缓旋转，阴阳相生，玄之又玄。之后，阴阳图文直接化为一道流光飞入老子神海之中。
<br/>　　“多谢大兄！”老子对着元始行了一礼。
<br/>　　“无妨，只要对你有帮助就好。”元始摆摆手，无所谓道。对于自家兄弟，他还是很照顾的。
<br/>　　“大兄，我呢？”通天见自家大哥传给二哥神通，却不见传给自己，顿时急了。
<br/>　　“哈哈，不急，大哥又怎会少了你呢，你也有。”
<br/>　　“我看三弟你走的是剑之一道，剑道我并不懂太多，但，我看你对四象之道也很是精通，我这有一印，名为四象印。”
<br/>　　说完，心念一动，虚空之中，四象印化为四大神兽，青龙、白虎、朱雀、玄武，相互厮杀咆哮，神威滔天，之后，再次化为一道四角神印，化为流光钻入通天神海之中。
<br/>　　“好，多谢大兄。”通天亦兴奋不已。
<br/>　　“好了，既然论道完毕，你们也回去吧，此次论道大家收获都很大，需要巩固一番，而且现在正值天地大劫，你们就在岛上好好闭关修炼吧。”
<br/>　　元始对着两人道。
<br/>　　“好，那——大兄，我等就先行回去了，有时间可去我们岛上论道。”两人再次向元始作了一揖，就出来玉虚宫，驾云而去。
<br/>　　“我也该好好闭关巩固一番了。”看着两人离去的声音，元始默默道。
<br/>　　之后转身，回到悟道台，再次闭关，参悟大道。
<br/>　　……
<br/>　　东海龙宫深处，龙后宫殿。
<br/>　　一个妩媚妖娆的贵妇身穿金缕玉衣，头戴黄金凤冠，斜卧在床榻之上。
<br/>　　“娘娘，四太子前来觐见。”这时，一个侍女走进帷帐，对着贵妇说道。
<br/>　　“嗯？”贵妇被人打扰，有些恼怒地皱起眉头，刚要大骂，但听到来话，端坐起来，对着那侍女道：“请他进来。”
<br/>　　“是，娘娘！”侍女微微行了一礼，躬身退下。
<br/>　　之后，一个身穿银色龙袍的青年男子走了进来，他脸色苍白无比，好像下一秒就会倒下，对着贵妇单膝跪地，双手抱拳，虚弱道：
<br/>　　“咳咳！母后！”
<br/>　　贵妇看到青年，微微一惊，道：“我儿这是怎么了，莫非出了什么事儿？赶紧起来。”
<br/>　　青年站了起来，走到贵妇身旁，轻声道：“我怎么会有事儿？只是近来思念母后了，特意前来看望母后。”
<br/>　　“你啊！我还不知道你吗？你的脸色如此苍白，定是出了什么事故？”贵妇见青年不说实话，轻笑一声，并没有怪罪青年。
<br/>　　“哈哈，真是什么都瞒不住母后你啊！咳咳！”男子也不尴尬，微微一笑，只不过脸色更加苍白了，他直接说道：
<br/>　　“我最近修炼遇到瓶颈了，想要突破不是非常容易，我估计，是我的血脉限制了我，所以，我想——”
<br/>　　说道这里，男子明显有些犹豫，不知道该不该说出口。
<br/>　　“你想怎么样？”贵妇问道。
<br/>　　“我想请母后你去父皇那里帮我要三滴精血，这样我的血脉就更加高贵了，马上就能突破了。”
<br/>　　“什么？你父皇的精血？”贵妇一惊。
<br/>　　“是啊，咳咳！母后，你看，孩儿自从上次被麒麟族的强者打伤后，回来就一直刻苦修炼，从没有怠慢过，但这修为到了太乙金仙圆满后，就一直不再进步，我估计是上次伤到了本源。难道母后要看着孩儿一直如此吗？”
<br/>　　青年一把鼻涕一把泪地哭了起来，那模样，要多惨就有多惨。
<br/>　　贵妇思索了一会儿，觉得有理，不过，她并不放心，眼中闪过金光，直射男子体内，在男子体内探查了一个周身，最后在龙珠之处，发现龙珠断裂了一条细缝，不由一惊。
<br/>　　对龙族来说，龙珠就是一条龙的命，一身修为，大多都集中在龙珠之上，要是龙珠出了问题，那不仅仅修为难得寸进，弄不好还会倒退，更甚至会要了龙命。由不得龙不慎重。
<br/>　　男子见贵妇探查他的身体，心中不由一声感叹，果然如此，要不是我早有准备，此时已经被母后发现了。不过，这种有损本源的事情，以后没有必要之时还是少干，不然我就没命了。不过，为了尊主所交任务，我还是拼了。
<br/>　　这青年男子自然就是被元始所度化的龙族四太子敖玉，现在为了完成元始交代的任务，忍住巨大痛苦，自损本源。
<br/>　　贵妇脸上阴晴不定，他很清楚龙珠对龙族的重要性，但祖龙精血对于龙族一样重要。一边是自己丈夫，龙族的一族之长，另一边是自己的儿子，自己唯一的儿子，她夹在中间，一时间不知如何是好？
<br/>　　敖玉见贵妇脸色已有松动，就决定再加一把火，说道：
<br/>　　“咳咳，母后，你看，这些年，二哥、三哥、四弟、五弟都遇害了，只留下我与大哥，为什么死的都是父皇的子孙啊，他们死得好惨呐，咳咳！
<br/>　　呜呜…要不是我一直在龙宫之中闭关不出，恐怕我都要被杀死了。现在我受伤了，他们恐怕又要盯上我了。呜呜呜…母后！你要为我做主啊！”
<br/>　　轰！
<br/>　　敖玉的一席话在贵妇的耳边炸响，为什么死的都是父皇的子孙？是啊，为什么死的都是自己丈夫的儿子，莫不是这里面有诈？以前死的不是自己的儿子，她也就没有多管，现在看来，这其中有大问题，莫非是大太子？
<br/>　　贵妇被自己的想法一惊，之后，有所决定，不管了，自己的儿子就这么一个，我就算舍弃颜面也要想龙祖索要三滴精血。
<br/>　　“我儿，你放心，我这就去你父皇那儿帮你要三滴精血。”贵妇对着敖玉说道。
<br/>　　“咳咳，多谢母后！”敖玉虚弱道，嘴角露出一个让人不易察觉的弧度。
<br/>　　“你先去宝库中取些弥补精血，滋补本源的灵药，回去补补身体，我这就去了。”贵妇对敖玉轻声叮嘱道。
<br/>　　“多谢母后！”
<br/>　　敖玉躬身一拜，之后才走出宫殿，向着龙族宝库走去，他知道，自己算是暂时过关了，不过，为了不易让人发现，还是先一直保持原样吧。
<br/>　　至于大哥那里，嘿嘿，这可不是我要陷害你，谁让你是大太子呢？你不死，我将来怎么做龙皇？
