　　“让我跪下？杀光九域所有人？”
<br/>　　楚风眠听到这严家众人一句句狂傲的话，怒极反笑道。
<br/>　　“很好，看来从今日起，严家也没有存在的必要了！”
<br/>　　楚风眠当即运转灵力，突然间恐怖的力量所楚风眠的身上涌现而出，凌空一挥，化为了一道龙爪，横扫一切。
<br/>　　那想要让楚风眠跪下的严家弟子，顿时被龙爪击中，被当场打飞，身躯都被这一爪的力量之下，硬生生的撕裂开来。
<br/>　　一名七劫古帝，在楚风眠的手中，就连蝼蚁都算不上。
<br/>　　这名严家弟子，就连反抗都没有做到，到死他的脸上还是那种狂傲之色。
<br/>　　都没有能够反映过来，在他刚刚看来，就蝼蚁一样可以随意拿捏的楚风眠，居然是一尊真龙。
<br/>　　那一名七级古帝的精血，都没有留下，在龙爪的轰击之下，凌空湮灭！
<br/>　　“该死！”
<br/>　　“严寂死了！”
<br/>　　“杀了此人，为严寂报仇！”
<br/>　　其他的严家古帝，看到那严寂的死，震怒了，无数灵力当即向着楚风眠轰击而来。
<br/>　　“来再多也没有用！统统给我破！”
<br/>　　楚风眠面对无数灵力的轰击，脸色都没有丝毫的变化，始祖天龙的力量在楚风眠的身躯之中运转起来，凝聚在了楚风眠的掌心之中。
<br/>　　一爪轰击，龙力凝聚，化为了始祖天龙的力量，横扫一切。
<br/>　　始祖，第一式！
<br/>　　轰隆！
<br/>　　这一爪跟那严家无数古帝的力量凌空碰撞，比起十几名严家古帝的力量，还要强大，将他们硬生生震飞了出去。
<br/>　　时另一只手，楚风眠猛然一捏，被本来捏在楚风眠手心之中的那严皆，当即是被捏碎。
<br/>　　一团精血，随着吞天秘术，被全然吞噬殆尽。
<br/>　　“龙族武技？”
<br/>　　“好玄妙的武技！”
<br/>　　严家众人之中，那严君，严苍，一直没有出手，在看到楚风眠施展出的第一式的一刻，都是目光之中显现出几分惊奇。
<br/>　　他们二人，都是八劫古帝巅峰的高手，纵然是在阎罗星域，都是绝对的强者了，见多识广，一眼就足以看出，这始祖第一式的玄妙之处。
<br/>　　几乎是没有破绽，接近完美的武技。
<br/>　　“这武技是龙族武技，我们没用，不过得到了，卖出去也是一大笔财富！”
<br/>　　严君盯着楚风眠，眼神越发的炙热贪婪起来，转过头对着严苍开口道。
<br/>　　“看来这小子得到的奇遇可不少，去，抓住此人，要活的！”
<br/>　　轰！
<br/>　　严苍没有说话，而是直接身躯一动，猛然向着楚风眠冲了过去，这速度之快，几乎是超出了楚风眠的想象。
<br/>　　一眨眼的功夫，就来到了楚风眠的面前！
<br/>　　“好快的速度！”
<br/>　　楚风眠目光闪烁，他也看的出来，这严苍绝不是什么好对付的人，当即运转灵力，又是一爪轰击而去。
<br/>　　“始祖第二式！”
<br/>　　龙爪横扫一切，凌空对着那严苍轰击而去。
<br/>　　可是这严苍，却是丝毫不退，而是以一双手，硬接这始祖第二式。
<br/>　　严苍双手握拳，呈现出极致的武道，一拳跟龙爪，凌空相撞。
<br/>　　“轰隆！”
<br/>　　力量横扫。
<br/>　　那些其他的严家古帝，就连这力量的余波都承受不住，都是连连退后，可是那严苍，却是硬生生凭借他的双拳，抗了下来。
<br/>　　这严苍的双拳，被震破流下了无数鲜血，但是对这个级别的强者来说，除非是一击重伤，不然这种伤势，瞬息之间就足以恢复。
<br/>　　“这严苍的力量真是恐怖！”
<br/>　　楚风眠目光之中露出几分忌惮，这严苍的实力，已经是接近到了小叶帝那个级别了。
<br/>　　当然小叶帝比起严苍，强的不是力量，还有他接受的力量传承的剑道，如果抛开力量传承的剑道。
<br/>　　小叶帝都未必是这严苍的对手。
<br/>　　阎罗星域，居然也有如此天才。
<br/>　　“小子，给我死来！”
<br/>　　就在这力量碰撞的刹那之间，一道身影，突然出现在楚风眠的背后。
<br/>　　正是那严君，如今他脸色狰狞，盯着楚风眠，突然大手一挥，拿出了一枚玉符，随即口中吐出了一大口鲜血，喷在了这玉符之上。
<br/>　　顿时这一枚玉符的力量，被催动起来，融入到了严君的身躯之中，猛然一拳对着楚风眠轰击而去。
<br/>　　偷袭！
<br/>　　在楚风眠跟这严苍交手的一刻，这严君突然偷袭。
<br/>　　那一枚玉符，必须是以严君的精血催动，这严君是宁可损失一些精血，今日也要将楚风眠所轰杀。
<br/>　　“纵然是损失一些精血，杀了这小子，一切都能弥补，这小子身上的秘密实在是太多了，肯定是有着惊天奇遇，杀了他，我严君未来必定无敌！”
<br/>　　严君看着那一拳已经是轰击在了楚风眠的背后，兴奋的大笑道。
<br/>　　但是他的笑意，却是突然间的凝固了。
<br/>　　只见在楚风眠的背后，一层层的龙鳞浮现而出，在这龙鳞之下，楚风眠的身躯，更是化为了一枚枚晶体状。
<br/>　　那一拳的力量，在接近到龙鳞的一刻，突然被弹开，随即一股灵力更是向着严君反噬而去。
<br/>　　“不好！”
<br/>　　这严君脸色骤然一变，他完全没有想到，在楚风眠跟这严苍交手的时候，居然还留有余力，守护背后。
<br/>　　瞬息之间严君的身躯连连后退，打算离开。
<br/>　　可在楚风眠的背后，天涯剑匣当即出现，两道剑光，数道剑光从这剑匣之内飞了出去，对着那严君猛然斩了过去。
<br/>　　楚风眠已经是被这严君偷袭过一次了，可不会被他偷袭第二次，这严君一直没有出手，楚风眠便是明白，他就是在等待着偷袭楚风眠的机会。
<br/>　　所以楚风眠在跟严苍交手的时候，也在时刻注意这严君。
<br/>　　严君在等这一刻，楚风眠一样在等。
<br/>　　这严君距离楚风眠，不足十米，这数道剑光，几乎是瞬息而至。
<br/>　　“扭转乾坤！”
<br/>　　严君脸色大变，双手一挥，他的身躯开始变的虚无起来，随即向着远方遁逃而去。
<br/>　　可是还是来不及，两道剑光，还是斩在了严君的身躯之中，顿时严君吐出了一大口的精血。
<br/>　　严君刚刚为了催动那玉符的力量，本就是吐出了一大口的精血了，这一下子又是吐出大量精血，他身上的境界都开始跌落下来，从八劫古帝巅峰，一直是跌落到了寻常八劫古帝的地步。
