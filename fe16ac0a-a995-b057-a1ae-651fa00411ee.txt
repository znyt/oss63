张疑问的脸，陈默还能怎么说，只能将所有的东西捡能说的给他们细细的说了一遍！<br/>　　<br/>　　    “你说你姐现在忙着的酒厂，是你和陈老四一起做的？”姥爷问道。<br/>　　<br/>　　    “是的！”<br/>　　<br/>　　    “嗯！不错！陈老四那个人很不错，实诚，不是什么滑头的家伙，还是可以合作的！”姥爷知道陈老四这个人，所以也就点点头认可了。<br/>　　<br/>　　    “我说大丫怎么这么多天没有过来，原来是忙着建酒厂啊！哈哈，姐夫，你家二娃有出息了，也能让帮衬着兄弟姐妹了！”大舅也是笑着说道。<br/>　　<br/>　　    以前姐夫家里，他是知道的。而且自己也帮衬了不少，所以有这样的发展，他是从心里高兴。<br/>　　<br/>　　    “对了，怎么大丫这么忙，那么关明他们一家没有意见啊？”姥爷有些担心的问道。<br/>　　<br/>　　    他还不知道关明和陈萍已经离婚，这边都是不知道的。陈建国和付慧丽其实已经知道了，这件事姐姐没有办法隐瞒住自己的父母，所以她就全部给说了，也算是得到了父母的支持。<br/>　　<br/>　　    可以说家里有了陈默以后，自然也就发展的不错，对于关明一家如此对待陈萍，自然是自家的闺女自家心疼，所以都是持支持的态度。<br/>　　<br/>　　    但是姥姥姥爷他们不知道，所以陈建国也不好说什么，就那么转头示意陈默说说。<br/>　　<br/>　　    “这个，姐姐就是因为和关明吵架，才想着到家里住一阵。我看着姐姐也没有个事情，也没有啥收入，就给姐姐找个事情做！”陈默只能这样说。<br/>　　<br/>　　    “嗯？关明和你姐吵架，什么原因？”姥爷问道。<br/>　　<br/>　　    “哎！还能有什么原因，都是老一套！”陈默一撇嘴说道。<br/>　　<br/>　　    “这个，夫妻吵架床头和，没有什么隔夜仇，也没有什么过不去的坎！有时间你还是去劝劝的好！”姥爷心中也是比较心疼自己自家的外孙女，但是在农村这里，嫁出去的闺女自然就是别人家的孩子，所以姥爷还是想让陈默劝劝，让她们好好过日子。<br/>　　<br/>　　    &lt;/content&gt;<br/>　　<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
