        &lt;h3&gt;第三卷第一百二十九章  寒家总裁&lt;/h3&gt;
<br/>
<br/>        寒家公司会议室。
<br/>
<br/>        寒伯安坐在会议室里，大屏上正在播放一起运输中发生的事件，他们在这个行业已经做了几代了，随着时代的变迁，寒家人也是努力的跟上社会的步伐，让寒家的事业不被时代所淘汰。
<br/>
<br/>        他们现在所接触的业务都是世界性质的，面对的也是社会上的一些精英人士和大型的企业家，这一次的事件是有人藏了违，禁，物品，被他们给查出来了。
<br/>
<br/>        对方开始不承认，但是后来又求着他们，说这一次就放过了，下一次坚决不犯同样的错误。
<br/>
<br/>        这件事情纠缠了几天，还没有解决。
<br/>
<br/>        寒伯安知道后，召开了公司高层会议，像这种小问题完全都不用拖拖拉拉，直接一票否决。寒家能一直站稳脚跟，跟他们的处事原则有很大的关系。
<br/>
<br/>        他们从来不沾染不好的东西，有竞争，但是不违心。他们没有办法管别人，但是他们能保证他们自己，在他们这个行业里，寒家占据着第一，带领着一帮业内人士兢兢业业，稳步前进。
<br/>
<br/>        “据调查，这一家一直是黑三他们帮他们运输的，不知道为什么和黑三闹矛盾了，才转手让我们运送。”运输部的老总说。
<br/>
<br/>        “现在他们不卸货，一直占用着我们的设备。而我们通知他们了，他们也不来处理，还有，就是这几天有了好几拨都是说情的电话。”运输部的老总为难的说。
<br/>
<br/>        “寒家从来不运送违，禁，物品，这个是全世界都知道的事情，谁这么大的胆子？”寒伯安冷着声音问。
<br/>
<br/>        “这家企业是原来陈家的合作伙伴，落家将陈家收购之后就不和他们合作了，他们就转投海外市场，他们这次找了吴家的一个人打电话给我们。”运输部的老总额头冒汗。
<br/>
<br/>        大家听说吴家，心里一惊，难怪他们顶不住，难怪人家不来卸货，摆明了就是你不送也得送。
<br/>
<br/>        “这是故意给我们找事？”寒伯安说完，整个会议室只听得见呼吸声，大家都噤若寒蝉。
<br/>
<br/>        “通知他们，给他们三天期限自己来卸货，三天以后，让媒体曝光，下次再有人说情，录音。”寒伯安的语气里不带一丝温度。
<br/>
<br/>        “可是.....”
<br/>
<br/>        “没有可是，我们寒家能走到今天，靠的就是我们的规矩，无规矩不成方圆，今天你为吴家打破规则，明天你就会为更多这样的人打破，以后你就得被别人牵着鼻子走。就算是天王老子也不行，别说是什么吴家！”寒伯安掷地有声。
<br/>
<br/>        “是。”大家原有的担心烟消云散，吴家这几年的势头无两，他们不敢得罪，但是看着自己年轻的老总毫不犹豫的样子，大家心里有谱了。
<br/>
<br/>        寒伯安的气魄更胜老总裁，如果是老总裁怕是会和他们商量一番，寻找一个折中的方法，但是寒伯安行事作风强硬，不拖泥带水，也是一副天不怕地不怕的性子。
<br/>
<br/>        散会后，助理李长卿问寒伯安：“寒总，你不怕吴家会报复我们吗？”
<br/>
<br/>        “怕？我还不知道怕是怎么写的？只要你行的端走得正你怕什么？做生意亦如做人，你规规矩矩做事情，他能拿你怎么样？不过防小人不妨君子，以后给我盯紧点，小心有人栽赃陷害。”寒伯安吩咐李长卿。
<br/>
<br/>        “是。”
<br/>
<br/>        “最近我弟弟有没有消息？”寒伯安问起了他的弟弟，弟弟从到落家之后就一直在落家，对寒家的事情真的是不闻不问，连他这个大哥也很少联系。
<br/>
<br/>        “二公子现在和落家小总裁在欧洲，落家在欧洲有庄园，大概过年的时候他们会在欧洲过。”李长卿是助理，事关寒家大小人、大小事情他都得知道。
<br/>
<br/>        “你说我弟弟和落家那个女子到底是个什么关系？我相信他去落家肯定是为了那个女人，这都几年了还没有什么结果，反而让别人捷足先登了。”寒伯安摇摇头，他的弟弟就是一根筋，情商不敢恭维。
<br/>
<br/>        “你这么一说，我到是想起来了，二公子之前被人陷害过，但是还是在落家一直呆在，你说这一次的事情和二公子有没有关系？”李长卿确定的问
<br/>
<br/>        “愚子可教也，长点见识吧，如果我们同意了帮那家运送违禁货物，到时候我们将会陷入万劫不复的地步。”寒伯安冷冷的说。
<br/>
<br/>        “呵呵，那个傻子当我们寒大帅哥是傻子呢，真是瞎了他的狗眼。”李长卿拍了一下马屁。
<br/>
<br/>        “滚去干活吧，盯紧了，另外查一下我们内部的问题，接单的人是谁。”寒伯安再次叮咛。
<br/>
<br/>        “这个不用你说，都已经查出来了，这个人和破落的陈家、吴家都有点瓜葛。我现在并没有惊动他，想看看还有几个人和他是一起的。”作为寒总裁的助理，事事都要关注，他就是寒伯安的眼睛。
<br/>
<br/>        “让大家谨慎些，虽然我们是安安分分的做事情，但是各种错综复杂的关系也会牵扯到我们。这个世界，没有人是能独善其身的。”
<br/>
<br/>        “就你这聪明聪明的脑袋，谁能动得了你？”
<br/>
<br/>        “得了，你不就是想给你脸上贴金吗？”寒伯安学着李长卿的声音：“聪明人是相互吸引的，只有我这么优秀的人才能做你的助理。”
<br/>
<br/>        “那是当然！”李长卿很是嘚瑟的笑。
<br/>
<br/>        “我提前联系一下媒体，万不得已的情况就利用媒体曝光，不过那个时候怕真是撕破脸皮了。”
<br/>
<br/>        “做好准备吧，该来的总会来的。”寒伯安不搞事情，但是也不怕事情。
<br/>
<br/>        寒伯安点上一支雪茄，并没有抽，而是任雪茄的烟雾缭缭，办公室里弥漫着雪茄的味道。
<br/>
<br/>        大上海，从来都不是一块安然之地，社会发展到今天，全仗着国家强盛，才有今天的地步，但是每一个发展的社会都不会是一帆风顺的。
<br/>
<br/>        现在的商场是不见硝烟的，是满脸含笑的，但是其争夺性一点也不会减少。
<br/>
<br/>        他带领的不但是他们寒家的企业，他是他们这个行业的标杆，一杆旗帜，如果他都能运送违，禁，物品，那么这个行业就会泛滥。
<br/>
<br/>        这是寒伯安不能忍受的。
<br/>
<br/>        元旦和张函的聊天，张函那铿锵的话还响在耳边：“做一个有良心的企业，做一个有良心的中国人。”
<br/>
<br/>        他们这一代年轻人赶上了好时代，更要有一颗热爱这片土地的心，他也经常出差，看到了有些战乱的地方真是唏嘘不已，那些人生活在水深火热之中，连一顿饱饭都吃不上。
<br/>
<br/>        往往那个时候看的他热泪盈眶，身为中国人，是他的骄傲，是他的自豪！
<br/>
<br/>        “做一个有良心的企业，做一个有良心的中国人。”被写成语录，挂在了他办公室的墙上！
