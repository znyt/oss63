　　村子外丛林中，罗昊耳麦中传来了布谷的声音。
<br/>　　“狂龙，我们得手了。巴坦伦的直升机在起飞的时候，被我一枪打爆了油箱，跟着直升机一起坠入了山林，除非他是属猫的，有九条命，不然死定了。”
<br/>　　“干得漂亮！”
<br/>　　罗昊拿出手机对着之前给比洛多发送短信的那个号码，又发送了一条短信，“巴坦伦已死，可以动手！”
<br/>　　两架满载弹药的AH-64D“长弓阿帕奇”从cocaine种植园西侧一座山的背后飞了出来，朝着cocaine种植园的方向俯冲而来。
<br/>　　当年比洛多前往美国交流学习的时候，跟美国军方达成了一笔价值三亿美金的军火交易，其中包括了M16A3自动步枪，M4A1自动步枪，单兵反坦克导弹在内多种单兵武器，以及这两架AH-64D“长弓阿帕奇”武装直升机，配套的武器弹药和后期维护。
<br/>　　比洛多把这价值三亿美金的军火全都用来装备了他手下那些去美国驻南美洲军事基地的部队，这也使得比洛多手下的那支部队成了整个联合政府军中最精锐的部队，战斗力远超联合政府军中其他的部队。比洛多这次把这两架“长弓阿帕奇”武装直升机给派出来配合龙隐部队的这次行动，可以说是把自己家底全都掏了出来。
<br/>　　但是这对于比洛多来说，值！巴坦伦一死，比洛多就成了这次大选最有利竞选总统的人选，如果这次能顺利拔除掉布克尔特这个大毒枭，打掉他的武装势力和贩毒集团，对于比洛多来说是巨大的大选筹码，可以用来力压瓦斯汀的大选筹码。
<br/>　　所以把自己的家底拿出来配合龙隐部队的行动，对于比洛多来说，简直就是一本万利，太值了！
<br/>　　种植园内，负责看守种植园内工人的武装毒贩注意到朝着种植园冲来的武装直升机，立刻端起M4A1自动步枪对着天空中那两架AH-64D“长弓阿帕奇”武装直升机进行射击。
<br/>　　哒哒哒哒哒……
<br/>　　铿铿铿铿铿……
<br/>　　5.56mm口径的步枪子弹根本打不穿“长弓阿帕奇”的防弹装甲和防弹玻璃，子弹打在直升机上迸溅起点点火星。
<br/>　　机首下方那门30毫米口径的机炮对着种植园进行射击。
<br/>　　砰！砰！砰！砰！
<br/>　　轰！轰！轰！轰！
<br/>　　机炮炮弹落到cocaine种植园中炸起一个个弹坑，每次爆炸都会炸起大片的泥土和古柯（提炼cocaine的原植物），大量的古柯叶子散落在种植园中。
<br/>　　种植园内的工人在两架武装直升机出现的时候，就惊叫着逃散开来。
<br/>　　这个种植园是布克尔特重要的cocaine原料产地，所以在这里有数名从特种部队退役的职业军人进行看守。
<br/>　　这些退役的特种部队队员有着丰富的作战经验，他们没有像其他那些武装毒贩那样拿着M4A1自动步枪去射击武装直升机，他们清楚，就是把M4A1的枪管打红了也不可能打穿“长弓阿帕奇”的防弹装甲和防弹玻璃，他们以前还在美国特种部队服役的时候，可没有少跟这种被称为武装直升机中“黄金圣斗士”的大家伙配合，所以他们很了解这种有着强悍火力的武装直升机。
<br/>　　两架“长弓阿帕奇”两侧短翼挂架上挂着八枚“地狱火”导弹和两个火箭弹发射巢，对着种植园用机炮完成一轮攻击后，火箭弹和“地狱火”导弹紧随而至。
<br/>　　布克尔特手下的武装毒贩的确是经过专业的军事训练，有着很高的军事素质，而且装备也足够精良，但是这并不意味着他们就能干得过两架武装直升机。
<br/>　　两架AH-64D“长弓阿帕奇”后面，两架“黑鹰”运输机和一架“支奴干”运输机从山的背后飞了出来，悬停在种植园上空，直升机上的联合政府军士兵开始实施索降。
<br/>　　这些联合政府军的士兵全都是比洛多手下那支精锐部队中的精锐，这个国家的联合政府军中没有特种部队的编制，但是这些士兵从单兵作战能力和单兵装备上而言已经相当于特种部队的规格。
<br/>　　“妈的，是比洛多手下的机动部队！”
<br/>　　从“第75游骑兵团”退役的派伯克?李从掩体背后探头看了眼从直升机上机降下来的联合政府军，脸色变得有些难看。
<br/>　　派伯克?李所在的连队就驻扎在美军在南美洲的军事基地，而他自己还被委派去给比洛多的机动部队做过教官，训练过他们！
<br/>　　曾经自己训练出来的士兵现在掉过头来，把枪口对准了自己，这他妈实在是太讽刺了！
<br/>　　布克尔特也对这个国家的情况有很深的了解，从一开始，他最想要拉拢的，并不是巴坦伦而是比洛多，只是多次交谈无果，才选择了巴坦伦作为自己的合作对象！
<br/>　　派伯克立刻使用步话机跟布克尔特进行联系，比洛多的机动部队单兵素质很强，靠种植园这边的守备力量根本不可能抵挡住他们的攻势，更何况他们还有两架AH-64D“长弓阿帕奇”在空中虎视眈眈。
<br/>　　正在布置村子内防御措施的布克尔特在接到派伯克从种植园传来的消息时，吃惊程度不亚于派伯克，“比洛多？比洛多为什么会参与到这中间来！？”
<br/>　　“我也想知道原因，但是我可以肯定现在出现在种植园中正在对我们攻击的，就是比洛多的机动部队，他们在南美洲的军事基地接受过训练，而我做过他们该死的教官！”
<br/>　　看到种植园中的武装人员被“地狱火”导弹炸得四分五裂，派伯克的情绪变得有些激动，他不在乎布克尔特手下那些武装毒贩会怎么样，他只在乎他自己跟他那些兄弟会怎么样。
<br/>　　布克尔特结束了跟派伯克的通讯后，握着步话机沉默了几秒钟，随后用卫星电话拨通了巴坦伦的电话，他需要向巴坦伦质问为什么比洛多会出现在自己的种植园中，为什么会跟自己公开跟自己敌对！布克尔特隐隐察觉到了事情的异样，他感觉到这件事情背后酝酿着巨大的危机！
<br/>　　可是巴坦伦秘密别墅中的电话并没有人接听！
