陆寒霆把话说完就愤然离开了。
<br/>
<br/>        夏夕绾僵在原地，她没有想到他知道她结过婚反应会这么大，什么叫接盘侠，他是接自己的盘好吧？
<br/>
<br/>        但是，她不能告诉他。
<br/>
<br/>        夏夕绾看向陆寒霆，陆寒霆已经坐在了篝火边，他整个人心情不佳，脸色阴沉的几乎能滴出水来了。
<br/>
<br/>        她身上还是很热，而且热浪一波接着一波，夏夕绾看着陆寒霆只觉得口干舌燥，再这样下去她恐怕就忍不住了。
<br/>
<br/>        夏夕绾走了出去。
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾出了山洞，这夜间的森林十分的寒冷，寒风瑟瑟的，正好缓解了一些她身上的燥热。
<br/>
<br/>        妈咪的第二个宝箱究竟在哪里？
<br/>
<br/>        这森林看起来无边无际，浩瀚又神秘，想要在这里面寻找一样东西，简直是大海里捞针。
<br/>
<br/>        夏夕绾又来到了那个万丈悬崖边，刚才她就是在这里看到了幻想，看到了妈咪的宝箱。
<br/>
<br/>        这个万丈悬崖究竟藏了什么秘密？
<br/>
<br/>        嗷呜。
<br/>
<br/>        这时狼的嚎叫声响起了，夏夕绾心头一动，整个人瞬间警觉，现在她的心口还在隐隐作痛，就是狼锋利的爪子给抓下来的。
<br/>
<br/>        夏夕绾回头看去，只见黑暗里走出了那两头狼，狼看着她，收敛了一身的兽性，格外的温顺。
<br/>
<br/>        很快夏夕绾就看到了震惊的一幕，那两头狼对着她缓缓单腿跪了下来。
<br/>
<br/>        狼给她下跪了！
<br/>
<br/>        夏夕绾澄亮的瞳仁倏然一缩，她看着此刻跪在她面前的两头狼，她能感觉到它们的虔诚和敬意，它们好像在向她行什么礼。
<br/>
<br/>        这是怎么回事？
<br/>
<br/>        夏夕绾扭头再次看向那个万丈悬崖，这时震惊的事情又发生了，万丈悬崖上面起了一层雾腾腾的仙雾，在仙雾下面波光粼粼，竟然是澈蓝的大海。
<br/>
<br/>        夏夕绾看到了此生最波澜壮阔的画面，她面前是滚滚东水流的大海，大海神秘到没有尽头，它的尽头好像是某个地方，或者是某个…国度。
<br/>
<br/>        此时森林寂静无声，两头狼王虔诚的跪在她的面前，晚风拂动着夏夕绾乌黑的长发，她整个人飘飘欲仙。
<br/>
<br/>        这时耳畔传来了一道低沉而焦急的嗓音，“夏夕绾！夏夕绾！”
<br/>
<br/>        夏夕绾迅速回神，是陆寒霆的声音，他找来了！
<br/>
<br/>        夏夕绾看向那两头狼，狼王已经起身，甩着尾巴离开了，它们很快就没入了黑暗里，不见踪迹。
<br/>
<br/>        夏夕绾又看向前方的大海，大海也消失了，再次变成了万丈悬崖，好像刚才的一切不过就是幻象，是她眼花了。
<br/>
<br/>        夏夕绾心里已经震撼到了极点，今天她所看到的都无法用常理来解释，她必须回去翻阅典籍，寻找蛛丝马迹。
<br/>
<br/>        夏夕绾！
<br/>
<br/>        夏夕绾！
<br/>
<br/>        陆寒霆的声音越来越近了。
<br/>
<br/>        夏夕绾迅速转身，去找陆寒霆。
<br/>
<br/>        很快她就看到了陆寒霆，陆寒霆已经出了山洞，他不停叫唤着她的名字，正在四处寻找她。
<br/>
<br/>        夏夕绾看到了他紧锁的眉心，已经成了一道“川”字，他整个人看着阴郁又冷鹜，但更多的是对她的紧张和关心。
