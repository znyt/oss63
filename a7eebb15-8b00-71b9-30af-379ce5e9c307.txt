<br/>　　结束了？
<br/>　　这就结束了？
<br/>　　墨狼败了？秒败？
<br/>　　所有人都愣住了，所有人都反应不过来，实在是这一切发生的太快了！
<br/>　　战斗还没开始，被认为必胜的墨狼，就这么直接倒了下去。

<br/>　　“这是怎么回事？”
<br/>　　“好诡异，这是战魂之力吗？”
<br/>　　“这怎么可能？武帝境八重修为瞬间秒败墨狼，这太恐怖了吧？”
<br/>　　嘈杂的声音此起彼伏，喧嚣震天，整个武斗场都变得一片火热。
<br/>　　所有人都震惊了，这是众人从未想到过的结局，出乎意料的结局。

<br/>　　以武帝境八重修为，秒败墨狼，这是什么概念？
<br/>　　就算是妖孽天才，也做不多吧！
<br/>　　而且，这苏莫的能力，居然如此的诡异，那紫雾是战魂吗？又是什么战魂？
<br/>　　此时，最为震惊的人，莫过于一众道宗的弟子和长老了。
<br/>　　冯光肥胖的脸庞，脸色非常的精彩，犹如见了鬼一般，他从未想过苏莫有这种恐怖的实力
<br/>　　吕罡满脸呆滞之色，心中的震撼无以复加，此时，他不由得想到了上次之宗务堂之时，苏莫说要割了他的舌头。
<br/>　　当时，他觉得苏莫很可笑，但现在，他发现这并不可笑。
<br/>　　脸色最为精彩的人，莫过于真枯长老了，他先是一呆，随即满脸惊愕，最后是露出了大喜之色。
<br/>　　苏莫居然如此的强大？
<br/>　　他居然看走眼了？
<br/>　　不由得，真枯半圣想到了真贤道圣，对方这是猜到了会武的情况，而让苏莫来此助阵的吗？
<br/>　　这苏莫如此强大，瞬间就能击败了墨狼这种天才，实力比之扬天成、炎澜天等人，也不会弱多少吧！
<br/>　　有了苏莫之助，他道宗不说能翻盘，获得一个前十名额，还是不成问题的。
<br/>　　如此的话，这次会武，道宗也不至于丢脸了！
<br/>　　有人欢喜有人忧，啸天狼族的人，个个脸色难看了起来，墨狼的惨败，让得他们大感脸上无光。
<br/>　　毕竟，墨狼是败在了一名武帝境八重的武者手中。
<br/>　　“有点意思！”幽狼大嘴一咧，反而露出了一丝感兴趣的笑容。
<br/>　　此刻，不仅是幽狼来了兴趣，炎澜天、扬天成、灵枫、卓剑豪、雷云皓等各大势力的妖孽人物，都对苏莫产生了兴趣。
<br/>　　因为他们不知道墨狼是如何败的，苏莫那紫雾到底是何种攻击，这种诡异的攻击手段，他他们非常好奇。
<br/>　　北方看台上。阴泽长老微微颌首，面上露出了一丝笑容，果然，此子不凡，而且是相当不凡啊。

<br/>　　“这……这人隐藏的也太深吧！”绿衣女子灵萱，面带一丝尴尬之色，他本以为苏莫如此做，是想要引起云悠悠的注意，本想到真的是妖孽啊！
<br/>　　“的确，他的气息一般，真的很难看出来！”云悠悠点了点头。
<br/>　　事实上，这也是因为苏莫体内玄力没有融合，所以气息倒是没有多强大，外表很难看出真实实力。

<br/>　　战台上，苏莫瞥了一眼倒下的墨狼，微微的摇了摇头，太弱了，不堪一击。
<br/>　　在梦魔空间之中，他只是折磨了对方半个时辰，就将对方折磨的崩溃了。
<br/>　　不过，让他郁闷的事，或许是因为神魂受损，精神力减弱的原因，动用了梦魔战魂之后，他感觉精神非常的疲惫。
<br/>　　看来接下来的战斗，还是不用梦魔战魂这种消耗精神力的本领了。

<br/>　　墨狼已经被他折磨的精神奔溃，肯定无法回去了，只能让人抬回去了。
