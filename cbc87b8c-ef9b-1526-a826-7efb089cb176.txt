　　玉虚宫中，悟道台上，神兽飞舞。
<br/>　　直至良久，四大神兽才停了下来，分别落于元始的四周。
<br/>　　元始运转功法，《玄黄不灭》第三重四象化四极，原本就已经小成的第三重，现在随着四象神兽本源精血的出现，变得更加雄浑了。
<br/>　　四大神兽分别化为四滴精血，钻入元始四肢之中。顿时，元始四肢变得巨大，如若天地四极，地水火风涌动，四滴精血再次化为神兽本体，镇压地水火风。
<br/>　　神兽轰鸣，元始四肢分别发出黄、黑、赤、青四种颜色，四道光源在缓缓凝聚，各自凝聚成一个旋涡黑洞，吸收着这天地之中的地水火风之力。
<br/>　　天地灵气狂涌而来，与四象之力一同汇聚至黑洞，凝聚四象之力，再于四肢化天地四极，弥散到全身各处，增加着整个真身的强度与属性。
<br/>　　轰！
<br/>　　这种状态不知过了过久，伴随着轰的一声，元始全身气息大涨，四肢发亮，如同四道天柱，支撑整个天地！
<br/>　　《玄黄不灭》第三重，四象化四极大成，四象真身，成！
<br/>　　但，元始并没有睁开双眼，而是继续闭目参悟，任由身上四象之力流转，四大神兽之力环绕。
<br/>　　直至良久，元始睁开双眼，眼中闪过神兽飞腾，地水火风暴乱之象，宇宙生灭，尽在其中。
<br/>　　元始双手捏大印，顿时，空中出现一黑白赤青的四色大印，印有四角，每一角各有一神兽占领，青龙、白虎、朱雀、玄武，各自身上都有四象之力扩散。
<br/>　　一印轰出，四色大印快速旋转，道光顿生，化为四大神兽奔啸而出。
<br/>　　白色化为白虎神兽，虎啸生风，庚金之气弥漫，化为巨剑，斩破苍穹！
<br/>　　黑色化为玄武神兽，四肢立地，上首顶天，玄冥之气环绕，化为大江大河，淹没天地。
<br/>　　青色化为青龙神兽，腾云驾雾，木气弥漫，能隐能升，能大能小，苍龙抬头，天崩地裂！
<br/>　　赤色化为朱雀神兽，神火沸腾，鸟鸣天惊，化为大火，焚烧苍穹！
<br/>　　四大神兽，各有其能，焚天战地，乾坤倒转！
<br/>　　“四象印！”
<br/>　　“还不错！”元始赞叹一声，心念一动，四大神兽顿时化为四道流光分别钻入元始四肢之中，镇压四极，再生造化。
<br/>　　“嗯？有人来了？是昆吾老祖？”元始心念一动，眼中闪过精芒，看向阵外，便知道大阵之外，有人来了，顿时皱起了眉头。
<br/>　　“也罢，既然回来了，那本座今天就来会会你，试试我这新创的四象印的威力如何？”元始眉头舒展，心念一动，人便来到了阵外。
<br/>　　元始走出阵外，几乎同时，将目光投向不远处，只因目光所向，那山谷后飞来一个人影，由远及近，脚步虽然缓慢，却如行云流水一般，眨眼间，便到了近前。
<br/>　　那人身穿白色道袍，背负金色长剑，目光凌厉逼人，一头白发随风飘散。看见元始，目光直视于他，咄咄逼人，横扫元始，道：
<br/>　　“你是何人，为何出现在贫道洞府之外？是否为夺贫道洞府之人？识相的快快道来，不然休怪贫道手下不留情，让你灰飞烟灭！”
<br/>　　此时，元始身上并无太大的法力波动，来人正是算出自己洞府被抢，从阴阳老祖洞府赶来的昆吾老祖，如今见一个人鬼鬼祟祟出现在自己洞府之前，而自己的洞府却被抢了，要是两者之间没有关系，他可不相信。
<br/>　　昆吾老祖在打量着元始，元始同样也在打量昆吾老祖，元始见昆吾老祖一身法力雄厚无比，比自己也不遑多让了，而且他身上气势凌厉逼人，霸道无比，背上之剑，剑意森寒。
<br/>　　元始知道此人虽然只有大罗金仙圆满，但一身实力，无论是法力积累，还是对于大道的领悟，比起那龙祖，风祖等人强了不止一倍。对于这种人，不能以等闲对之。
<br/>　　“你就是昆吾老祖？本座元始，至于为何本座会出现在你的洞府之外，呵呵，那是因为——”突然，元始大喝道：“从今以后，此处就是本座的洞府了！”
<br/>　　元始知道，就算自己说不是，眼前此人也不会相信，毕竟大家都不是傻瓜，自己还不如爽快承认呢。况且，元始从来就不怕他。
<br/>　　“哼！”
<br/>　　昆吾老祖听到此话，顿时大怒，冷哼一声，带有几分张狂，怒而反笑道：“哈哈哈！好！够狂！既然你承认了就好，敢抢贫道洞府，就要有身死道消的领悟。”
<br/>　　话音刚落，只见他猛然抬手，霎时，红光灿灿，当空怒舞，化作一只硕大无比的手掌，呼啸刺耳，径直朝着元始抓了下来。
<br/>　　“哈！”
<br/>　　见状，元始不惊反喜，口中朗笑道：“好，本座今天就看看你昆吾老祖到底是否真如传说中那般厉害！”
<br/>　　长笑间，只见他微微抬手，大手向着天上太阳猛然一抓，顿时，无边太阳之火从天而降，，被元始抓在手中，形成一颗小型太阳。
<br/>　　随即，元始屈指一弹，这小太阳便落到了昆吾老祖变幻的大手上。
<br/>　　轰隆！
<br/>　　无边巨响传来，先前的大手早已不见了，而昆吾老祖也不再试探元始了，经过初步的交手，虽然只是简单的交手，但双方对于各自的法力手段都有了一定了解。
<br/>　　昆吾老祖手捏剑诀，昆吾神剑化作一柄巨大神剑，横亘于高空。
<br/>　　一剑变两剑，两剑变四剑，四剑变八剑……
<br/>　　眨眼间，九天之上，便出现了亿万神剑，布满了密密麻麻的金色巨剑。
<br/>　　巨剑遥指元始，剑意锋芒，无坚不摧，冲霄而起，化作剑阵，一同冲向元始，势要将他切为粉碎。
<br/>　　“嗯？万剑归宗？有点意思。”元始看到这一幕，顿时就想起了前世看到的小说中看到的万剑归宗，不过，这剑招可比那什么万剑归宗强了不止千万倍。
<br/>　　元始不慌不忙，伸出大手，对着朝自己纷乱飞来的神剑剑阵遥遥一指。
<br/>　　顿时，天地寂静无声，天空化为一道镜面，神剑也静止不动，之后，镜面轰然破碎，亿万神剑便化作无尽碎片，散落一地，天空只留下一柄金色神剑。
<br/>　　寂灭指！
<br/>　　“哼！”
<br/>　　昆吾老祖见此，冷哼一声：“有点能耐，难怪能偷偷闯入贫道洞府，竟然如此，那就再接贫道一招！”
<br/>　　“九天——剑界！”
<br/>　　随着昆吾老祖一声大吼，他手持金色神剑，持剑向天，一件斩下，剑气开元，幻化天地，天地皆剑，剑山，剑湖，剑草，剑木……
<br/>　　剑气凌然，剑意森然，剑威煌煌，剑网遍布整个剑道天地。
<br/>　　这完全是一个由剑组成的世界！
<br/>　　一剑生世界，显然昆吾老祖已经将剑道推演到了剑道巅峰。
