        晚霞好像没有认出她来……
<br/>
<br/>        童筱颖跟在了晚霞的身后，只听着晚霞谆谆告诫：“阿英，你记住了。八爷今天还带了四爷一起来，今天八爷做东，四爷是客。所以，你得什么话都要听八爷的。八爷要你做什么，你就做什么。”
<br/>
<br/>        童筱颖聋拉着脑袋，默不吭声。
<br/>
<br/>        晚霞身心一怔，突然停下脚步，回头瞥了童筱颖一眼，纳闷地问道：“你今天这是怎么呢？脸色这么差！没化妆吗？”
<br/>
<br/>        “……”童筱颖不敢出声，继续低着头不说话。
<br/>
<br/>        晚霞似乎并未多心，转过头去，接着说道：“算了，你就算身体不舒服，今天也得去伺候八爷。”她一边说着，一边把对讲机挂在童筱颖的脖子上，补充了一句，“有什么事，随时联系我，记住，不要把里面的爷给得罪了。”
<br/>
<br/>        包厢门一开，童筱颖便被晚霞给推了进去。
<br/>
<br/>        童筱颖打了个趔趄，本能地扶着墙壁，站稳了身子。
<br/>
<br/>        偌大的包厢里开着暖色调暗沉的橘色的灯光，香烟的烟雾迷蒙缭绕，茶几上觥筹交错，沙发上坐着四个男人，旁边还站着四个黑衣保镖，牵着一条凶神恶煞的狼犬。
<br/>
<br/>        而茶几前的地板上却跪着八个衣着暴露，大胸长腿，打扮妖娆的女人。
<br/>
<br/>        “你们一个个给我徒手把我的名表从那缸里拿出来，我就给你们每人一万块的现金。”
<br/>
<br/>        说这话的人，就是八爷。
<br/>
<br/>        童筱颖一眼就认出了他。
<br/>
<br/>        那次她被他绑架到那座古堡里，虽然未看全他的面貌，但是让她记住了他的样子以及他的声音。
<br/>
<br/>        跪在地上的女人，一个个瑟瑟发抖。
<br/>
<br/>        因为，那玻璃缸的底部，全是蜈蚣覆盖在那块名表上。要想拿到名表，就得徒手把蜈蚣拨开。这光看着就令人心里发麻，鸡皮疙瘩掉了一地。
<br/>
<br/>        “不想徒手拿名表，那就伺候我的‘二郎神’。我家这小二，这几天正处于发情期。谁把我家这小二伺候好了，我就给谁三万。”八爷抖了抖手里的香烟，又放到唇上吸了口，将烟圈从嘴里吐了出来，“你们二选一，若是都不选，就那你们去做人体实验好了。”
<br/>
<br/>        这个八爷，果然是个变态！
<br/>
<br/>        而就在这无意间，童筱颖注意到沙发上隐匿在光线最昏暗处的那个男人。
<br/>
<br/>        他坐在八爷的身旁，虽然她看不清他的样貌，但他与另外三个男人的气质截然不同。
<br/>
<br/>        这个男人的气场森冷，对着包厢里的事情好像漠不关心，丝毫不感兴趣。
<br/>
<br/>        除了八爷以外的另外两个男人，都咧着嘴，热血沸腾地等着看这八个女人的“生死抉择”。
<br/>
<br/>        第一个女人选择了去徒手拿名表，结果名表还未拿到手，手指就被蜈蚣给咬住，吓得她嗷嗷大哭起来。
<br/>
<br/>        有了前车之鉴，第二个女人则脱了衣服，选择了去伺候八爷的“二郎神”，也就是那只大狼犬。
<br/>
<br/>        八爷还特意让保镖给那狼犬喂了药，让狼犬处于兴奋状态。
<br/>
<br/>        看到这里，童筱颖只觉胃里在翻滚，扶着墙，捂着嘴干呕了起来。
<br/>
<br/>        “阿英，别在那儿傻站着，过来伺候。”八爷这个时候才注意到在门口的墙边站着的童筱颖。
<br/>
<br/>        童筱颖踏着沉重的步伐，缓慢地挪步到这个八爷面前。
<br/>
<br/>        她刚走到他的身前，他便伸腿将她撂倒。
<br/>
<br/>        童筱颖没来得及稳住自己的身子，一个趔趄，双膝跪地，紧接颈项突然被一只手掐住，一道力度压迫着她的脑袋往前倾去。
<br/>
<br/>        坐在八爷身旁的这个男人，刚好敞着双腿。
<br/>
<br/>        男人坐着，她跪着。
<br/>
<br/>        八爷一直按着童筱颖的脖子，让她保持着这个尴尬的姿势，打趣地对这男人说道：“阿四，五年不见，今天这个见面礼，喜欢吗？这女人叫阿英，是个聋子，但是会唱歌。今晚，让阿英给你唱摇篮曲听！”
<br/>
<br/>        听到这里，童筱颖这才恍然大悟，为什么起初晚霞姐对她的身份未起疑心的原因，原来是她两耳上这助听器的缘故。
<br/>
<br/>        只是，现在的这个姿势，令她很不舒服。
<br/>
<br/>        “小八的美意，我就不客气了！”男人挥开八爷按在童筱颖脖子上的手，然后抓着童筱颖的手臂，将她从地上拉起来，抱坐在自己的大腿上。
<br/>
<br/>        童筱颖惊怔地看着面前这个抱着自己的男人。
<br/>
<br/>        刚刚，他的声音……
<br/>
<br/>        如此近距离的面对面，童筱颖抬眸看清了他的长相，顿时犹如五雷轰顶，惊愕的目光再也难以移开。
