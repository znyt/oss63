　　数道剑光打出的一刻，楚风眠也是在观察着这一道人影，在想着对策。
<br/>　　楚风眠也没有想到，在这天元商会遗迹之中还能碰到其他人。
<br/>　　毕竟天元商会，可是毁灭了足足十万年了，十万年的时间，里面居然还有人的存在？
<br/>　　剑光打出的一刻，从天涯剑匣之中两道剑光飞了出来，飞到了楚风眠的手心之中。
<br/>　　祖龙至尊剑，巨阙神剑，双剑齐出，被楚风眠紧紧捏在手中，于此同时剑神境的剑意，也是从楚风眠的背后涌现而出，冲天而上。
<br/>　　这人影的实力，足以是徒手打破始祖第一式，绝对是九劫古帝巅峰级别的强者。
<br/>　　甚至极有可能，此人都是半步天人境的强者，面对这样的强者，楚风眠不能有任何的保留。
<br/>　　数道剑光，对着那人影当即斩杀过去。
<br/>　　那人影面对这数道剑光的袭来，毫不逼退，手中拳风当即轰击而下，猛然之间，剑光全数崩灭。
<br/>　　“这？此人难道是一个疯子吗？这是什么实力？”
<br/>　　徒手硬接天涯剑匣的剑光？楚风眠都没有想到这人影敢这么做。
<br/>　　楚风眠诧异的瞬间，这人影已经是逼近了过来，楚风眠手持双剑，当即迎了上去。
<br/>　　天龙变！
<br/>　　在这一道拳风接近到楚风眠胸口的一刻，楚风眠突然将始祖天龙的力量全然催动起来。
<br/>　　他的身形瞬间消失，移开了三米之远，躲开了那一拳。
<br/>　　于此同时，剑神境的剑意在剑锋之上凝聚，猛然一剑打出。
<br/>　　嘶拉！
<br/>　　这一剑斩在了这人影的头顶上，将他的长发，削去一部分，也让楚风眠突然看清楚了这人的样子。
<br/>　　嘶！
<br/>　　这人的脸庞，已经是呈现出了一种剧毒的紫色，仿佛是浑身上下，都已经是被剧毒侵蚀了。
<br/>　　只要是站在他的身边，就会无时无刻受到剧毒的侵蚀，幸好是楚风眠以吞天之力，在加上不朽真身，才足以抵挡。
<br/>　　不然的话，半步天人境的人物，也不可能足以跟此人交手。
<br/>　　而且楚风眠也注意到了，这人影出手的时候，动用的只有蛮力，以纯粹的蛮力，就抵挡住了数道剑光。
<br/>　　同时这人的出手，也毫无章法。
<br/>　　“这人，难道不是一个活人？”
<br/>　　楚风眠眼神之中露出几分异样，这人影出手的方式，的确是跟活人不同，这种纯粹蛮力的打发，简直是就连一名最弱的武者都不如。
<br/>　　简直是纯属凭借本能在战斗。
<br/>　　这种打法，纵然是一身再强的实力，也发挥不出两三成来。
<br/>　　眼前这人影的真正实力，应该是已经的达到了半步天人境的层次，甚至是更高，可能是距离天人境，只差一步之遥。
<br/>　　但是他不懂武技，不懂武道，爆发出的威力，也就是在九劫古帝的级别。
<br/>　　“多半是此人修炼了什么毒功，但是失败了，以至于被走火入魔，成了毒尸一样的存在！”
<br/>　　楚风眠看了一眼这人影，已经是猜测了出来，这人到底是发生了什么。
<br/>　　走火入魔的强者，并不少见，曾经九域之中便是就有着许多的强者，在修炼的最后一步，走火入魔，以至于成为了只知道杀戮的怪物。
<br/>　　这人影也多半如此，只是他修炼的是一种毒功，被毒功所反噬，占据了身躯，最终成了这样一个毒尸。
<br/>　　虽然身躯还保留着几分生机，但是这人，已经是跟死了无异，只是没有意识，只能够凭借本能战斗的毒尸。
<br/>　　“只有蛮力的话，杀了此人，应该不难！”
<br/>　　楚风眠目光闪烁，一名半步天人境的强者，想要捏死楚风眠，简直是轻轻松松，但是如果只有纯粹的蛮力，还不懂章法的话。
<br/>　　眼前这毒尸的力量，也许还不如九劫古帝难以对付。
<br/>　　楚风眠身躯连连闪动，躲开了这毒尸的无数轰击，同时手中剑锋，也没有停下，直接刺向了这毒尸的身躯。
<br/>　　连连几剑，都刺入到了这毒尸之中。
<br/>　　只是划破这毒尸的身躯，留下来的，却不是鲜血，而是紫色的毒血，这些毒血蕴含的剧毒，比起周围的毒雾还要强得多。
<br/>　　一滴毒血，滴在地上去，居然是将主殿之内，都给腐蚀出了一个巨大的缺口。
<br/>　　这主殿，可是一件极品极高的天级灵器，不亚于云羽天宫那个级别的灵器，可是依然无法抵挡这毒血的侵蚀。
<br/>　　楚风眠也是急忙拉开了一段距离，这毒血，楚风眠沾染上一点，哪怕是他的半步不朽真身，也未必抵挡得住。
<br/>　　“吼！”
<br/>　　这毒尸不知疼痛，也不知疲倦，只懂得杀戮，看着楚风眠后退，直接冲了过去，一拳一拳的轰击而来。
<br/>　　这毒尸纯粹的肉身蛮力，速度都快到极致，足以比拟九劫古帝巅峰的人物。
<br/>　　但是他的出手，实在是太简单了，楚风眠几乎是看到这毒尸出手的瞬间，就足以猜出他的目标。
<br/>　　楚风眠身躯连连闪动，以至于足以做出完美的应对。
<br/>　　纵然是速度再快，也打不中楚风眠的身躯，而楚风眠却是在这毒尸每一次逼近他的时候，手中剑锋都会连连斩出，在这毒尸上打出几个缺口。
<br/>　　“唰！”
<br/>　　又是一剑剑锋，直接斩在了这毒尸的胸口之上。
<br/>　　这毒尸身上破旧的衣衫，被当即斩开，在这毒尸的背后，大大的两个天元二字，突然露了出来。
<br/>　　在这天元二字之上，还刻画这一条战龙，盘旋在其上。
<br/>　　天元盘龙？此人竟然是天元商会的一任会长？
<br/>　　天元盘龙的标致，只有是天元商会的会长，才有资格拥有，楚风眠本以为这毒尸应该是天元商会之内的一位强者，可没想到居然是天元商会的会长。
<br/>　　在想起九龙神苍鼎上面还存在的烙印，楚风眠一下子就明白了过来。
<br/>　　这毒尸就是天元商会的末代会长，也就是因为他修炼毒功，走火入魔，最终是导致了天元商会的毁灭。
<br/>　　怪不得天元商会的毁灭，众说纷纭，却是没有一个真正的结果，原来不是外敌入侵，而是自己毁灭的。
<br/>　　这天元商会的末代会长，在天元商会内修炼毒功，只怕是一下子爆发而出，其他天元商会的弟子，都没有反应过来，都被毒雾所杀，最终导致一代势力的毁灭。
