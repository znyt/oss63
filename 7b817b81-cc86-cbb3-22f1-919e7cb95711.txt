　　第639章
<br/>　　她穿了一身绣绒花袄，模样不过十一二岁，现在身上湿漉漉的，脸上还挂着泥泞水草，模样其实十分狼狈。但哪怕是这样打扮，也是粉雕玉砌的惹人怜爱。
<br/>　　程大雷开始阅读她的信息。
<br/>　　姓名：杨小妹
<br/>　　年龄：12
<br/>　　技能：无
<br/>　　隐藏属性：无
<br/>　　此子尚未成年，目前能阅读到的信息也仅仅就是这些。至于隐藏属性和技能有可能随着成长获得。程大雷心里也有些含糊，不知她是否是那位传说中的人物。
<br/>　　如果真是那位，潜质至少当是传说级，有可能比刘关张的潜质更高一些。
<br/>　　程大雷心中念头闪烁，一时没回过身来，倒是小女孩款款道：“我与家兄乘船寻亲，途中遭遇强人才沦落到此地，幸得先生相救，多谢先生的救命之恩。”
<br/>　　“嗯，好呀，好呀。”
<br/>　　程大雷虚伪的应承两声，问明白了二人的来历。这杨国忠兄妹乃是荆州人士，出身也是豪门世家，不过后来家道中落，才渐渐落魄。这次是从京州到凉州，寻几位亲人。
<br/>　　程大雷也知这是系统安排的人物背景，倒也没有深究。他心里琢磨这杨国忠是自己召唤来的，当然要归附自己，于是开门见山道：“某家正是蛤蟆城的程大雷，既然你们走投无路，我愿收留你们二人在我麾下，你们可愿意？”
<br/>　　不听这话还好，一听这话，杨国忠一哆嗦差点又跪倒在地。就连刚才还镇定自若的杨小妹也脸露慌张。
<br/>　　“怎么，你们不愿意么？”
<br/>　　“这个……那个……”
<br/>　　杨国忠吞吞吐吐说不出一句完整的话，但对程大雷嫌弃的意思十分明显。程大雷莫名有些恼怒，什么意思，你一个反面角色凭什么看不起我，难道我的名声真已经糟糕到这种地步了么。
<br/>　　“程当家勿怒，先生于我有救命之情，本该粉身碎骨绝无二话。不过，我的父母都死在强人的手中，杨某虽然无能，却也知道为父母报仇的道理。等我杀了仇人报仇，再投靠在程当家麾下效力。”
<br/>　　“你们说的那伙强人是什么来历？”程大雷问。
<br/>　　杨国忠眨眨眼睛，道：“听他们说话，当是黑风寨的山贼……”
<br/>　　“哇哈哈……这可再好也没有了，今次我就是找黑风寨麻烦，等我平了黑风寨的贼人，岂不就等于为你父母报仇了，你们正好投靠我。”
<br/>　　杨国忠面露苦色，他这番话只是托词，他的父母早已病故，哪里是死在山贼手中。从他本心出发，他并不愿意投靠程大雷，做山贼就是给祖宗蒙羞的事，何况是投靠程大雷。
<br/>　　谁知话说出口，正好被程大雷截住。如果自己再拖拖拉拉，很明显程大雷不是一个好说话的主儿。
<br/>　　程大雷瞄了杨国忠一眼，确实也不知道他心中想些什么，关键是程大雷知道自己名声糟糕，但毕竟不清楚有多糟糕。这杨国忠不过二十出头，已经是绝世人物，程大雷对他印象也是不错。
<br/>　　有了他的投靠，山寨必然多一助力，程大雷心中高兴，与大队汇合后，继续向黑风寨进发。
<br/>　　……
<br/>　　一水绕山而过，夜晚发出哗哗的水声，水边就是黑风寨。山寨三千多弟兄，老大混江龙更是恶名在外，方圆百里无人敢惹。
<br/>　　话是如此说，但委实也是方圆百里没有什么势力。而把目光放到百里之外，便有京州禁卫军，凉州宋伯康，还有个程大雷。如此，混江龙便不敢太猖狂了。
<br/>　　而他也有他的烦心事，麾下三千人无论如何也是一股不小的势力，免不得就要遭到其他人的忌惮。已经有数位诸侯有意招揽他，凉州、京州、荆州都有人过来谈这件事。
<br/>　　今夜，混江龙与军师在屋中谈这件事。
<br/>　　“那凉州王已派人屡次三番前来，若我们再不随他的意，怕他便要兴兵来讨伐。”军师名唤周三两，有个外号唤作绿头蝇，平日里舞文弄墨被混江龙引为智囊。
<br/>　　“官家官家，咱们这条命迟早都是官家的。”混江龙叹道：“可是，那宋伯康心胸狭窄，未必容得下咱们兄弟，咱们真归顺了他，谁知道前程如何。”
<br/>　　周三两也是无语长叹，山贼这一行其实不好混，世道太好了不行，官家不时来剿一次，变得东躲西藏。世道太差了也不行，世道太差了，也没人可以抢呐。
<br/>　　混江龙在山贼这一行还是有些天赋的，靠着心狠手辣四个字，百里之内被他搞得没有人烟。刮地三尺，山寨里搜刮了一批粮食。可这相当于涸泽而渔焚林而猎，现在周围已没什么人好抢。
<br/>　　就算归顺官家，也并非坦途，在招安之前，自然许下各种条件。可一旦成了他人的兵，天下有几个人能看得起山贼。
<br/>　　这便是混江龙如今烦心的事，面前两条路，那条路也不好走。最后他狠狠一咬牙：“招安招安，招他娘的鸟安，宋伯康真要动兵，我倒看他有没有本事拿下我这水寨。”
<br/>　　说这话，混江龙是有自信心的。黑风寨依水而立，山寨中多备陷阱弩箭，而且，还有一条直通混江龙卧房的暗道，实在守不住的时候，他还可以逃生。
<br/>　　做出决定后，混江龙倒是平静下来。仔细想想，还是做山贼痛快些，大碗喝酒大碗吃肉，看上那家婆姨抢过来便是，不用受谁的窝囊气。
<br/>　　这混江龙本是军中的一员小卒，因为得罪了上官，才逃出军营拉起一支人马聚义。委实是过了几年痛快日子，而他在男女之事上又太随便些，现在山寨还有七八位压寨夫人，都是从周边抢来的女子。从开始到现在，他不知掳过多少女子，有些性情刚硬的直接碰壁死了，有些是被他摧残至死。现在还活着的几位，要么是被他糟蹋过后发现新的人生，甘心做他的压寨夫人。要么就是既不愿意忍受现在生活，也不敢寻死，只能捱着现在的日子。
<br/>　　阿喜打探到的情报，只是说混江龙此人好色，对于他的生活作风程大雷当真不是很了解。
<br/>　　不然的话，程大雷该发现，山贼当还有另一种人生。
<br/>(本章完)
