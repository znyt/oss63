　　楚风眠的境界先不提。
<br/>　　单单是如今楚风眠展现出来的实力，已经是达到了内门弟子的范畴了。
<br/>　　能够击败王林，这种实力放在内门，也算得上是在中游了，不会垫底。
<br/>　　对一个新人来说，拥有这样的实力，的确是已经极为不可思议了。
<br/>　　这样的实力，楚风眠居然是没有在收徒大会上被一些长老挑选中，也的确是令人惊讶了。
<br/>　　唯一的解释，只有是楚风眠，自己不想要去争取，不然的话，楚风眠表现出实力来，绝对会被一些宗门长老看重，收为弟子的。
<br/>　　楚风眠看着白萱，并没有回答她的话，反问道。
<br/>　　“白萱师姐今日找我来，可不是来问我这种事的吧。”
<br/>　　“自然，我有件事，本来是想要去跟寒月漓说的，不过看你如今的实力，似乎却是多了一个选择。”
<br/>　　白萱看着楚风眠，目光闪烁，突然开口道。
<br/>　　“在我的手中，掌握这一处远古洞府，快要开启的消息，不知道楚师弟，有没有兴趣一起前去？”
<br/>　　“远古洞府？快要开启？”
<br/>　　楚风眠思考了一下，直接敞开天窗说亮话道。
<br/>　　“这种好事，白萱师姐不会是想要跟我分享吧。”
<br/>　　远古洞府，这便是九域中最大的奇遇之一，楚风眠曾经便是前往过那丰饶大帝的洞府，从中得到了无数好处。
<br/>　　这种远古洞府，尤其是即将开启的远古洞府，一旦发现，进入其中，都会得到数不清的好处。
<br/>　　这种事，向来可是不会外传的，如今白萱居然是叫楚风眠一起去，这让楚风眠的脸上，都露出了几分惊异。
<br/>　　“好事？这洞府的确是好事，不过却也没那么简单。”
<br/>　　白萱听着楚风眠的话，急忙的解释道。
<br/>　　“这一处洞府，实际上并非是我一个人知晓的，是我跟学院中的几尊内门弟子，一同探索得到的消息。”
<br/>　　“不过这洞府，可是危险重重，以我们的实力，还不能够探索，所以才会寻找一些人，一同探索。”
<br/>　　白萱仔细的打量了楚风眠一眼，开口道。
<br/>　　“本来我是打算，去找那寒月漓，一起探索洞府的，但是我跟她的关系并不熟，所以偶然碰见你，就是想要卖你一个人情，好去说服寒月漓一同探索洞府。”
<br/>　　“不过现在看来，似乎你，也是一个不错的选择。”
<br/>　　白萱的话，倒是毫不掩饰，直接开口，说明了一切。
<br/>　　“是谁的洞府？既然你已经明白了其中的危险，为何要来找我？”
<br/>　　楚风眠随意笑了笑开口道。
<br/>　　“以我的实力，似乎并不足以跟你们一同探索吧。”
<br/>　　白萱可是地榜前十的弟子，能够跟他一同探索洞府的，至少也要都是地榜前十的内门弟子。
<br/>　　楚风眠这样一个外门弟子，就算是有着几分实力，也不至于会被白萱如此看重才对。
<br/>　　“不，这洞府说起来倒是跟你，也许有着几分关系，所以我才会突然改变主意来邀请你。”
<br/>　　白萱开口解释道。
<br/>　　“据我查清楚，这洞府，应该是颜古龙帝的洞府，在其中，有着无数的妖兽，其中更是蕴含了恐怖龙威，足以压制我们的实力。”
<br/>　　“你所修行的功法，正是龙族的功法，身上蕴含真龙血脉，你进入其中，也许不会被龙威压制，所以我才考虑，来邀请你去参加。”
<br/>　　颜古龙帝？
<br/>　　听到这个名字，楚风眠一愣，随后心中也是泛起了滔天巨浪。
<br/>　　这颜古龙帝，可不是什么小人物，相反，这可是五万年前的一尊叱咤风云的龙皇，在真龙一族中，地位崇高。
<br/>　　只怕是现在整个北邙学院，都找不出一个足以跟当年颜古龙帝抗衡的人物。
<br/>　　如今居然是他的洞府，现世了。
<br/>　　楚风眠不由的开口询问道。
<br/>　　“真的是颜古龙帝的洞府？像是颜古龙帝这样的存在，就算是陨落，他的洞府也应该会隐藏在虚空深处才对，怎么会被人发现？”
<br/>　　白萱缓缓的开口道。
<br/>　　“这颜古龙帝的洞府，其实是我们是虚空海中发现的，应该不假。”
<br/>　　虚空海。
<br/>　　这虚空海，楚风眠也曾经听说过。
<br/>　　这是在北域之中，一处极为古怪的地方，身处于无尽虚空之中，其中却是充满了虚空乱流。
<br/>　　许多在虚空中迷失的人物，都会被随着虚空乱流，带入到虚空海中，是一片极为混乱的地域。
<br/>　　不过因为这个特性，在虚空之中隐藏的许多洞府，也有很多，会被冲入到虚空还中去。
<br/>　　所以虚空海，虽然极不稳定，危险重重，却也是有着不少武者，会选择前去探秘。
<br/>　　这颜古龙帝的洞府，若是在虚空海中现身的话，多半不是假的了。
<br/>　　不过就算是有可能是假的，这样的机会，也决不能放过。
<br/>　　毕竟颜古龙帝的洞府中，只要是稍微寻找到一些好处，便是真正的大机缘。
<br/>　　“你觉得怎么样？”
<br/>　　白萱看着楚风眠沉默，也清楚楚风眠自己是在思考，许久之后，才会开口道。
<br/>　　“若是你不愿意，也无妨，我也会去找其他人的。”
<br/>　　“可以，我答应跟你一齐去虚空海，探索那颜古龙帝的洞府。”
<br/>　　楚风眠点了点头道。
<br/>　　“不过在此之前，我有一个要求。”
<br/>　　“但说无妨。”
<br/>　　看到楚风眠答应下来，白萱的脸上露出几分笑容来道。
<br/>　　“只要是不太过分的，我都可以答应你。”
<br/>　　“我需要借一万功勋值。”
<br/>　　楚风眠看着白萱，缓缓的开口道。
<br/>　　“等到从虚空海回来，我便还给你。”
<br/>　　功勋值，除了从学院中完成任务，也可以拿财富去兑换。
<br/>　　但是能够兑换功勋值的财富，却是只有天地奇物，楚风眠的身上，虽然宝物众多。
<br/>　　丰饶碑，人皇罐，天元神丹，龙丹，这些东西随意去兑换一个，都可以换取数万的功勋值。
<br/>　　可是这些东西，对楚风眠却有着大用，一般在外的财富九鼎丹，在北邙学院中，却是没什么用处。
<br/>　　这也就意味着，楚风眠倒是成了一个穷光蛋了，如今若是要去虚空海，探秘颜古龙帝的洞府。
<br/>　　楚风眠的打算，至少也要趁着这几日时间，前去圣山修炼突破，随后再启程去，也会保险许多。
