　　魔君蚩尤攻战南方领土的消息，瞬间传遍天下，各路诸侯以及势力，纷纷大惊。蚩尤此时代表的是劫魔殿，他如今插手人间的王朝争夺，是否代表着万劫魔尊的意思呢？更何况，蚩尤乃是与人皇争夺过人皇之位的人，虽然最后失败了，但是谁也不敢小瞧他。
<br/>　　朝歌，皇宫中。
<br/>　　刚刚亲自出手收服东伯候姜文焕的纣王等人，一回到宫中，便听到蚩尤屠灭三山关，占领南方的消息，这对他们来说，是一个十分糟糕的消息，邓九公、赵云、黄天化等人战死不说，单单多出一个魔君蚩尤，就足够让人忌惮不已了。
<br/>　　“魔君蚩尤，竟然没有被轩辕黄帝杀死，反而被劫魔殿那位给救走了，如今再次出现在人族九州，掀起腥风血雨，再加上西岐的兵力，大商真是多事之秋啊。”纣王感叹道。
<br/>　　初始听到蚩尤出现的消息，他也是大吃一惊，这与他来自未来的消息有所不符，元始圣人的诡异不说，还多出了个劫魔殿，甚至就连蚩尤也没死，这不得不让他打起十二分精神来对待这个与众不同的洪荒世界。
<br/>　　“蚩尤的出现，是否是代表着劫魔殿的意思，劫魔殿可是出现一位混元大罗金仙的证道强者，能够与西方准提圣人大战之人，可不是好对付的。再加上一个阐教的元始天尊，我殷商正面应对的便是两位圣人，而我们只有一位态度不明的截教教主，这事情恐怕不好对付啊。”旁边，一手摇着羽扇的诸葛孔明叹息道。
<br/>　　纣王双眼一眯，缓缓道：“恐怕不止两位，你还没说太清圣人与西方两位圣人呢。”
<br/>　　“那现在该如何是好？”
<br/>　　一旁，一个叫敖无天的壮汉说道。这敖无天是一名烛龙，但不是洪荒烛龙，而是纣王当初攻打一个名为仙剑的世界收服的手下。
<br/>　　“不急，蚩尤的出现，恐怕有人比我们还要急，要知道当初蚩尤可是与黄帝大战之人，他得罪的人应该有不少。更何况，魔族的出现，人族祖地或者三教弟子们，特别是阐教，会比我们更加着急。”纣王不急不缓地分析道。
<br/>　　“不错，不过，我们也应该多加防范，特别是东方，刚刚收服不久，蚩尤还打着那里的注意，我们需要派何人前往。”孔明问道。
<br/>　　“嗯，无天，你带着杨家将去帮朕镇守东方，抵御蚩尤大军，太师与子牙去对付西岐一方。”纣王吩咐道。
<br/>　　“是，大王，臣领命！”太师闻仲与姜子牙对着纣王一拜，便出去了。
<br/>　　走到门外的太师对着姜子牙问道：“子牙，你与阐教熟悉，可知他们下一步会派出那些人？”
<br/>　　姜子牙郑重道：“阐教三代弟子都可能出现，特别是杨戬，他被尊师钦定为阐教护法，特别厉害。申公豹师弟也很厉害，不可小觑。”
<br/>　　“嗯。”太师点了点头，道：“我要回截教去请几个帮手，之后再到汜水关与你会合。”
<br/>　　“太师尽管前去，子牙定然会拖住战局。”姜子牙拱手道。
<br/>　　“嗯。”太师点了点头，便取出他老师金灵圣母给的法宝度人舟，坐上法宝，之后飞入云端，消失不见。
<br/>　　……
<br/>　　阐教之中，对于蚩尤所做之事，元始并没有如纣王猜测那般，派出弟子前去围剿，而只是淡淡的回答了两个字“不管”。
<br/>　　阐教子弟虽不明所以，但既然是天尊的旨意，他们就要遵守，便也不去管蚩尤之事，继续关注西岐一方面的战事，随时给予增援与帮助。
<br/>　　阐教不管，人教更不会去管，截教的通天教主也没有去理会，但他门下的弟子，却有几个经不住朝歌一方的诱导，出山帮忙殷商抵御住蚩尤一方的攻击，但都是有去无回。
<br/>　　三教弟子都不会管，西方教更是在看热闹，他们巴不得东方混乱呢。不得以的情况下，人族祖地方面，武宫之中，终于派出高手前来对付蚩尤了。
<br/>　　南方是人族之地，而如今蚩尤却是魔族，人族是不可能任由魔族之人在人族地盘上大肆屠杀的。是以，人族祖地派出五位转轮皇者境的高手出动，前来对付南方魔君蚩尤。
<br/>　　但劫魔殿也有准圣级的高手，双方交手都不分胜负，一时间僵持下来，谁也奈何不了谁。
<br/>　　……
<br/>　　昆仑山坐忘峰头弯弯曲曲的石径上，红叶铺满，申公豹低着头，急匆匆地拾阶而上。
<br/>　　“师弟，你来了，师兄在此恭候多时了。”
<br/>　　“师兄！”申公豹抬头一看，只见一名老者，手持木杖，站在山头，原来是南极仙翁啊。
<br/>　　“师兄，不知老师唤我回来所谓何事？”
<br/>　　南极仙翁呵呵一笑，并没有回答他的问题，而是说道：“师弟，且随我进宫拜见老师。”
<br/>　　两人穿过层层门户，到天尊殿上，只见元始天尊高居八宝云光座上，九仙山，五龙山、乾元山、少华山等各山的师兄们都在蒲团上坐定，见申公豹前来，只是随意看了他一眼。
<br/>　　八宝云光座右边，挂着一张大榜，大榜之前，白鹤童子手持青丝拂尘而立。
<br/>　　申公豹一见大榜，顿时犹如打了鸡血般激动，难道这就是封神榜？他却也不敢怠慢，连忙拂袖下拜：“弟子申公豹，拜见老师，愿老师圣寿无疆！”
<br/>　　元始点了点头，伸手指向那大榜，说道：“这便是封神榜，你先看一看。”
<br/>　　果然是封神榜！
<br/>　　申公豹虽说一直都听说过封神一事，但却从来没有见过榜文，他按耐住心中激动，抬眼看去，只见那大榜高六尺四寸，宽二尺八寸，悬浮于虚空之中，青雾氤氲，也不知是何材料制成，浑然不似纸帛，倒是仿佛那青天之上取下的一块，就好似一块青天，这个比喻极好。
<br/>　　他仔细观看，却见一片氤氲光芒之中，几许灵文浮现，他仔细辨认，那些灵文上写的是雷、火、风、雨、瘟、斗、群星列宿、三山五岳等共三百六十五个神祗的官职名号，每一个神位之上，都有符文闪烁，都只有一个字，时隐时散，申公豹初始不在意，那一笔一画就在眼前，却不解其中意思，他定睛一看，却还是朦朦胧胧，好似雾里看花，越是用心，却越是看不清。
<br/>　　封神榜上之名，乃是神官之名讳，更是天地承认的神明，并非凡俗之名。虽然这榜文高悬于此，并不限制人来观看，但不要说申公豹只有小小天仙的修为，就是广成子、孔宣等准圣强者，道行高深，法力广大，也一样熟视无睹，看之不清，观之不明，不知道上面写的是什么。
<br/>　　申公豹看了一会儿，看得头晕，也看不清是什么，无奈的他只好请教元始天尊：
<br/>　　“请老师指教！”
