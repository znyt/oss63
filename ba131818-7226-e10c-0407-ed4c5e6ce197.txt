　　八劫古帝巅峰的力量冲天而起，凝聚成为了一尊远古战龙虚影，笼罩在了楚风眠的身上。
<br/>　　楚风眠的境界虽然是八劫古帝巅峰，但是他的一身实力，比起九劫古帝还要雄厚。
<br/>　　尤其是楚风眠身上的战龙虚影，已经不再是寻常远古战龙的模样，而是近乎龙皇一般，无比尊贵。
<br/>　　瞬息之间楚风眠的力量淋漓尽致的展现了出来。
<br/>　　“八劫古帝巅峰？”
<br/>　　碧螺老祖在看清楚风眠身上境界的时候，无比不可思议，眼神之中的惊讶之色许久无法消散。
<br/>　　上一次他见到楚风眠的时候，楚风眠的境界可只是刚刚踏入八劫古帝。
<br/>　　八劫古帝的境界，踏入八劫古帝巅峰，让许多天才来，也要花费数十年的岁月，碰到一些资质一般的古帝。
<br/>　　数百年的时间，都未必足以从八劫古帝，踏入八劫古帝巅峰，而楚风眠这才短短一日不见，楚风眠的实力就提升的如此之快。
<br/>　　这种力量的提升，简直骇人听闻。
<br/>　　碧螺老祖本以为他得到了那建木树枝，吞噬了其中的一部分力量，得到了惊天奇遇，想要杀了楚风眠无比简单。
<br/>　　可现在，楚风眠的实力超出了他的想象。
<br/>　　在这碧螺老祖惊讶的刹那间，楚风眠已经是出手了，面对那翠竹长矛的轰击而来，楚风眠只是轻轻抬起手掌。
<br/>　　在楚风眠背后，战龙虚影的力量，统统凝聚在了楚风眠的掌心之中。
<br/>　　旋即一掌轰击而出，群龙咆哮，如同一尊真正龙族施展出的龙族武技。
<br/>　　始祖第二式！
<br/>　　血脉之力沸腾，这一次施展出始祖第二式的威力，比起之前又是强横了无数倍。
<br/>　　一掌轰击而去，凌空跟那翠竹长矛轰击在一起，这翠竹长矛被当场打爆，一掌打碎。
<br/>　　强横的力量，逼得碧螺老祖连连后退。
<br/>　　现在楚风眠的实力再进一步，寻常的九劫古帝，根本难以抗衡楚风眠的力量。
<br/>　　“该死，此人的力量怎么会提升到了如此地步？难道这独桥尽头的宝藏，被此人得到了？”
<br/>　　碧螺老祖的身形被打飞了出去，他也一下子清醒了过来，楚风眠的实力已经不再是他可以交锋的了。
<br/>　　这种实力，纵然是在九劫古帝之中，都是一位绝对的强者。
<br/>　　短短一日时间，楚风眠的实力足足提升一倍，除了这独桥尽头的宝藏是被楚风眠得到了之外，没有其他的可能。
<br/>　　“又得到了一个大型宝藏，再让此子发展下去，真的是第二个云尊了！”
<br/>　　碧螺老祖眼神之中露出几分决断之色，突然间狂吼一声，一声狂吼响彻整座独桥之中。
<br/>　　这一声狂吼，仿佛是一种独特的呼唤声音。
<br/>　　楚风眠已经是利用灵识发现，在这碧螺老祖的一声吼叫之下，几尊九劫古帝都是飞速的靠近了过来。
<br/>　　这碧螺老祖，是在呼唤着其他三大势力一方的九劫古帝到来。
<br/>　　他现在自知自己一个人不是楚风眠的对手了，所以也放弃了想要一个人独吞楚风眠身上宝藏的想法。
<br/>　　要呼唤其他九劫古帝来对付楚风眠。
<br/>　　“还想要叫人来？我就先杀了你再说！”
<br/>　　楚风眠眼神之中杀机涌现，也不知道不能在拖延下去。
<br/>　　单打独斗楚风眠杀了这碧螺老祖还有机会，但是一旦是让碧螺老祖呼唤到其他九劫古帝到来。
<br/>　　倒时候楚风眠以一敌二，以一敌三，在想要杀了这碧螺老祖，在得到建木树枝，几乎是没有可能。
<br/>　　刹那之间，剑光飞出，楚风眠背后的天涯剑匣之中，瞬息之间便是打出了数百道剑光。
<br/>　　这每一道剑光的威力，都达到了八劫古帝巅峰的全力一击。
<br/>　　随着楚风眠实力的提升，这天涯剑匣所爆发出来的剑气威力也就越发的强大。
<br/>　　天涯剑匣，毕竟曾经可也是天人境强者的灵器。
<br/>　　随着楚风眠实力的提升，也展现出了其中真正的威力。
<br/>　　数百道剑光，凝聚在一起，从天而降对着碧螺老祖斩了过去。
<br/>　　“朽木不死！”
<br/>　　碧螺老祖见到这剑光的到来，也是察觉到了其中的危险，长啸一声，在他的手心之中，突然出现了一小段枯木。
<br/>　　这一段枯木，虽然枯萎，但是其中蕴含的生命力，却是几乎足以跟建木树枝所媲美。
<br/>　　碧螺老祖的身形一闪，直接跟这一小段枯木融为了一体，化为了一株腐朽的树木。
<br/>　　人木合一。
<br/>　　这朽木将碧螺老祖身躯，团团包裹住，其中强大的生命力，守护者碧螺老祖的身躯。
<br/>　　“这朽木的什么东西？怎么会有如此惊人的生命力？”
<br/>　　楚风眠见到这朽木的一刻，眼神一变。
<br/>　　纵然只是第一眼见，楚风眠就看的出来这朽木的不凡之处，这种恐怖的生命力，比起建木树枝都毫不逊色。
<br/>　　现在这碧螺老祖，跟着一段朽木融为一体，碧螺老祖身上的气息在暴涨。
<br/>　　虽然这样的状态之下，碧螺老祖无法出手攻击，但是如此惊人的生命力，想要杀了碧螺老祖，几乎是不可能的一件事。
<br/>　　只见天涯剑匣数百道剑光凝聚而成的巨大剑光，从天斩下，斩在了这朽木之上。
<br/>　　剑光所落之处，一道裂痕顿时撕裂开来，可就当着朽木仿佛要被斩断的一刻。
<br/>　　其中惊人的生命力，当即爆发出来，出现的一道道裂痕，被完美的修复。
<br/>　　一剑之威，都无法从现在碧螺老祖的身上留下任何一道伤痕。
<br/>　　“真是棘手！”
<br/>　　楚风眠眼神之中都露出了几分焦急，他已经是感觉到，这其他的九劫古帝即将赶来了。
<br/>　　这碧螺老祖的这一招，就是要拖延时间，等到其他九劫古帝到来，在围攻楚风眠。
<br/>　　碧螺老祖看着楚风眠眼神之中的焦急，嘴角露出了几分冷笑道。
<br/>　　“小子，你以为你得到了几分奇遇，就真的想要杀一名九劫古帝？等过一会，就是你的死期！”
<br/>　　他只要是拖延足够的时间，等到其他的几位九劫古帝敢来，纵然是楚风眠的实力再强。
<br/>　　也休想要以一敌三，以一敌四，到时候围攻之下，就是楚风眠的死期。
<br/>　　“先把建木树枝夺走拿到手！”
<br/>　　楚风眠脑海之中，也在思考这，突然之间楚风眠下了决定，身形一闪直接向着碧螺老祖冲了过去。
