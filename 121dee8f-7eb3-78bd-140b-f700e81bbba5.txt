　　金光幽狼，皇境妖兽。
<br/>　　其实力，大概在圣皇境三重左右。
<br/>　　这个层次的妖兽，萧逸自无法单凭剑气斩杀，故祭出了冷焱剑。
<br/>　　金光幽狼，这等妖兽放在金光险地内，亦是实力不俗的霸主级妖兽了。
<br/>　　可对于手中持剑的萧逸来说，这等妖兽，还不够看。
<br/>　　吼…
<br/>　　一声兽吼，金光幽狼咧开了它那嗜血而锋利的尖牙。
<br/>　　萧逸怡然不惧。
<br/>　　金光幽狼，瞬间而动，朝萧逸袭来。
<br/>　　萧逸手中冷焱剑一扬，正要劈出。
<br/>　　忽然，金光幽狼袭来的身影，陡然一变。
<br/>　　袭来的身影，霎时跃至高空。
<br/>　　随后，金光幽狼大口吞食着空气中的氤氲金光，身上气息，亦在不断飙升。
<br/>　　“好狡猾的孽畜。”萧逸双眼一眯。
<br/>　　妖兽的直觉，异常灵敏。
<br/>　　且越是实力强横的妖兽，直觉愈加惊人。
<br/>　　显然，在这头金光幽狼看来，萧逸这个人类武者，绝非善茬。
<br/>　　故它看似瞬间袭向萧逸，实则，不过是要出其不意，吸收高空中更加浓郁的金光力量。
<br/>　　萧逸并未阻止，只是手中握剑，等待着。
<br/>　　不多时，待得金光幽狼吸收完毕，其所散发的一身气息，竟已达圣皇境七重。
<br/>　　萧逸脸色微惊。
<br/>　　对比之前遇到的伏息树蟒，吞食金光后，圣王境六重气息达到圣王境八重巅峰气息，大概提升了近三重。
<br/>　　而现今，这头金光幽狼，却是足足提升了四重。
<br/>　　而且还是圣皇境内的巨大差距。
<br/>　　想来，这里的妖兽，能吸收多少金光，发挥多少效果，亦与这头妖兽本身的实力有关。
<br/>　　吼…
<br/>　　这时，金光幽狼已然呲牙咧嘴。
<br/>　　泛着凶光的双眸，仿佛在讥讽萧逸这个人类之前没有阻拦的大意。
<br/>　　讥讽的眼神中，又带着不屑以及冰冷，仿佛在看一个死人。
<br/>　　“狼类妖兽，果真狡猾而冷血。”萧逸冷笑一声。
<br/>　　嗖…
<br/>　　金光幽狼的身影，瞬间消失在原地。
<br/>　　一只泛着寒芒的利爪，陡然朝萧逸咽喉而来。
<br/>　　“速度倒是够快。”萧逸脸色一凝，随后一个侧身，躲过了袭击。
<br/>　　金光幽狼的锋锐利爪，擦肩而过。
<br/>　　带起的锋锐气息，直将萧逸身后数百米范围夷为平地。
<br/>　　狼类妖兽，本就擅长速度。
<br/>　　现今这头金光幽狼的速度，绝不亚于一般的圣皇境八重妖兽。
<br/>　　嗖…嗖…嗖…
<br/>　　金光幽狼，化作了一道道幻影，在空气中疾速跃动。
<br/>　　这般幻影重重下，足以让武者虚实难分。
<br/>　　对比之前的忽然袭击，此举，无疑更显示了金光幽狼的狡猾。
<br/>　　萧逸的身影，亦在一瞬间动了。
<br/>　　嗖…嗖…嗖…
<br/>　　傲然的身躯，同样化作了一道道幻影，闪烁于空气之中。
<br/>　　两股幻影，截然不同。
<br/>　　但萧逸的速度，显然快于这头金光幽狼。
<br/>　　八万星光，早已降临。
<br/>　　不过，星光并未加诸于剑上，而是化作星光之力，加持与萧逸身上。
<br/>　　萧逸的幻影，每每比金光幽狼快上一丝。
<br/>　　整整半个时辰后。
<br/>　　金光幽狼停下了跃动，喘起了粗气。
<br/>　　如此长的时间内，它全力袭击之，却连萧逸的一根头发都未伤到。
<br/>　　而现在，则脸色淡然，虽未喘气，却也额间露出汗水。
<br/>　　显然，那般高强度下的不断跃动，躲避，对他而言是不小的消耗。
<br/>　　吼…
<br/>　　金光幽狼的吼声，仿佛是从喉咙中发出，低沉而带着嘶吼。
<br/>　　原本泛着寒芒的利爪，忽然添上一抹幽光。
<br/>　　幽光之上，传来阵阵阴冷气息。
<br/>　　滋滋滋…
<br/>　　金光幽狼的爪子，踏于地面，地面竟在不断腐蚀。
<br/>　　不过片刻，金光幽狼身下，已被腐蚀出一个大坑。
<br/>　　“好阴冷的腐蚀气息。”萧逸微微皱眉。
<br/>　　金光幽狼，可是擅使幽冷气息的妖兽。
<br/>　　原本这种妖兽，就十分危险。
<br/>　　它们异常狡猾的本性，加上不俗的实力，让得它们初初散发着危险的气息。
<br/>　　吼…
<br/>　　又是一声嘶吼。
<br/>　　金光幽狼的身影，再次袭来。
<br/>　　速度比之前更快，爪锋，亦更加危险。
<br/>　　它甚至相信，只要自己的利爪击中面前这个‘灵活’的人类，这个人类武者，必死无疑。
<br/>　　锵…
<br/>　　这一次，萧逸没有躲闪。
<br/>　　而是手中仗剑而出。
<br/>　　清脆的剑鸣，陡然响起。
<br/>　　冷焱剑剑身，瞬间与金光幽狼的利爪交锋起来。
<br/>　　嘭…
<br/>　　忽然，一声爆响。
<br/>　　金光幽狼那散发着幽光的利爪，在与剑身触碰的一瞬间，竟开始溢散。
<br/>　　幽光，化作了黑色幽雾。
<br/>　　萧逸脸色微变。
<br/>　　黑色幽雾，出现得极快，几乎是出现的一瞬间，已然充斥周遭。
<br/>　　萧逸嗅了一下，顿觉脑袋一阵微微眩晕。
<br/>　　显然，这些黑色幽雾中，带着剧烈的毒性。
<br/>　　这头金光幽狼的危险程度和狡猾程度，超过了萧逸的预料。
<br/>　　嘶…
<br/>　　金光幽狼狰狞的面容上，闪过一道极其人性化的冷笑。
<br/>　　手中利爪，瞬间朝萧逸咽喉而来。
<br/>　　面前人类武者，已然中了它的剧毒，心神不清。
<br/>　　它的利爪，足以割破面前人类武者的咽喉。
<br/>　　然而，锵的一声…
<br/>　　冷焱剑剑身，瞬间挡下了利爪的袭击。
<br/>　　凛冽的剑气，甚至将金光幽狼重重轰飞。
<br/>　　金光幽狼，确实狡猾，掌控的幽光毒雾，亦威力不小。
<br/>　　不过，在一个炼药师面前使毒？特别是萧逸这等手段强悍的炼药师面前，这些毒雾，不过是个笑话罢了。
<br/>　　锵…锵…
<br/>　　萧逸的剑，动了。
<br/>　　一连两剑，直将这头金光幽狼轰伤。
<br/>　　“没别的本事了吗？”萧逸皱了皱眉头。
<br/>　　“若如此，战斗也该结束了。”
<br/>　　嗖…
<br/>　　萧逸的剑，宛若惊鸿。
<br/>　　冷白剑芒，一闪而过。
<br/>　　金光幽狼嗜血的双眸中，映衬出一道巨大的星光长河倒影。
<br/>　　还未等它反应过来，冷白剑芒，已然来到它眼前。
<br/>　　星光长河，亦碾压而至，将它吞噬。
<br/>　　待得星光消散，绞杀之力肆虐而过。
<br/>　　金光幽狼，已然身死。
<br/>　　萧逸摇了摇头，面露失望之色。
<br/>　　他来金光险地，可不单单是为了收集妖兽精血，最重要的是为了历练。
<br/>　　若是寻常的圣皇境三重妖兽，他早便直接杀了。
<br/>　　可这头金光幽狼，吸食金光后，有了圣皇境七重实力，便值得他一战。
<br/>　　萧逸大手一挥，收了精血与妖兽内丹。
<br/>　　刚要离去。
<br/>　　手中冷焱剑，忽然剑身一颤。
<br/>　　“嗯？”萧逸面露惊疑。
<br/>　　在他的印象中，冷焱剑会有此举动，往往只是有血气力量出现时，它需要需收。
<br/>　　可现今，这里并无血气力量。
<br/>　　而冷焱剑的剑锋，亦缓缓指向了周遭氤氲金光。
<br/>　　“你想吸收这里的金光力量？”萧逸脸色一惊。
