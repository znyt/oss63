　　第327章
<br/>　　程大雷的确没想到，莫鸣米手上还握着一支重甲军。当然，琴川关有铁矿，莫鸣米有打造重甲军的条件。
<br/>　　说起来，幸亏自己今天反应及时，没陷入莫鸣米的包围圈，否则后果不堪设想。直到现在，程大雷想起来还是一阵阵后怕。
<br/>　　战场之上，瞬息万变，说来自己还真是有些大意了，小窥了莫鸣米这一方枭雄。
<br/>　　以后这样的错误万不可再犯。
<br/>　　今日会议的主要议题，就是如何应对莫鸣米的重甲军。
<br/>　　众人聚在一起，各抒己见，程大雷只是作为一个听众，偶尔主持一下会议纪律，一直没有发表自己的看法。
<br/>　　“依我看，莫鸣米的重甲兵根本不足为惧，重甲兵虽然防御强，但机动性太差，我们尽可用骑兵屠杀对方小股部队，对方的重甲兵根本就是一只不会动的乌龟。”说这话的是关鱼。
<br/>　　“可他们若直奔我们大本营而来怎么办，虽然我们布着重弩，但对重甲兵杀伤力有限。”李行哉。
<br/>　　“这……”关鱼微微迟疑：“我再想想。”
<br/>　　“要不咱们撤吧。”刘悲道：“咱们在青牛山撤过一次，在这里尽可再撤一次。”
<br/>　　“好不容易打下的家业，难道真能舍了。咱们舍了这里，又去何处落脚？”赵子龙。
<br/>　　“嗨，这你就不懂了，咱们可以去琴川关呐。他们都来打我们了，咱们正好趁机占领琴川关。”高飞豹。
<br/>　　“你说的这不叫打仗，这叫搬家。”刘悲。
<br/>　　“军师，你一直不说话，莫非心中已经有了主意。”李行哉问徐神机。
<br/>　　“呃……哈哈。”徐神机长笑一声，把所有人的视线都吸引到自己身上。颇有一种睥睨天下无对手的豪横感。
<br/>　　“请军师高见？”刘悲。
<br/>　　“还没有。”
<br/>　　“……”众人。
<br/>　　“好啦，好啦，先散了吧。”程大雷挥挥手，结束了今天的会议，还是别让徐神机开口了，莫让他把大家智商都带偏了。
<br/>　　战斗状态下，即便夜里也保持着警戒，以防备敌人发动夜袭。大战之前，每个人都是枕戈待旦，即便程大雷也是很难好好休息。他在夜里检查各个哨岗，很晚才回到房间。
<br/>　　苏樱帮他将外套脱了，小桃从厨房端着洗脚水过来，经过这些日子，程大雷已渐渐习惯被人服侍。
<br/>　　“仗是不是很难打。”苏樱一边帮程大雷洗脚一边问。
<br/>　　“嗯呐。”
<br/>　　程大雷不是很愿意说话，劳累一天的他已经是筋疲力尽。而他也明白，接下来的日子，莫鸣米不攻则已，攻必是倾尽全力，到时候也就是决战的日子。可直到现在，自己还没想好如何应付莫鸣米的重甲兵。
<br/>　　在如今这个时代，重甲兵就相当于战地坦克，虽说行动缓慢，却是坚不可摧。
<br/>　　如果真对付这些重甲兵也不是没有办法，可以用火攻，把这些困在铁壳子里的人活活烧死。然而这样的天气用火攻，程大雷还没有蠢到和徐神机的程度一样。
<br/>　　实在不行的话，程大雷就只有多唤点人出来，用数量优势压垮对方。
<br/>　　只不过，这样的牺牲是程大雷不喜欢的，而他的恐惧值留着还有其他用处。
<br/>　　“天气实在是太冷了，滴水成冰，你在外面莫要冻伤了……”
<br/>　　苏樱正说着，程大雷忽然腾地一下站了起来。吓了苏樱一跳，洗脚水溅在她脸上。
<br/>　　“你说什么？”
<br/>　　“你别冻伤了。”
<br/>　　“前面一句？”
<br/>　　“滴水成冰？”
<br/>　　“是呐，滴水成冰。”程大雷猛地将苏樱抱起，逗得她花枝乱颤。
<br/>　　这样的天气不可用火，但不代表不能用水呐……
<br/>　　休整三天后，莫鸣米发动了总攻。这一次不再是为了试探或者说消耗，是准备一举攻下程大雷。
<br/>　　依旧是命令十八家首领为前阵，也就是炮灰，重甲兵负责主攻，再然后是自己的人压阵。
<br/>　　队伍从蛤蟆寨的东面发起主攻，迎来了一阵阵重箭，这一次莫鸣米命令手下冒死冲锋，一直冲到蛤蟆寨的大门下。
<br/>　　这里已经是重弩射不到的位置，莫鸣米开始用攻城车冲击大门。
<br/>　　很短时间后，大门被攻破，莫鸣米开始对箭楼发动攻击，层层拔下程大雷的攻击力量。
<br/>　　看到这一幕，莫鸣米脸上的阴云才终于舍得散去，及到现在，已可以确定自己将取得这次战斗的胜利。
<br/>　　时间是漫长的，战斗是残酷的，但是，若不是一直在消耗程大雷的力量，恐怕要付出更多代价。
<br/>　　幸运的是，充当炮灰的不是自己的人，其他人死多少莫鸣米都是不在意的。
<br/>　　而他的重甲军，可个个都是宝，能撑着上百斤铁甲战斗的人，本身就是不可多得的人才。
<br/>　　而同样，这些人一旦派出去，必然是攻无不克，战无不胜，即便面对程大雷也是一样。
<br/>　　想这程大雷也算是一个人物，闹长安，乱戎族，千军万马都留不住他，最后却死在自己手中。
<br/>　　莫鸣米不免有几分唏嘘，他已决定，当战斗结束后，要给程大雷厚葬。
<br/>　　当然，此刻杀死程大雷还是毫不留情的。
<br/>　　这时候，变故突生，从箭楼浇下一桶桶水来。都是凿开河面取来的冰水，此刻兜头浇下，瞬间将重甲兵淋了个透心凉。
<br/>　　西北的天气虽然不像幽州那样寒冷，但其实也差不了多少。穿着上百斤的铁甲本就行动不便，此刻冰水一浇，极短时间内，他们的盔甲就被冻得连接在一起。
<br/>　　这三日，程大雷可是一天也没有闲着，不停从河道中取水，等的就是今日。
<br/>　　然后，关鱼等人冲了出来。
<br/>　　战神旗，破军旗，鼓舞旗，铁血旗……五面旗帜的作用叠加在一起，恐惧在敌人心中绵延，同时，己方的士气高涨。
<br/>　　用的自然是关鱼的法子，莫鸣米的重甲兵直接放过不管，专门屠杀对方的骑兵步兵，而首要目的，依旧是诛杀莫鸣米。
<br/>　　战场之上，胜利和失败来得太突然了。莫鸣米惊慌失措看到这一幕，内心第一次浮现自己将会失败的感觉。
<br/>　　程大雷因大意犯过一次错，莫鸣米今天也大意了。而有些错误，是只能犯一次的。
<br/>(本章完)
