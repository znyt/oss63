　　　　詹姆斯无奈的点头，“目前为止，我们只发现了KSK队员的下落，而且这还不是全部KSK队员，对方可能是把抓走的人质分几处关押，这样不仅增加了我们营救的难度，也方便他们看管。至于其余人的下落，我们还在调查，只要他们在中东，我们就能把人给找出来！”
<br/>　　“找到西伦科特了吗？”罗昊沉吟了一声，对詹姆斯问道。
<br/>　　打蛇打七寸。
<br/>　　如果可以找到西伦科特，就可以通过西伦科特，直接找到那些被抓的人质下落。罗昊能想到这一点，詹姆斯自然也是想到了。
<br/>　　詹姆斯无奈的摇头道：“暂时还没有找到他，唯一能确定的是这家伙还在中东。”
<br/>　　中东太大了，如果没有明确的消息，要想把西伦科特找出来几乎是不可能的事情。
<br/>　　“将军，我现在需要带我的人进入敌占区。”罗昊对詹姆斯说道：“我希望能有直升机，送我们过去。”
<br/>　　“好。”詹姆斯没有拒绝罗昊的要求，说道：“我会让一队海军陆战队跟随你们一起行动，完全听从你的指挥。”
<br/>　　“不，将军。”罗昊拒绝了詹姆斯的提议，说道：“我知道，这样可能会引起你的反感，但是我现在只相信我自己的人。”
<br/>　　“你很谨慎。”詹姆斯盯着罗昊看了一会儿，随后说道：“你是一个合格的指挥官。去吧，这艘航母上的后勤部门会为你和你的队员提供你们所需要的一切支持。”
<br/>　　罗昊对着詹姆斯敬了一个礼，然后转身离开；云熙也紧跟着向詹姆斯敬了一个礼，转身朝着房间门口走去。
<br/>　　“少校！”
<br/>　　詹姆斯从背后叫住云熙，说道：“你的丈夫是个优秀的军人！”
<br/>　　“他是我心中的英雄。”
<br/>　　“如果他是星条旗下的军人，他必定会成为一个将军！”詹姆斯说道。
<br/>　　“作为五星红旗下的军人，他也一定会成为一个将军。”
<br/>　　云熙迈步走出房间。
<br/>　　看着那扇被重新关上的门，詹姆斯的脸上突然露出一抹微笑，呢喃着说道：“两个人都是那么自信，要说你们不是夫妻，我都不会相信。”
<br/>　　有了詹姆斯的命令，这艘航母上所有的后勤部门为罗昊、慕容飞、李毅峰、李然四个人大开绿灯。
<br/>　　四个人在武器库中挑选着他们的武器。
<br/>　　他们原本携带来的武器在遭到导弹攻击时丢失了一部分，保留了一部分。罗昊、齐凡、慕容飞的武器都保留了下来，只有李毅峰原来所使用MSR狙击步枪丢失了。
<br/>　　为此，李毅峰从武器库中又重新挑选了一把M2010超级狙击步枪，作为自己的这次潜入敌占区所使用的武器。
<br/>　　M2010超级狙击步枪虽说是在M24SWS基础上升级后的产物，但是不管是外观，还是枪管、枪膛、膛线缠距都是全新的设计，原来M24上唯一保留下来的只有机匣，所以，M2010完全算作是一把新枪。
<br/>　　在弹药方面，子弹从原来的7.62×51mm北约制弹药换成了.300温彻斯特-马格南弹，这种子弹采用塞拉公司生产的14.26克重的MatchKing空尖尾锥弹头，比7.62mm口径的北约制标准单具有更强大的动能，能提升接近50%的有效射程。尤其是在******经过优化之后，可以在M2010狙击步枪枪管中充分燃烧，有效射程可以达到1370米。而且由于弹头外形阻力小，所以保持了很高的弹道系数，在较远的距离上仍然有较高的存速。
<br/>　　瞄准系统上，使用Leupold可变倍白光瞄准镜，最高可以将目标放大20倍，在夜间作战时，只需要在白光瞄准镜前串列一个AN/PVS-29狙击夜视仪，不用对瞄准镜重新归零。
<br/>　　罗昊、李然、慕容飞三个人在补充了弹药后，又各自从武器库中挑选了一把伽利尔ACE-GAP39SB短管步枪，这是以色列武器工业公司在美国民用市场推出的步枪。ACE-GAP39SB短管步枪，采用折叠式枪托和211mm短枪管，全枪长度只有679mm，使用7.62×39mm弹药，只能半自动射击。
<br/>　　虽说这是一款针对民用市场的武器，但是因为其短小紧凑的结构设计，是一款非常理想的背包步枪，经常被特种部队作为备用武器使用。
<br/>　　“尼米兹”级核动力航母甲板上，在甲板边缘的位置，竖立着好几个钢靶，距离各不相同。罗昊、慕容飞、李然、李毅峰四个人对着钢靶进行射击，校准着自己的武器。
<br/>　　三十分钟后，一架UH-60“黑鹰”直升机在两架AH-64D“长弓阿帕奇”的护卫下，起飞离开航母，这家“黑鹰”直升机将把罗昊、李然、李毅峰、慕容飞四个人送入被影子部队和反政府武装势力所控制区域外的一个镇子上。
<br/>　　“黑鹰”直升机两侧舱门口，两个机组成员戴着飞行员头盔坐在舱门口，两条腿很随意的垂在机舱外，他们手中各抱着一挺M134机枪，如果有人想要对直升机进行袭击的话，他们会用这两挺机枪给对方一个好看。
<br/>　　经过一个小时的飞行，“黑鹰”直升机在距离敌占区还有十公里的一个小镇上降落。
<br/>　　直升机只能把四个人送到这里，如果再继续深入很有可能会被反政府武装势力或者影子部队发现，然后遭到火箭筒，甚至单兵放空导弹的攻击。
<br/>　　“祝你们好运！”
<br/>　　机舱门口抱着机枪的机组成员拍了拍罗昊的肩膀，对着罗昊竖起一个大拇指。
<br/>　　罗昊冲着对方点了下头，然后跳下直升机，迅速冲进路边的一栋已经废弃的楼房中。
<br/>　　等到四个人全部进入那栋废弃的楼房中后，这架UH-60“黑鹰”直升机在两架“长弓阿帕奇”的护卫下，快速离开，返航。
<br/>　　罗昊手中握着安装了消音器的G36C突击步枪，同样安装了消音器的ACE-GAP39SB短管步枪折叠起枪托后，被固定在背包侧边，现在武器所使用的消音器在经过一系列改良之后，不会再对武器原本的精度产生影响，这也使得特种部队在行动过程中，需要采取无声战斗的时候，变得更加得心应手，不用担心因为枪声而暴露自己的踪迹。
