　　第158章
<br/>　　“是罗将么？”程大雷的声音响起。
<br/>　　罗铁之前没和程大雷照过面，只听说他斩了熊老大，心里还有几分期待。今日一见，则是大失所望。
<br/>　　程大雷被两个人搀着，蜡黄脸色，说话时也有气无力的，一脸谄媚的笑意。
<br/>　　“你就是程大雷！”罗铁没有掩饰自己的不屑。
<br/>　　“正是区区在下。”程大雷道：“不知手下什么地方惹了罗将不开心？”
<br/>　　罗铁冷哼一声，道：“程大雷，你要明白，你就是个山贼，是义父提携你，才有的你今日，你若不识抬举，义父提得起你，也踩得下你。”
<br/>　　小黑搀着程大雷，重重哼了一声。程大雷捏了捏他的手掌，示意他小不忍则乱大谋，蛤蟆寨如今还没有实力和幽州王对着干。
<br/>　　“明白，明白。”程大雷垂着手道。
<br/>　　见程大雷如此没骨气，罗铁心中更加不屑。
<br/>　　“这是给人喝的东西么！”罗铁拿起酒坛：“你们山寨就拿这些招待我们！”
<br/>　　“快来人，去给罗将换好酒，一定要罗将尽兴。”
<br/>　　程大雷逆来顺受，到是搞得罗铁没有法子。心道，你怎么不生气呢，你一生气我正好趁机生事，去义父那里请得精兵，趁机灭了你蛤蟆寨，我岂不就能重新做人了。
<br/>　　“慢！”罗铁眼珠一转，计上心头：“你把这坛酒给我喝了？”
<br/>　　“哈，禀罗将，我身染风寒，不便饮酒。”
<br/>　　“怎么，你不给我面子，还是不给幽州王面子。”罗铁道：“我现在要你把这坛酒喝尽，一滴也不许剩。”
<br/>　　程大雷垂首，眼底一道狠戾闪过。徐神机几人已经气得面色发白，用眼神不停示意程大雷。只要程大雷一声令下，眼前这百十号人，真不够蛤蟆寨杀的。
<br/>　　“好，我喝。”程大雷忽然换上一副笑容。
<br/>　　蛤蟆寨等人都感觉如临奇耻大辱，心中窝着一口气吐不出来。
<br/>　　“大当家，我替你喝！”
<br/>　　小黑忽然叫了一声，去接那酒坛。罗铁心中一喜，锵地拔出佩剑，冲着小黑撩出。
<br/>　　“你算个什么东西，你也配喝？”
<br/>　　一剑撩出，割下小黑一只耳朵，地上瞬间鲜血淋漓。罗铁都剑果然好快，程大雷竟来不及作出反应。
<br/>　　“什么狗东西，这次只是给你个教训，下次再不长眼，小心你的狗命。”
<br/>　　看着地上惨叫的小黑，程大雷缓缓抬起头来：“你说什么？”
<br/>　　“我……”
<br/>　　罗铁刚吐出一个字，就注意到程大雷的眼神。笑意敛去，只余杀机。
<br/>　　罗铁先是一惊，随之一喜，怒了，终于怒了，早知道刚才一剑直接杀掉那小子，让你怒得更厉害些。
<br/>　　“姓程的，你要管好你的狗，不要让他乱叫，我的快剑可不认狗主人。”
<br/>　　“我的狗我会管教的，那么，你这条狗有没有人管？”程大雷握紧剑柄。
<br/>　　罗铁没有生气，甚至有几分好笑的意思。他哈哈大笑：“你竟然握剑，竟然敢在我的面前握剑，哈哈！”
<br/>　　说着，罗铁从怀里逃出一样物事，挥手掷在空中，只听锵锵之音，六瓣铜板落在地上。
<br/>　　铜板，又见铜板。
<br/>　　看来和罗铁有随身携带铜板，随时给人表演的习惯。
<br/>　　周围人一愣，这才想到，这罗铁素来有幽州第一快剑的称号，曾有江湖客挑战他，被他三招击败。
<br/>　　罗铁身边的运粮队，此时也昂起头来，在山贼面前逞威风，也是他们的荣耀。
<br/>　　“你也有这样的快剑么，能一剑斩开铜板？”罗铁嚣张大笑：“若你也能斩开铜板，今日的事情就一笔勾销。”
<br/>　　程大雷摇头：“我的剑不是用来斩铜板的。”
<br/>　　“喔，那是做什么，切萝卜么？”罗铁笑得愈发放肆，连他身后的兵丁脸上也露出笑意。
<br/>　　程大雷缓缓吐出两个字：“杀人。”
<br/>　　“杀谁，杀我么，凭你手中的剑能杀得了我么？”
<br/>　　徐神机等山贼已经感觉到程大雷身上的杀意，可罗铁犹在梦中浑然未觉。
<br/>　　“大当家，大当家！”徐神机已经喊出声音来：“小不忍则乱大谋啊。”
<br/>　　程大雷置之不理。
<br/>　　他手握剑柄，问：“你可听说过一招剑法，唤做直来直往？”
<br/>　　“什么？”罗铁不解。
<br/>　　“你马上就可以见到了。”
<br/>　　锵！
<br/>　　如听龙吟，程大雷仗剑出鞘，一招直来直往，宛若电闪。
<br/>　　程大雷出剑收剑一气呵成，像是什么都没有发生，只有剑身淅沥沥滴着鲜血。
<br/>　　有人惊讶的发现，罗铁喉咙出现一个血洞。
<br/>　　“罗将，罗将！”兵丁大叫。
<br/>　　罗铁眼中的光芒渐渐熄灭，他用最后的生机睁大眼睛，紧紧盯着程大雷。
<br/>　　“世上……竟有如此快的剑……”
<br/>　　世间竟有如此快的剑？
<br/>　　有。
<br/>　　就握在程大雷手中，剑名匹夫。
<br/>　　匹夫一怒，血流五步。
<br/>　　众兵丁大惊，慌张操起兵器。他们不是第一次从蛤蟆寨经过，向来程大雷都逆来顺受，甚至显得有些软弱可欺。渐渐，大家也就不把他当回事了。
<br/>　　可没想到他不出手则已，出手就杀人，杀得还是幽州王的义子。
<br/>　　哗啦啦！
<br/>　　从蛤蟆寨冲出一帮人，瞬间便将这帮兵丁围住，人人一脸凶悍，像吃人的恶魔，饮血的怪兽。
<br/>　　“程将军……有话好好说，我们不想和程将军动手……”粮队的百夫长道。
<br/>　　程大雷让手下抬着小黑去治伤，又冲这些兵丁道：“放下你们手中的兵器，把身上皮甲脱掉。”
<br/>　　兵丁想反抗，可他们本就是运粮兵，也无多少战斗力，瞬间就被蛤蟆寨的山寨解除了装备，只穿着小衣在料峭的春风中发抖。
<br/>　　“程将军，咱们都在幽州王手下做事，向来关系还不错。您这是，您这是做什么？”
<br/>　　“把粮车拉回山寨。”程大雷又吩咐一道命令。
<br/>　　“程将军，您这是做什么，不能夺粮车，幽州王知道了，会砍了我们脑袋的！”这百夫长急得大叫，奈何手无寸铁，却只能眼睁睁看着山贼把粮车抢走。
<br/>　　“把他带走！”程大雷踹了踹脚边罗铁尸体：“给幽州王传句话，就说罗铁以下犯上，被老程斩了！”
<br/>　　
<br/>(本章完)
