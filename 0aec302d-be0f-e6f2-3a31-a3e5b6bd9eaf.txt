　　酒足饭饱后，慕清霄在酒楼订了一间上等客房，随后便携着小医仙在中域城区闲逛起来。
<br/>　　对于他来说，现在修行是其次，红尘历练才是主要，大乘瓶颈不破，他就无法迈入渡劫，倒不如厚积薄发。
<br/>　　城区商店林立，玲琅满目，好东西有许多，但慕清霄完全用不上，小医仙亦是如此。
<br/>　　若是用灵药来堆积修为，反倒是绝了自己的前路。
<br/>　　虽然不用，可只要看到不错的灵药，慕清霄都会购买下来，犹如雁过拔毛，等回去后倒是可以种植在庄园里。
<br/>　　随着时间的推移，一周的时间转眼即逝。
<br/>　　在这一周内，慕清霄与小医仙则在不同的城区中闲逛，可由于中域城区实在太大，想要全部转一遍，没有三两月估计是不可能的。
<br/>　　于是，慕清霄决定，即刻前往焚炎谷。
<br/>　　……
<br/>　　……
<br/>　　焚炎谷，坐落在中域西南方向的炙火山脉。
<br/>　　炙火山脉完全是有炽热的火山组建而成，放眼望去皆是千篇一律的赤红色景象，群峰之巅还不断袅袅升起阵阵白烟，偶尔还有岩浆流淌而出。
<br/>　　炙火山脉，即便是放眼整个中州也是拥有极强的名声，当然这都是因为焚炎谷坐落于此。
<br/>　　山脉中的环境虽有些恶劣，但由于连绵不绝的火山，空气中也充斥着一股极强的火属性能量，在这里修炼火属性功法，无疑事半功倍。
<br/>　　炙火山脉距离城区西南数千里之遥，对于其他人来说路途遥远，但对慕清霄来说却只需几个呼吸的时间就能抵达。
<br/>　　立于空中，慕清霄与小医仙就已经看到一条无限延伸，极其雄伟的赤红色山脉，并且周围空气中的能量也炙热起来。
<br/>　　望着眼前糟糕的环境，慕清霄不禁想起，仙剑位面鬼界中极乐世界里的景象，何其相似。
<br/>　　脚下看不到一丝葱郁，除了赤红色土地别无他色，地面上更是连草根都没有一颗。
<br/>　　对于空气中炙热的能量，慕清霄完全免疫，至于小医仙与彩鳞则感觉到些许不适，但运转心法后，不适的感觉也随之消散。
<br/>　　当然，除了炙热的能量与恶劣的环境，炙火山脉中并非没有生命。
<br/>　　只是生活在这里的魔兽，性情都极其的暴躁，危险性远比其他地方的魔兽要高的多。
<br/>　　就在这时，一道赤红色流光，自千米外的一座赤红巨山上疾掠而出，不多时就停在了慕清霄等人不远处的对面。
<br/>　　来者是一名身穿赤红长袍的中年男子，六星斗宗。
<br/>　　男子打量了一眼慕清霄与小医仙，拱了拱手道：“两位，前面乃是焚炎谷范围，禁制外人进出，前勿在往前了。”
<br/>　　慕清霄微微一笑，道：“听闻焚炎谷谷主正在寻找高阶炼药师，不知可有此事？”
<br/>　　“不错，我家谷主想要炼制一味等级不低的丹药，需要一名高阶炼药师配合。”
<br/>　　男子眼中闪过一抹精芒，狐疑道：“难道小友是一名高阶炼药师？”
<br/>　　“九品够不够？”
<br/>　　“呃……”
<br/>　　男子一怔，语气严肃道：“小友，莫拿我开玩笑，九品炼丹师只有中州圣城才有，并且也不可能来焚炎谷吧。”
<br/>　　慕清霄道：“此言差矣，我对你们焚炎谷的九龙雷罡火可是非常感兴趣，这可是炼药师一生所求。”
<br/>　　话语刚落，男子眼中闪过一丝微怒，刚要开口，身旁空间一阵扭曲，一名身穿火红长袍，发丝皆白的老者就出现在虚空中。
<br/>　　看到老者的一瞬间，男子呼吸都有些急促，恭敬行礼道：“唐炎见过老祖。”
<br/>　　此时，男子心中有些不敢置信，火云老祖已经近百年没有出过焚炎谷，今日为何现身，难道面前的两名年轻人身份不一般？
<br/>　　微微点头，火云老祖看向慕清霄，叹道：“小友如此年纪已有如此恐怖的实力，一枚九龙雷罡火对于你来说，可有可无吧？”
<br/>　　火云老祖心中也有些郁闷，若是平日里，他根本不会出焚炎谷，可谁让来者的修为不比他低呢。
<br/>　　甚至，以他二星斗圣的修为，居然完全看不透面前的青年，这就有些可怕了，难道是八族中人？
<br/>　　他们焚炎谷在中州虽有极高的地位，但比起神秘的远古八族，却是相差甚远。
<br/>　　一来是不能得罪，否则后果不堪设想，二来九龙雷罡火是焚炎谷的传承火焰，将之交出去，心中又有不甘。
<br/>　　慕清霄嘴角牵起一抹笑意，道：“不知焚炎谷可否忍痛割爱。”
<br/>　　“……两位里面请。”
<br/>　　火云老祖沉吟片刻，无奈道：“唐炎，让谷主来见我。”
<br/>　　说罢，火云老祖的身影就已经消失，留下一脸震惊的唐炎，慕清霄则搂着小医仙紧随其后。
<br/>　　穿过焚炎谷的空间结界，慕清霄跟在火云老祖身后来到焚炎谷内部，顿时眼前一亮。
<br/>　　焚炎谷内部并未如外界一样，郁郁葱葱的树木林立，山谷上瀑布在烈阳的照射下，如银色匹练悬挂而下……
<br/>　　有山有水，自成一界，倒也算得上是炙火山脉中的世外桃源了。
<br/>　　焚炎谷拥有斗圣坐镇，能够开辟出微型世界也并非难事。
<br/>　　来到焚炎谷中，空气中能量已经温和，其中包含的炙热狂暴能量已经全部消失的无隐无踪。
<br/>　　跟在火云老祖身后，走在焚炎谷内部，不时能够听到焚炎谷弟子们切磋武斗的声音。
<br/>　　不多时，穿过武斗场，顺着碎石道路来到焚炎谷后山，脚步一顿，前方有一幢隐于葱郁间的建筑物，其上有着武技阁三字。
<br/>　　说是武技阁，倒不如说这里是火云老祖修身养性的后花园。
<br/>　　不过，焚炎谷将武技阁安排在这里，确实无比的安全，没有任何比斗圣更有保障性的了。
<br/>　　来到武技阁旁的一栋竹屋，火云老祖拿起茶壶，道：“没有好东西招待两位，希望不要见谅才好。”
<br/>　　慕清霄坐下，小医仙紧挨在身旁，内心却是有些拘束，她隐隐能够听出，面前的老者似乎很强，恐怕也是斗圣级别的。
<br/>　　若非有慕清霄在，她在这种恐怖的老怪物面前能否保持镇定还是两说。
<br/>　　……
<br/>　　……
