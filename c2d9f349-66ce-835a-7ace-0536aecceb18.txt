　　在楚风眠的思考之中，楚风眠一行人，便是由雷绝的带领下，来到了这洞窟的最深处。
<br/>　　这洞窟之中的魔意，便是都从其中，传递出来的。
<br/>　　突然，众人停了下来。
<br/>　　“到了？”
<br/>　　“这是，祭坛？这些东西又是什么？”
<br/>　　在楚风眠的身边，无数惊异声突然响起，将楚风眠的思绪打断。
<br/>　　楚风眠看向前方，一眼望去，一座巨大无比的血色祭坛，位于这洞窟的最深处。
<br/>　　这血色祭坛，每时每刻之间，都在喷吐着无数魔意，化身为一尊尊魔头的样子，无数魔头，从中飞舞。
<br/>　　这整个洞窟之中的魔意，全部都是来自与这祭坛之中。
<br/>　　而在这祭坛周围，却是有着无数人，站在周围，似乎是在守护这座祭坛。
<br/>　　但是这些人的身上，却是毫无生机，反倒是眼神冰冷，浑身都散发这一股魔意，似乎是被魔意深种。
<br/>　　这些人，全都面目丑陋，狰狞到极点，令人看不出本来的样貌，一个个都如同死人一般。
<br/>　　“魔奴？”
<br/>　　楚风眠心中都微微惊讶，脱口而出道。
<br/>　　这些人，已经不是人了，而是一种半死不活的生物，魔奴。
<br/>　　这魔奴，便是由一些魔道强者，抓捕一些强者，将那些强者全部困住，然后用无上魔意影响，最终将其转化为只知道杀戮的魔奴。
<br/>　　这种魔奴之法，毕竟被禁制控制都要罪恶的多，禁制虽然控制，但是人还是人。
<br/>　　但是这魔奴之法，全是足以将人直接转化为人不人鬼不鬼的魔奴，失去意识，受人摆布。
<br/>　　这魔奴之法，本是属于九魔城的，在曾经数万年前，九魔城为未位列七大宗门的时候，九魔城便是大肆抓捕九域中的强者，天才，全部炼化为魔奴，受他们驱使。
<br/>　　在当时，九魔城更是组建了臭名昭著的魔奴军团，不知道是抓捕了九域多少的强者，天才。
<br/>　　一直到后来，九魔城加入了七大宗门之后，便是因为这魔奴之法，太过残忍，有伤天和，便是不在动用。
<br/>　　没想到这地魔门，在背地之中，居然是在利用这魔奴之法，制造魔奴。
<br/>　　“这祭坛，便是炼制魔奴的地方，如今北邙学院的任务，应该就是捣毁这祭坛，但是这些魔奴，为什么一直不动？”
<br/>　　楚风眠突然眉头一皱，察觉到了几分不对。
<br/>　　这些魔奴，应该在楚风眠一行人踏入到洞窟之中，便是察觉到了楚风眠一行人的踪迹才对。
<br/>　　但是这一路上，却是没有任何一尊魔奴出来阻止，似乎是被人，刻意的命令，来让这些魔奴不主动出手。
<br/>　　“这些魔奴的背后，还有着人，是谁？”
<br/>　　楚风眠的脑海中，突然灵光一闪，脸色猛然一变道。
<br/>　　“不好！”
<br/>　　“快退！”
<br/>　　“小子，现在想明白了？晚了！”
<br/>　　突然之间，一声冷笑，猛然响起，这声音，正是来自楚风眠的身前雷绝。
<br/>　　只见雷绝冷笑一声，大手一挥，周围的空间，便是突然被封锁起来。
<br/>　　通往洞府之外的通道，也都被彻底封锁。
<br/>　　“怎么回事？雷师兄？”
<br/>　　莽通看着雷绝，脸色也突然变了。
<br/>　　这到底发生了什么，他还不清楚，但是看眼前的状况，绝对是出现了极大的变化。
<br/>　　另外两人，也都是脸色慌张到极点。
<br/>　　“桀桀桀桀，雷绝，干的不错，这一次又带来了不少好货色。”
<br/>　　两道血色手臂，突然出现，随后，便是一尊黑袍人，从这周围的空间中，猛然现身。
<br/>　　这黑袍人出现的一刻，周围的空间，仿佛都受到了压迫，一股威压，顿时笼罩下来。
<br/>　　“生死境！”
<br/>　　感受着黑袍人身上恐怖的灵力，站在楚风眠身边的三人，都是脸色惨白到极点。
<br/>　　如今这出现的，居然是一尊生死境的强者。
<br/>　　“暗古魔尊，许久未见啊。”
<br/>　　雷绝看到这黑袍人出现，脸上倒是没有丝毫紧张，反而是看起来极为熟悉的样子。
<br/>　　“怎么样？这几人的资质，不错吧，用来炼成魔奴，也有着不错的实力吧。”
<br/>　　“那三个人，倒是极为普通，只能炼成最为普通的魔奴，不过这一个，倒是有些特殊，他的身上居然拥有真龙一族的血脉，可以将他炼成一尊魔奴王来。”
<br/>　　暗古魔尊，高高在上，看着楚风眠一行人的眼神中，便是就像是挑选货物一样。
<br/>　　货物，便是楚风眠一行人，买货人，是暗古魔尊，而最后的卖货人就是雷绝。
<br/>　　“雷绝！你居然跟地魔门的人勾结，要来残骸同门吗？”
<br/>　　“该死，我们都被这雷绝骗了！”
<br/>　　“走！快逃！逃出去，将他的事，告诉学院！”
<br/>　　事到如今，这到底发生了什么，只怕是一眼都能看出来了。
<br/>　　这雷绝，居然是勾搭上了地魔门，明面上是将楚风眠带来这里，完成人物，实际上却是要将楚风眠一行人，卖给暗古魔尊，炼成魔奴！
<br/>　　众人纷纷催动刚刚恢复的灵力，猛然向着洞窟外逃去，想要逃走。
<br/>　　任何人都知道，现在留在这里，便是只有死路一条！
<br/>　　“想逃？你们逃得了吗？”
<br/>　　雷绝冷哼一声道，身形一动，突然一拳，猛然打出。
<br/>　　这一拳之间，力量完全爆发而出，没有任何的保留，便是就要将楚风眠这一群人，给彻底镇压一般。
<br/>　　一拳之威，顿时莽通三人，一下子口中吐出了无数鲜血。
<br/>　　他们的实力，本就远远无法抗衡雷绝，现在更是因为在之前，灵力消耗一空。
<br/>　　如今他们就连反抗雷绝力量的能力都没有，一拳之下，险些都被直接镇压。
<br/>　　“你们这群废物，还不如被炼成魔奴，我还可以得到一笔好处，你们以为我为何会挑选你们？不过就算有着这点利用价值罢了，现在就算你们发挥价值的时候！”
<br/>　　雷绝眼神冰冷到极点，冷喝一身道。
<br/>　　“统统给镇压！”
<br/>　　“轰！轰！轰！”
<br/>　　随着这一拳的力量，其中拳风，顿时便是要将楚风眠一行人给彻底镇压。
<br/>　　这一股一股的力量，越来越强，几乎令人无法抵挡。
<br/>　　“该死！我们跟他拼了！就算是死，也不能变成这等人不人鬼不鬼的魔奴！”
