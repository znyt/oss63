　　天底下的一切，仿佛是没有任何的力量，足以抵挡这一剑的剑锋的。
<br/>　　小炎帝打出的无数武技杀招，在这一道剑锋之下，全然轰灭。
<br/>　　无数五灵宗圣子都是抬起头，看着眼前这惊天动地的一幕，脸色苍白无比，在这一剑之下。
<br/>　　哪怕是其中的余波，他们谁被卷入其中，都是必死无疑。
<br/>　　能够来到这佛寺之中的五灵宗圣子，最弱也至少是一尊古帝，实力及其强横，在大罗天星域之中都算是强者了。
<br/>　　可今日，却是没有人敢上前一步，这剑锋的余波，都足以将他们轰杀殆尽。
<br/>　　之前狂傲无比的小炎帝，如今也是脸色剧变，就算是他无论如何也无法想象到，楚风眠居然可以打出如此可怕的剑光。
<br/>　　在这剑光之下，纵然是九劫古帝只怕都会被当即轰杀，这已经是超出了小炎帝的想象之中。
<br/>　　像是小炎帝，可是接受过力量传承的天才，以八劫古帝的境界，足以击败九劫古帝。
<br/>　　这种实力拿出去，已经是名留历史的天才了。
<br/>　　可现在楚风眠的剑光，哪怕是一名九劫古帝，前来硬抗，都是必死无疑。
<br/>　　现在楚风眠的实力，已经是跟刚刚进入这佛寺之中的时候相比，截然不同了。
<br/>　　在佛寺之中，楚风眠之前猎杀了大量的魔人傀儡，得到了无数佛门舍利，来提升他的实力。
<br/>　　正好楚风眠还没有来得及试试他现在的实力，这小炎帝就送上门来了。
<br/>　　轰轰轰！
<br/>　　天空之上，响彻而起无数轰鸣声，这三阳烈炎剑的剑光，在斩破了小炎帝的无数杀招之后，其中的威力非但是没有减弱，反而是约战越勇。
<br/>　　漫天火焰，都被这剑光剿灭，这一道剑光直指小炎帝而去。
<br/>　　这小炎帝，既然是敢来针对楚风眠，那楚风眠也没有手下留情的必要了，楚风眠对待敌人，从来不会手下留情。
<br/>　　“灭世火龙！”
<br/>　　小炎帝看到剑锋的轰击而来，也是大声的长啸了一声，那灭世火龙，便是受到小炎帝的命令，来到了小炎帝的面前，打算来抵挡这三阳烈炎剑的剑光。
<br/>　　“无用的挣扎，今日在给你十条这样的灭世火龙，也救不了你！”
<br/>　　楚风眠看着小炎帝，眼神冰冷无比，说话之间。
<br/>　　三阳烈炎剑的剑锋，已经是斩杀在了那灭世火龙的身躯之上。
<br/>　　这灭世火龙，本就是以无比精粹的灭世帝炎所化，在三阳烈炎剑的剑锋斩在这灭世火龙身躯之上的时候。
<br/>　　灭世火龙，突然间的爆炸开来，化为了漫天火海，包裹在剑锋之上，来吞噬楚风眠的剑锋。
<br/>　　看着灭世火龙轰然破碎，小炎帝的脸色都是无比的难堪，这灭世火龙，有着足以媲美九劫古帝的实力，自然不是小炎帝可以随意凝聚而出的。
<br/>　　凝聚这灭世火龙，可是花费了小炎帝足足数十年的时间，更是加上了大量的诸天奇物，无数宝物。
<br/>　　才以这灭世帝炎，凝聚而出足以媲美九劫古帝的灭世火龙。
<br/>　　灭世火龙破碎，几乎是代表着小炎帝数十年的心血，就此毁于一旦，但是现在小炎帝也已经是别无选择。
<br/>　　为了对付楚风眠，小炎帝宁可破釜沉舟。
<br/>　　“小小火焰，也想要吞噬三阳烈炎剑的剑锋？”
<br/>　　楚风眠看这小炎帝的举动，从他的嘴角露出了几分不屑的笑容，楚风眠都没有出手，而是看着眼前的景象。
<br/>　　那灭世火龙爆炸，所化的火焰，是开始在吞噬这楚风眠的剑锋，但是很快，一团无色火焰，便是熊熊燃烧起来。
<br/>　　轰隆！
<br/>　　这无色火焰，越烧越旺，开始疯狂的吞噬这灭世帝炎，号称足以毁灭天地的灭世帝炎，在这无色火焰之下，被纷纷吞噬。
<br/>　　哪怕是炎帝，一尊半步天人境的强者，所凝聚而出的最强火焰，足以媲美九天天火的灭世帝炎。
<br/>　　在这真正九天天火，纯阳天火的面前，依然是不堪一击。
<br/>　　电光火石之间，这灭世帝炎，被全部吞噬，楚风眠的剑锋，威力更甚。
<br/>　　“纯阳天火！这种火焰不是以人力可以掌控的！你怎么可能掌控的了？”
<br/>　　小炎帝看着自己的灭世帝炎，被吞噬殆尽，他的目光也是无比惊骇，死死的盯着那无色火焰。
<br/>　　他已经是认得出，这无色火焰，到底是什么样的存在了。
<br/>　　纯阳天火，九天天火之中，最为纯粹，最为可怕的火焰，才足以吞噬这灭世帝炎。
<br/>　　看着灭世帝炎被全部吞噬，小炎帝如今几乎是要陷入了疯狂之中。
<br/>　　他的实力不如楚风眠，现在居然是就连火焰，都不是楚风眠的对手。
<br/>　　小炎帝，已经是破釜沉舟，底牌尽出，但是依然是无法撼动楚风眠。

<br/>　　“小子，你今日真的是要鱼死网破吗？你今日住手，我愿意跟你联手，去对付圣院宗，六绝宗的人。”
<br/>　　小炎帝死死的盯着楚风眠，他的眼神之中已经是没有多少的战意了，而是恐惧。

<br/>　　这在圣龙秘境的圣地之内，都是所向无敌，几乎是最为耀眼的天才之一的小炎帝，今日居然就这样败在了楚风眠的手中。
<br/>　　“楚兄，要不要收手？”
<br/>　　炎撼世不由的对着楚风眠询问道，与其是说劝解，不如说是询问，在见识到楚风眠的实力之后，炎撼世已经是坚定的站在了楚风眠这一边。
<br/>　　哪怕是五灵宗跟楚风眠为敌，他也选择站在楚风眠这一边，跟楚风眠一同对对抗五灵宗。
