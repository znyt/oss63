　　苏家最为强盛的，便是他们的锻造之术。
<br/>　　失去了锻造之术，这天兵苏家的没落，倒也正常了。
<br/>　　“前辈跟我苏家先祖，有着些交情？”
<br/>　　苏荷小心翼翼的向着楚风眠询问道。
<br/>　　对楚风眠的来历，他也是有些吃惊的，因为楚风眠所问的很多问题，都是曾经万年之前的事。
<br/>　　苏荷若不是曾经熟读过他们苏家的古籍的话，只怕也不一定会知道这些。
<br/>　　可是楚风眠怎么看也都只是一个二十岁左右的年轻人，根本不像是一个活了一万年的老古董。
<br/>　　这让苏荷也是极为好奇，楚风眠的身份，尤其是楚风眠的实力更是极为强大。
<br/>　　“我跟你们苏家，的确有旧，不过我的来历，你最好还是少打听清楚为好，不然你招来的灾难，比起那王家，还要恐怖千倍，万倍。”
<br/>　　楚风眠冰冷的开口道。
<br/>　　这话可不是在单单吓唬苏荷，现在楚风眠的身份，倒是没什么。
<br/>　　但是若是让人知道，现在的楚风眠便是万年之前的楚剑白的话，那七大宗门肯定都会不惜一切代价的想要抓捕楚风眠的。
<br/>　　苏荷知道了楚风眠的身份，带给她的，只有灾难。
<br/>　　“你有没有想要学习锻造之术的想法？”
<br/>　　突然间，楚风眠向着苏荷开口道。
<br/>　　“你的资质，不适合学习武道，从其一生都无法再武道上有着任何造诣。”
<br/>　　“锻造之术？前辈也懂得锻造之术？”
<br/>　　苏荷一楞，随后脸色大喜道。
<br/>　　她也一直知晓，他们苏家最为强盛的时候，便是都因锻造之术而起，但是锻造之术，哪里容易学习。
<br/>　　在各大宗门，各大世家之中，都是不传之秘，苏荷现在的身份几乎便是跟散修没有任何的区别。
<br/>　　根本没有可能接触到锻造之术。
<br/>　　“不错，略知一二罢了，念在你的苏家之人，这锻造之术，也正好就由你继承下去吧。”
<br/>　　楚风眠平静开口道，突然之间，楚风眠一掌直接向着苏荷的脑袋拍了过去，一股灵力轰击这苏荷的脑袋。
<br/>　　苏荷本能的想要后退抵挡，可是这一股灵力，却是根本没有伤及到她，而是直接的钻入到了苏荷的脑海之中去。
<br/>　　无数关于锻造之术的记载，全部都是充斥着苏荷的脑海之中。
<br/>　　“这些就是你们苏家的锻造之术，当初那些老古董不让我看，还是我悄悄翻阅进去看的，现在正好全部传授给你，也算是让你们苏家有着继承了。”
<br/>　　楚风眠不由的感叹了一声道。
<br/>　　他的锻造之术，都是从天兵苏家之中传来的。
<br/>　　这本来就是苏家的不传之秘，当时楚风眠还是跟青鸾算计半天，偷偷潜入到了苏家的藏经阁中，才将这锻造之术学来的。
<br/>　　现在这锻造之术，又传承给了苏荷，也算是还了当年的一份人情了。
<br/>　　“我们苏家的锻造之术！”
<br/>　　苏荷闭眼，自信的翻阅这脑海之中的无数书记，她的脸色，逐渐变的平静下来，整个人都是进入到了一种领悟之中去。
<br/>　　瞬息之间，便是领悟起了这锻造之术的真谛。
<br/>　　这么快，倒是让楚风眠都有些惊讶。
<br/>　　“不愧是有着天兵苏家的血脉，看来苏家的人，都是锻造之术一脉的天才，这样下去，将来这苏荷，必然也足以成为一代炼器大师了。”
<br/>　　楚风眠不由的感叹了一声。
<br/>　　他也不着急，就在旁边，安静的等待着，将周围布置下了隐秘的禁制，让其他人无法察觉到他们二人的存在。
<br/>　　一天，两天，三天。
<br/>　　足足一直是过了三天的时间，苏荷紧闭的双眼，才缓缓睁开。
<br/>　　“学的怎么样了？”
<br/>　　楚风眠看到苏荷睁眼，直接开口道。
<br/>　　三日时间，对楚风眠便是一闪而过，他如今在归墟天之中，实在是得到了太多的机遇，还没有来得及去仔细消化一下。
<br/>　　尤其是从云羽天宫之中所得到的那些剑术，还是无上剑气中记载的剑术，都是极为玄妙，就算是现在已经是被融入到了九域剑术之中去。
<br/>　　楚风眠也必须在再度的去领悟，积累底蕴。
<br/>　　不过想要完全将这些机遇全部消化，并非是一朝一夕的时间，所以楚风眠也不急于这一时。
<br/>　　“刚刚入门，还需要时间去尝试。”
<br/>　　苏荷缓缓的开口道。
<br/>　　锻造之术，看似简单，但是实际上在九域之中的炼器大师，比起炼药大师还要少的多。
<br/>　　因此这天级灵器，在现在的九域之中，每一件还都是至宝。
<br/>　　就算是在当初的天兵苏家，真正足以炼制天级灵器的人，也都寥寥无几，也算是当初苏家的天才，会被重点培养的人物。
<br/>　　这便是足以看得出，锻造之术，炼器之道到底是有着多难。
<br/>　　苏荷能够在这三日之内，刚刚入门，已经是超出了楚风眠的想象了。
<br/>　　“不急，你不是想要灭了那王家吗？等灭了那王家之后，你有的是时间来修炼这锻造之术。”
<br/>　　楚风眠随意开口道。
<br/>　　“走吧，带我去青河边城。”
<br/>　　“是。”
<br/>　　苏荷急忙点了点头，恭敬的带着楚风眠继续前进。
<br/>　　又是半日时间。
<br/>　　在经历了无数传送法阵之后，楚风眠跟苏荷，终于是来到了一处河畔旁。
<br/>　　在前方，正是有着一座巨大无比的城池，依靠青河而建，正是那王家所在，青河边城。
<br/>　　这青河边城，比起之前路过的东升城来说，倒是要大的多。
<br/>　　楚风眠已经是感觉得出，在这青河边城之中，一共有着三名生死境武者，两名生死境二重，一名生死境五重。
<br/>　　其中那生死境五重的武者，应该就是苏荷所说的那王家家住，道元公了。
<br/>　　“前辈，这王家的家住道元公，可是生死境的人物，您还要小心一点。”
<br/>　　来到青河边城外，苏荷还不由的提醒了一声。
<br/>　　“无妨，生死境的人物不过都是蝼蚁一样，就算是超越生死的武者，都算不得什么。”
<br/>　　楚风眠轻描淡写的开口道。
<br/>　　“放心，区区一个王家而已，既然我答应了你，自然会将其彻底灭门。”
<br/>　　“生死境，蝼蚁？”
<br/>　　听到楚风眠的话，苏荷都是愣住了。
<br/>　　生死境，那可是超越圣者的存在，在这终焉王朝，都是地位极高，足以封公。
<br/>　　可现在到了楚风眠的口中，居然是如此不堪？
<br/>　　就在苏荷还在惊讶之际，楚风眠已经是御空而立，站在了这青河边城前。
