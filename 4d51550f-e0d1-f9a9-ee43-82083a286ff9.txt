　　第315章
<br/>　　图像摊开在书案上，程大雷和苏樱都探出脑袋往上面看，没过多久，两人脸上的表情都像吃了一碗盐那样扭曲。
<br/>　　“像么？”程大雷问。
<br/>　　“像吧。”李婉儿。
<br/>　　“我怎么觉得不太像？”
<br/>　　“什么叫像，这根本就是。”徐神机像立了大功一样得意。
<br/>　　这幅图像上坐着一只蛤蟆，蛤蟆头顶一个圆环状物事，确实像极了蛤蟆寨大旗上的天蟾吞月图，可若说完全一模一样，也是不对，首先蛤蟆的颜色不同，然后这幅图月亮周围还有火焰状图像，更像是太阳。
<br/>　　天蟾吞月图在帝国也不罕见，是常用的祈福图案，并不是程大雷盖章申请专利独一份的。
<br/>　　程大雷盯着这张图，脸上肌肉扭曲变形，半晌，他狠狠一拍桌子，愤怒道：“这是神经病吧！”
<br/>　　现在正义教第一个向帝国举起了反旗，已经是朝廷的眼中钉肉中刺。你愿意崇拜什么崇拜什么，哪怕是石头，野狗，这些都是可以的么。有没有必要搞得和我的logo一模一样，如果这个时代有商标保护，我非告得你破产信不信。
<br/>　　假若朝廷觉得蛤蟆寨是正义教的子公司，派大兵来打，先不说打得过打不过，这无端端给自己添了多少麻烦。
<br/>　　“回头赶紧把大旗撤下来，别搞得这么招摇。”程大雷气鼓鼓的吩咐一句。
<br/>　　“我待会儿就让人办。”徐神机道：“酒楼还得到一本小册子，是正义教传教用的，你看看。”
<br/>　　程大雷接过这本册子，见不过掌心大小，里面还是手抄本的。程大雷翻开看着，越看脸上表情越难看。
<br/>　　这正义教信奉的神明叫做蛤蟆大王……就冲这四个字，就透着一股没文化的劲儿。据册子上说，这蛤蟆大王是开天辟地时，从盘古大神的左眼中诞生。
<br/>　　所以蛤蟆魔君的辈分和女娲、三清，佛祖这些人的辈分是一样一样的。
<br/>　　盘古大神的左眼中诞生蛤蟆大王，居于月宫之上，右眼中诞生赤蛇魔君，居于太阳真宫。
<br/>　　合着几千年历史，就是一条蛇和一只蛤蟆打架，蛤蟆打赢了，就风调雨顺，太平盛世。赤蛇打赢了，就连年战火，民不聊生。
<br/>　　在一百年前呢，赤蛇使了一条毒计，用酒把蛤蟆魔君灌醉了，而赤蛇的血裔坐了龙庭。也就是李氏一族……
<br/>　　程大雷看了一眼李婉儿，啧啧，没想到你是一条美女蛇呐。
<br/>　　然后正义教的目的，就是用自身的鲜血赶走赤蛇，唤醒蛤蟆大王，让他巨口吞净世间污秽，还光明于人间。
<br/>　　……
<br/>　　程大雷越开嘴巴咧得越大，见过扯淡的，还真没见过这么扯淡的。编造神话的时候，咱就不能用点心么，大鹏金翅鸟，赤日麒麟兽……这些都可以拎过来当神明么。你有必要非崇拜一只蛤蟆么，而且还搞得和我撞名，很拉仇恨的好不好。
<br/>　　程大雷将册子丢给李婉儿，自己低头琢磨着。还真不能让朝廷觉得自己是和正义教一伙的，他吩咐徐神机，以后在山寨少提蛤蟆大王的外号，也正好给自己改个外号。
<br/>　　比如玉面飞龙，天外飞仙，西北吴彦祖之类的，又酷又有型，听上去就像高手。
<br/>　　李婉儿读着小册子，也露出和程大雷一样的表情。
<br/>　　“好啦。”程大雷摆摆手：“一看就是群没文化的乌合之众，成不了大事。”
<br/>　　看罢正义教的教旨，程大雷已经彻底对这股新兴势力失望，打心眼里开始鄙视这帮人。他已决定不对正义教特意上心，等秦蛮将苏樱接回来，自己就安心过自己的小日子。
<br/>　　秋去冬来，便又是一年，大家夜以继日的忙碌，终于赶在冬天之前，完成了山寨建设。
<br/>　　冬天来了，草原上的商路也暂时停了，给蛤蟆寨干活的工人拿着工钱离开，他们也开始筹备今年过年的日子。有从蛤蟆寨得到的这笔工钱，今年当能过一个富年。
<br/>　　如今整座岛上，完全是一座军事堡垒。岛外围的河上，布着铁链暗礁，大船如果不懂水文，陷进去后便出不来。
<br/>　　岛上遍布箭楼，上面都布置着重弩，可以在五百步外发动攻击。岛上的民居、厨房、铁匠铺、点将台，织造坊等等一应俱全。程大雷还特意给自己修了一间办公室，当然也可以叫做书房。
<br/>　　最大的工程在填水造桥，在蛤蟆岛距离陆地最近处用山石铺路，再修一座木桥，可以允许马匹通过。但也可以随时收起来，挡住敌人。
<br/>　　程大雷走过崭新的蛤蟆寨，脸上露出满意的神色，毕竟这三级山寨的建设图纸是系统给的，不然如何防御，如何布置，程大雷也不精通。
<br/>　　嘟，三级山寨达成。
<br/>　　当程大雷在岛上走了一遍后，收到了系统提示，然后是一连串的系统奖励。
<br/>　　嘟，随即奖励山寨技能：固防（山寨防御恢复时间减半）
<br/>　　嘟，随即一人隐藏属性升级
<br/>　　嘟，银眸隐藏属性升级，铁力——豹体（体力值提升，力量提升，敏捷提升，痛感减半）
<br/>　　这条提示令程大雷怔了怔，他只知道技能可以升级，没想到隐藏属性也可以升级。这随机奖励恰好落在银眸身上，是她距离自己比较近的缘故么。
<br/>　　嘟，拥有者可以随机提升一人隐藏属性
<br/>　　这条系统提示令程大雷心中一喜，很明显隐藏属性要比技能管用，只是用在谁身上，却是需要好好想想。
<br/>　　嘟，系统商店开启。
<br/>　　程大雷松口气，建设三级山寨，最主要的目的，不就是为系统商店么。只要自己开了系统商店，系统才算是完全体，到时候还不是神挡杀神，佛挡杀佛。
<br/>　　程大雷交纳一万恐惧值的进门费后，意识进入了系统商店，可看到贩卖的商品时，程大雷却愣住了。
<br/>　　并不是火药，而是……衣服。
<br/>　　皮甲（普通）：防御提升10，体力提升3，敏捷提升3，售价：一万恐惧值。
<br/>　　程大雷无语，这皮甲的价值肯定是不能和火药相比，可为什么是皮甲呢。莫非是地理原因，在青牛山的系统商店就是火药，在这里就是普通皮甲。
