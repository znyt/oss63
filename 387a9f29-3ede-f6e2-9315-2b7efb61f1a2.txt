　　顶级包厢中，灰袍中年目光冷冽的盯着一级贵宾包厢中，坐在软椅上的慕清霄，杀意毫不掩饰。
<br/>　　慕清霄鸟都不鸟他一眼，区区一个斗灵，能掀起什么浪来。
<br/>　　冷哼一声，灰袍中年也不再加价，若非这里是黑印拍卖行，容不得他放肆，否则早就要了后者的小命。
<br/>　　随着灰袍中年的退出，拍卖场中再也没有人加价，定音锤落下，七彩韵灵果以八百二十一万的天价，被慕清霄拍获。
<br/>　　望着一旁的小医仙，慕清霄捏了捏她的玉手，道：“七彩韵灵果效果虽不错，但毕竟只是六品丹药，等往后我在给你找更好的。”
<br/>　　闻言，小医仙微微颔首，内心满是感动，微不可查的靠在他肩膀上。
<br/>　　......
<br/>　　......
<br/>　　片刻后，待女恭敬的将银盘托了上来，慕清霄剑眉一挑，感觉到一股颇为强烈的气息，四星斗皇，看来压轴物品要出来了啊。
<br/>　　老者脸色变得严肃，拍卖场中窃窃私语声也消失不见，所有目光都汇聚在拍卖台上，等待本次拍卖会的重头戏。
<br/>　　血宗少宗主，范凌眼中闪烁着兴奋的光芒，苍白的脸庞上浮现一抹红润，虽然七彩韵灵果没得到，但接下来的东西，他必须不择手段的弄到手。
<br/>　　不仅是血宗，天蛇府，黑骷墓也是严肃起来。
<br/>　　“啪啪！”
<br/>　　拍卖台上，老者严肃的拍了拍双手，随着掌声响起，拍卖台边缘，忽然发出一阵清脆的声响。
<br/>　　旋即，一圈墨黑色的栏杆，渐渐升起，最后成圆形状将拍卖台笼罩，严实无比。
<br/>　　“诸位请不要在意，这只是本拍卖行的一些防护措施，接下来的东西比较贵重，以免出现任何意外，这圈牢笼栏杆，是由黑陨打造，即便是斗皇强者，短时间内也难以破开。”
<br/>　　话语刚落，老者向四周扫了一眼，同时特地扫了一眼贵宾包厢与顶级包厢，言语间的意思，不言而喻。
<br/>　　对于老者特地提醒，某些势力都淡然一笑，也没有太过在意。
<br/>　　在往年的拍卖会中，出手抢夺拍卖品的事情，也并非没有发生过，毕竟这里是黑角域，正所谓林子大了，什么鸟都有，更何况在这种秩序混乱之地。
<br/>　　八扇门会如此谨慎，实属正常，否则，若在自己的地盘上，东西被别人抢夺了去，他们还有何颜面在黑角域立足。
<br/>　　此时，老者已经接过银盘，银盘中同样放着一块千年寒冰玉盒。
<br/>　　千年寒冰玉盒其面散发出一阵淡淡的白雾，其中物品朦胧，根本看不出是何物，但能放在玉盒中，且让老者小心翼翼的，价值必然不俗。
<br/>　　随着千年寒冰玉盒的出现，拍卖会上所有强横势力，都猛地挺直了身子，眼中带着不加掩饰的贪婪，紧紧盯着玉盒。
<br/>　　没有理会贪婪的目光，老者手掌略微有些颤抖，将银盘小心翼翼的放在拍卖台上，将盒盖掀开，顿时一股金光猛地爆射而出。
<br/>　　突如其来的金光，直接将整个拍卖场都照耀的通透起来，一些措不及防的修炼者，甚至习惯性的闭上双眼。
<br/>　　慕清霄虽隐藏在黑袍中，却不影响视线，目光锁定拍卖台上的愈合，其中，一枚龙眼大小的金色丹药，正安静的躺在其中。
<br/>　　丹药表面极其圆润，两丝金色气流在丹药内部流转不定，仔细看去，气流外表像是两条金龙，细微的龙吟声，透过空气扩散而出，使得所有人的灵魂都在这股威压下颤抖。
<br/>　　慕清霄眼中闪过一丝精芒，低喃道：“七品丹药，阴阳玄龙丹，整个黑角域能够炼制出七品丹药的，估计也就韩枫了吧。”
<br/>　　要说他为何能一眼认出丹药来，那是因为当初离开萧家的时候，药尘将千年前失传的《朱子率》传给了他。
<br/>　　《朱子率》是千年前失传的特殊丹法，珍贵程度已经不是区区金币能衡量的，其中拥有斗气大陆最特殊的炼丹方式，除了帝药，各种丹药都有记载.......
<br/>　　根据其上的记载，丹药内部，只要丹气凝成灵形，这就是七品，或者更高等级丹药方才具备的现象。
<br/>　　自阴阳龙玄丹出现后，整个拍卖场，都在若隐若现的龙吟声中，陷入一片死寂。
<br/>　　在场的客户不清楚这枚丹药的信息，不过先前的金光，与若隐若现的龙吟，都示意着丹药的不凡，都不由的露出垂涎之意。
<br/>　　相对于不清楚丹药作用的，黑角域强大的大势力，却是在此全都精神抖擞起来，许多强者，甚至因为情绪波动，导致体内斗气不受控制的溢散出来。
<br/>　　拍卖台上，老者目光在全场扫过，满意的点了点头，指着千年寒冰玉盒中的金色丹药，笑吟吟的说道：“诸位，这是本拍卖行本次的压轴物品，七品丹药，阴阳龙玄丹！”
<br/>　　“哗！”
<br/>　　随着老者话语落下，全场寂静，寂静过后就是震耳欲聋的哗然，语无伦次的惊呼声就笼罩整个拍卖场。
<br/>　　无数赤红的双眼，携带者急促的呼吸声，死死盯在金色丹药上，七品丹药，这种等级的丹药，在场几乎九成的人，一辈都都没资格见到。
<br/>　　却不想到，传闻中的高阶丹药，居然出现在了他们的视线中。
<br/>　　“想必，诸位都没有听说过阴阳龙玄丹的名头。”
<br/>　　望着拍卖场中许多迷茫的神色，老者神秘一笑，道：“阴阳龙玄丹，位列七品，据说想要炼制此丹，首先便需要两头死亡时间不超过七天的龙类魔兽魔核，并且魔核等级，至少六阶。”
<br/>　　老者的一席话，使得周围迷茫的目光，瞬间转为骇然，那岂不是两位斗皇强者！
<br/>　　“据说此丹练成时，天现异象，一阴一阳两道龙魂自鼎中直冲云霄，互相缠绕，最后凝聚丹身。”
<br/>　　说话的同时，老者自己眼中都露出无尽的狂热：“阴阳龙玄丹，并非直接作用于使人提升实力，而是，破而后立！”
<br/>　　慕清霄微微一笑，破而后立的意思，他比谁都明白，正因为他亲自体验过。
<br/>　　当初在地球，他吸收神药中的一部分力量后，渡化神劫，在神药中蕴含的力量下，破而后立，最终突破化神。
<br/>　　甚至，血脉完全相融，最后诞生神凤化身的神通。
