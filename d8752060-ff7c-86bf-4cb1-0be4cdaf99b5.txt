　　对于鬼界的信息，慕清霄了解的不多，所谓的双煞鬼王也完全没听说过，自然不可能一口答应。
<br/>　　“双煞鬼王的实力如何？”
<br/>　　“两百年前，相当于仙门修士合体六重的修为，两百年过去，他们占据阴寒域，实力恐怕有所提升。”
<br/>　　若是只有一位鬼王，天鬼皇当然不会忌惮，但天煞鬼王是两名鬼王，实力似乎都不比他弱。
<br/>　　两者联合在一起，足以让任何一名鬼王忌惮。
<br/>　　鬼王是鬼界霸主，双煞鬼王又是兄弟，自然不可能内讧，其他鬼王也不会闲着没事去招惹他们。
<br/>　　不过，阴寒域是距离鬼城最近的领域，虽不是鬼界中阴气最浓郁的地方，但也算的上一处鬼修宝地。
<br/>　　最重要的是，除了双煞鬼王兄弟，其他鬼王似乎都比较难惹。
<br/>　　比如火鬼王，极乐世界有岩浆作为天然屏障，不管是他或者他的部下都无法渗入其中。
<br/>　　而且，火鬼界的地点是在鬼界最深处，那种地方距离地面太远，太过炙热，不管是天鬼皇或者其他鬼王，都不喜欢那里。
<br/>　　至于森罗域中，阎罗鬼王实力更是强的离谱，连火鬼王曾经都是他的手下，只是现在翅膀硬了，有些不听话了而已。
<br/>　　除了隐藏在暗中的鬼王，阎罗鬼王就是鬼王中最强的战力，天鬼皇自然不敢打他的注意。
<br/>　　见慕清霄陷入沉思，天鬼皇也没有打扰，坐在黑龙宝座上，露出一副享受的模样……
<br/>　　根据天鬼皇叙述的信息，两百年前，双煞鬼王兄弟就是合体六重的鬼修，身在宝地百年，两者不可能原地踏步。
<br/>　　不过，慕清霄并不担心，大乘以下的修士，在他眼中皆是蝼蚁。
<br/>　　除非，双煞鬼王拥有类似于悟道菩提子的东西，否则根本不可能打破壁障，更何况，双煞鬼王若是突破了，绝对不可能如此安份。
<br/>　　鬼王都不甘屈居于他人之下，火鬼王就是最好的例子，天鬼皇亦是如此。
<br/>　　“可以，我替你解决双煞鬼王，你带我进入鬼界。”
<br/>　　鬼界是各族禁地，除了鬼族，根本无人知晓如何进入其中，鬼族更不得带人类进入其中，此乃大忌。
<br/>　　由此可见，天鬼皇为了达成目的，孤注一掷。
<br/>　　“成交。”
<br/>　　得到慕清霄的承诺，天鬼皇并未怀疑，在这种层次的强者对承诺都非常的注重，反悔的几率不大。
<br/>　　“想要进入鬼界，需要通过专门的传送阵，本座这就命人安排，阁下暂且在殿中休息几日，等一切都准备好，本座会通知阁下。”
<br/>　　“也好。”
<br/>　　微微点头，慕清霄道：“给我准备一些鬼丹，这种东西对你来说，价值不高吧？”
<br/>　　天鬼皇想也没想就点头同意，命令鬼皇宫中的侍女准备好休息的地方，并且将鬼丹送到，待女则留下服侍慕清霄。
<br/>　　同时，与鬼丹一同送到的，还有一株养魂花与其他高阶灵药。
<br/>　　看到养魂花的瞬间，慕清霄就知道，街边摊位上的老者，估计是天鬼皇的眼线。
<br/>　　对此，慕清霄并没有感到不快，后者能将养魂花与一堆高阶灵药全都无偿送给他，可见非常的有诚意。
<br/>　　至于鬼丹则有数千颗。
<br/>　　鬼丹也分三六九等，越是高级的鬼丹，其中蕴含的阴寒能量就越浓郁，价值也就越高。
<br/>　　望着空间玉瓶中的鬼丹，慕清霄愣愣出神，脑海中闪过一抹白色倩影。
<br/>　　回过神来，将鬼丹与灵药收进储存空间，慕清霄瞥了一眼待女，顿时觉得索然无味，命她们在门外等候。
<br/>　　旋即，独自盘坐在床榻上，开始运行心法。
<br/>　　鬼皇宫中的侍女自然都是有些姿色的，只是慕清霄现在，对那方面没有太多的想法而已。
<br/>　　若是平时，吃干抹净拍拍屁股走人，这种不用负责的好事，慕清霄并不是干不出来，现在有更重要的事情要做。
<br/>　　灵魂悬浮在阴阳玉中，望着其中的赤红色长剑，慕清霄嘴角牵起一抹笑意。
<br/>　　等拿到火灵珠与水灵珠，并且将焚寂剑炼化，聚时在配合《黑凤化身》，天上地下便在也没有他去不得的地方。
<br/>　　……
<br/>　　……
<br/>　　与此同时，鬼皇宫正殿中，殿中正站着两道身影，一男一女，皆是有着很高的修为。
<br/>　　这时，其中颇为俊朗的青年开口道：“父亲，孩儿觉得此事欠缺考虑，人族修士伪君子居多，根本不值得信任。”
<br/>　　闻言，坐在魔龙宝座上的天鬼皇目光看向靓丽女子：“魉妹，说说你的想法。”
<br/>　　男女正是天鬼皇的子女，魉妹与天魑皇。
<br/>　　“魉妹觉得，爹爹肯定有自己的想法，魉妹一切遵从爹爹的指示。”
<br/>　　天鬼皇满意的点了点头，道：“行了，本座既然已经决定，此事就没有商量的余地，你们两个去让人安排，启动传送阵。”
<br/>　　“是，父亲。”
<br/>　　天魑皇虽不信任慕清霄，但天鬼皇决定的事情，他也不敢忤逆，至于魉妹，则一直保持着平淡的模样。
<br/>　　待魉妹与天魑皇离去，天鬼皇一叹，他何尝不懂后者说所。
<br/>　　当初，他被困在锁妖塔中八年可不是白困的。
<br/>　　当然，这也令他结识了许多值得信任的伙伴，只是那些伙伴已经失去了踪影。
<br/>　　在他的印象中，人族确实狡诈多端，但也不乏有正人君子。
<br/>　　看到慕清霄的一瞬间，天鬼皇潜意识里就觉得后者值得信任，特别是那种出尘的气息，或许他本人都没有发现这一点。
<br/>　　令他做出决定，还有一个尤其重要的因素。
<br/>　　那就是身为鬼王的他，拥有合道七重鬼修境界的一方霸主，居然完全看不透慕清霄的实力。
<br/>　　虽长得俊逸如画，但当他站在自己的面前，又如同普通的在不能普通的人类，连一丝灵气波动都没有。
<br/>　　能将自身气息内敛到这种地步，甚至让他看不透，后者的实力就有点令人心惊了。
<br/>　　所以，他在清楚后者的目的后，便顺水推舟。
<br/>　　他冒险带慕清霄进入鬼界，慕清霄则帮他斩杀双煞鬼王。
<br/>　　虽然各怀目的，但并不冲突。
<br/>　　而且，天鬼皇非常清楚，哪怕他不这样做，慕清霄也会进入鬼界，只是他的下场会比较凄惨而已。
<br/>　　……
