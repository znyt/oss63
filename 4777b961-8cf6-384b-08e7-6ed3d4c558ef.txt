　　陈少爷将狙击枪打出了机关连射，不要命的疯狂射击，甚至将自身承受力都抛在之外，就是这种疯狂的子弹飞射，颗颗命中同一个点的可怕命中率，一下子让这只丧尸头领受到了极大创伤。
<br/>　　第一发子弹在被后续多发子弹的持续推进之下，一下子冲入了更深的血肉，黑色的血在不断溢出，这头丧尸头领的内脏也在极短的时间内被子弹填满！
<br/>　　丧尸头领本想主动出击的行动顿时受限，它感受着自身躯体在承受着巨大危险，因为那后背的中弹处第一发虽只是陷入皮肤的轻伤，但后续的猛射之中，那些子弹已经冲入了它的心脏。
<br/>　　对于丧尸来说已经腐烂的内脏，在五阶之后会重新组织再生，等阶越高的丧尸身躯完整率越高。
<br/>　　而这只九阶的丧尸头领，不仅五脏六腑俱全，更有了神经末梢和其余越发类似于人类的东西，包括了头发。
<br/>　　而疼痛感，也是如此！
<br/>　　这强烈阻挡了这只丧尸的行动能力和当下思维，也让它失去了最后反击的机会。
<br/>　　因为楚涵，已经在陈少爷疯狂射击的时候，用最快的速度狂奔而来！
<br/>　　他从未停止过狂奔而来的行动，而那二十米的距离，在当下更断绝了这是一个失误的可能。
<br/>　　二十米的距离并不是楚涵不小心跑过头，而是刻意如此，他的目的也根本不是让陈少爷射·出一发子弹，而是此时此刻能让丧尸头领行动迟缓的无数发。
<br/>　　并且，射击还在继续。
<br/>　　这与一开始楚涵在丧尸头领附近导致陈少爷无法扣下扳机的情况不同，对于神之狙击来说第一发与接下来的任何一发也不同，眼下只要丧尸头领依旧在那个位置，哪怕楚涵与丧尸头领脸对脸，陈少爷也依然可以开枪。
<br/>　　因为第一发子弹的命中，已经奠定了射击轨迹的基础，对于陈少爷这种逆天的能力来说，只要第一发射中，接下来无论多少发他都可以命中同样的位置。
<br/>　　所以才会有后续的一连串子弹出现，才会有让丧尸头领从轻伤瞬间变成重伤的转折！
<br/>　　噗！
<br/>　　终于，那第一发命中的子弹从丧尸头领的胸前溢出，‘叮当’一声脆响落于地面。
<br/>　　那是被其后多发子弹的推进，一路从后背处穿过心脏再挤压出，最后从正面穿膛而过！
<br/>　　无法想象用这种方式落于地面的第一颗子弹背后，有多少子弹已经聚集在丧尸头领的胸膛之内，那可怕的数量又是如何在这眨眼间的时间里，被陈少爷一举射·出？
<br/>　　这一奇迹就是如此发生，瞬间改变了当下格局！
<br/>　　这时候的楚涵已经跑到了最佳的进攻距离，修罗战斧在他的手中龙虎生威，终于在此时此刻给这头丧尸头领带来了之前从未有过的威胁感。
<br/>　　“爆炎斩！”
<br/>　　一声大喝，从楚涵的口中咆哮而出，他狞着比丧尸更可怕的身躯，‘轰’的一声爆发了体内的能量。
<br/>　　呈现出实质感的能量浓雾好似黑色的火焰，一路攀爬至修罗战斧的斧刃，在其上幽幽燃烧。
<br/>　　陈少爷的射击也在楚涵爆发的一瞬间终止，他站在极远距离的高楼顶端边缘，双手颤抖的放下绝命狙击枪，而后一屁·股坐了下来，开始静静的等待那个未知的结果。
<br/>　　他已经在这最重要的瞬间做尽了一切，甚至接下来的三天内，恐怕都难以再举起任何一把枪。
<br/>　　陈少爷不是进化者，哪怕八阶也与其余楚涵或者徐锋等人的体能相差很大，这是身为强化者的最大劣势，哪怕手中这第三代绝命狙击枪已经被和上最大程度的减轻了后坐力，但陈少爷在用如此段的时间开了不知道多少抢，依然大幅度超过了他当下所能够承受的极限。
<br/>　　如果楚涵不是跑开了二十米而是更远，陈少爷或许还会在更多的时间里继续射击，但那样的后果，可能就不是三天不能开枪那么简单。
<br/>　　这个从末世初期就一路跟随楚涵的大胖子，此时此刻忽然一笑叹了口气：“老大还给我留了后路。”
<br/>　　陈少爷很了解楚涵，比任何人都了解，也比任何人都信任楚涵。
<br/>　　楚涵要说明天太阳是从西边升起，陈少爷也会坚定的认为明天太阳升起的方向就是西边！
<br/>　　也就是这种信任，才让两人有了这一回的神级配合。
<br/>　　二十米的距离，不仅是能给丧尸头领多次失去机会的最合适距离，也是能让其重伤的距离，更是楚涵一路跑来时，陈少爷开枪的极限时间。
<br/>　　一米都不能多，一米都不能少！
<br/>　　这一精准的范围，在楚涵做出最初的决定之时，就已经完美定下！
<br/>　　待在另一处楼层内观战的蒋天庆，在陈少爷的枪声停下之后，终于看懂了这一战的最关键点，也总算明白了为何楚涵会选择二十米这一距离。
<br/>　　失误？跑过头？没算清？乱了节奏？
<br/>　　若说这些会出现在楚涵身上，蒋天庆断然不会相信，这其中一定有什么他没想到的东西。
<br/>　　而这个没想到，终于在陈少爷退出这场战斗的时候，浮出·水面！
<br/>　　震惊不足以形容蒋天庆的心情，狼牙战团成立了这么久，楚涵总是站在最高的位置指点江山，战术谋略他们见得多了，但这种单打独斗的战斗，对象是极其强大敌人的战斗，却很少有人见过。
<br/>　　蒋天庆同样如此，这是他第一次观战楚涵战斗强敌！
<br/>　　也是第一次知道，原来单打独斗的眨眼间，有这么多的技术，有如此多的诡计！
<br/>　　惊叹的情绪，与下方战斗场地中的一声巨大爆破声一同响起，那是楚涵奔跑而来瞬发的爆炎斩的引爆。
<br/>　　丧尸头领被陈少爷多发子弹穿过胸膛，破开了心脏，让其受了重伤，但这不足以致死。
<br/>　　丧尸头领哪怕再如何特殊，都还是丧尸，只要是丧尸，都必须破坏大脑才算真正的杀死！
<br/>　　而显然极限射·出第一枪并没有命中头部的陈少爷，无法再有机会找出丧尸头领的致命部位，只有最大可能的将其重伤。
<br/>　　最后的杀死，还是需要楚涵出手！
