　　一道声音，突然是从雕像之中传了出来。
<br/>　　这声音，极为清澈，却足以令人听见有着一种稳重，极为安心的声音。
<br/>　　“大师兄？”
<br/>　　听到这声音的一瞬间，楚风眠的神情都是激动无比。
<br/>　　这正是他的大师兄，剑凌的声音，虽然是隔了万年时间，但是如今楚风眠听到，依然是感觉到极为亲切。
<br/>　　曾经楚风眠的无数师兄弟，在师门之中，剑道之主高高在上，事务繁忙，很多事情当时都是剑凌负责的。
<br/>　　尤其是许多年少的师弟，都是由剑凌负责照顾的。
<br/>　　楚风眠便是从小便是从天绝峰上长大，无数师兄师姐中，除了四师姐之外，对他照顾最多的，便是大师兄剑凌了。
<br/>　　正因如此，时隔万年，楚风眠也是一下子听出来，剑凌的声音。
<br/>　　“大师兄，你没死？”
<br/>　　楚风眠向着雕像大声喊道，只是这雕像，却是没有理会楚风眠，而是继续开口道。
<br/>　　“这极道剑印，本以为没有人会破解，不过你既然可以听到这段话，也就意味着你足以破解了。”
<br/>　　雕像中，剑凌的声音再次响起，不过这声音却是如同机械一般。
<br/>　　楚风眠这才明白过来。
<br/>　　这声音，的确是剑凌的，只是却是剑凌万年前留下的，现在的剑凌，早已经陨落，只是留下了这一道声音。
<br/>　　“我，剑凌，师从剑道之主，却是被七大宗门围杀，师尊陨落在七大宗门的手中，我的无数师弟师妹，全部陨落。”
<br/>　　“我，剑凌，不愿师尊传承就此消亡，特意建立这北邙学院，继续传承，只是在七大宗门面前，只能是隐姓埋名。”
<br/>　　“你能够打开这极道剑印，便是意味着，你跟我们有缘，我便是将传承，留下给你，愿你修成之后，可以杀上七大宗门，为师报仇！”
<br/>　　剑凌的声音，就此消散。
<br/>　　许久，楚风眠的脸色，才缓缓的缓了过来。
<br/>　　再一次听到剑凌的声音，让楚风眠欣喜无比，可是剑凌的话，却是让楚风眠的心头，变得再次沉闷。
<br/>　　他还以为，他的无数师兄师姐，还会有着一些人活着。
<br/>　　可是如今听剑凌话中的意思，他的无数师兄师姐，应该都已经全部都死在了七大宗门的手中了。
<br/>　　唯一剩下的，便是他一个人，建立起了这北邙学院，为了让传承，遗留下去。
<br/>　　就在楚风眠的面前，便是有这样一本古籍，这古籍上，仅仅只有剑道二字。
<br/>　　楚风眠清楚，这边是曾经剑道之主一生剑道的凝聚，七大宗门梦寐以求的宝物，如今便是摆放在了这里。
<br/>　　“万年之后，只怕是大师兄也不会想到，居然是我，活了下来吧。”
<br/>　　楚风眠看到眼前的这古籍，眼神中复杂无比。
<br/>　　就是这古籍，令如今楚风眠家破人亡，不过剑道之主，就算是陨落，都不愿意交出这古籍。
<br/>　　在他的心中，这古籍，比起生命更加重要。
<br/>　　“师尊的仇，我会报的，大师兄留下的基业，我也会让他一直传承下去的。”
<br/>　　楚风眠面对这剑凌雕像，跪下，叩了三个头。
<br/>　　算是对剑凌，为了剑道之主留下传承的感谢，也算是回报曾经的恩情。
<br/>　　这古籍，被楚风眠一把抓住，收下，其中的无数剑意，也是猛然之间，冲入到了楚风眠的脑海之中去。
<br/>　　“这是？”
<br/>　　在楚风眠脑海之中，便是有这无数图像，在其中舞动这，其中的一尊尊人影，都是利用的无数精粹剑术。
<br/>　　这，便是曾经剑道之主的积累，是曾经剑道之主，游历九域，所见识过的无数剑术。
<br/>　　在楚风眠的身上，无数剑意，猛然散发出去，这一股股的剑术，全然都被楚风眠融入到九域剑术之中。
<br/>　　这剑道古籍中的剑术，比起楚风眠曾经所了解过的，还要多上十倍，百倍。
<br/>　　简直是将整个九域，甚至是域外天魔，无数妖兽一族中的剑术，都给全部记载，是天下无数剑术的总纲。
<br/>　　怪不得曾经的七大宗门，不惜一切代价也想要得到，这剑道古籍，对任何一尊剑修来说，都是真正的无价之宝。
<br/>　　楚风眠坐在地上，一点点的参悟这其中内容。
<br/>　　他的剑术，也是越来越强，就在楚风眠的面前，便是化为了九域二字。
<br/>　　在这九域二字的背后，便是剑意凝聚出的海洋，楚风眠的浩瀚剑意，如今已经是如同无边无际的海洋一般。
<br/>　　一日时间，两日时间，三日时间。
<br/>　　就在这诡异的空间之中，楚风眠就坐在原地，一点一点的参悟这剑道古籍中记载的无数剑术。
<br/>　　将这些剑术的所有精华，统统都全部汲取，融入到了他的九域剑术之中。
<br/>　　他的九域剑术，如今已经小成，单单是论起威力来说，已经是足以超越天级武技。
<br/>　　什么杀戮剑术，丰饶剑术，在这九域剑术面前，都算不得什么，因为这两道剑术最为精粹的部分，也全部都被楚风眠融入到了九域剑术之中。
<br/>　　创造出了一门，截然不同极为玄妙的剑术。
<br/>　　这九域剑术中，甚至还能感觉到阿修罗的剑术，龙族的剑术，无数异族的剑术，全部融入其中。
<br/>　　这，还远远没完。
<br/>　　又是三日时间。
<br/>　　楚风眠坐在地上，每时每刻都在参悟这剑道古籍中的剑术，就算是楚风眠参悟每一道剑术，都不需要多久的时间。
<br/>　　但是这剑道古籍中的剑术，也实在是太多了，繁多的剑术，千种，万种，甚至是许多早已经失传的剑术，都在其中见识到了。
<br/>　　楚风眠的九域剑术，也逐渐凝聚大成，一步步开始向着完美的地步蜕变。
<br/>　　“轰隆！”
<br/>　　一共七日之后，在楚风眠的身躯上，一股澎湃剑意，全然爆发了出来。
<br/>　　这澎湃剑意出现的一刻，仿佛便是代表着，楚风眠的剑术，达到了一种新的地步。
<br/>　　他的九域剑术，终于大成。
<br/>　　在楚风眠的身上，他的一举一动之间，仿佛都是蕴含这剑道真理。
<br/>　　这九域剑术，本来就算是有着这众圣堂中的无数剑术，楚风眠也必须要花费一段时间才行的，可现在，却是全然凝练完成了。
