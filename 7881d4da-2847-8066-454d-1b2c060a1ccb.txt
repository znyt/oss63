　　随着所有人全部到齐，新生联谊会也正式拉开了帷幕。
<br/>　　这个活动，是由生物学院学生会举办的，张连鹏身为生物学院学生会主席，自然是以他为首。
<br/>　　“诸位！相信在座的很多人，应该都认识我了吧，我是咱们学院学生会主席张连鹏！”
<br/>　　说到这里，张连鹏故意微微一顿，众人也都很识趣的，纷纷鼓掌。
<br/>　　张连鹏似乎很享受这种被人追捧的感觉，脸色浮现出一抹得意的笑容，等到众人掌声落下，方才又缓缓开口道：
<br/>　　“首先，我代表学院，欢迎诸位学弟学妹，加入我们生物学院，成为学院的一份子！从今以后，我们就是一家人了……”
<br/>　　这张连鹏能够坐到学生会主席的位置，显然还是有两把刷子的，一番煽情的话语之后，将整个宴厅的氛围也调动了起来。
<br/>　　“这样吧，为了让大家更好的了解彼此，我们所有人，先挨个介绍一下子自己，介绍一下自己的家庭，好不好？”
<br/>　　他既然把话说到这个份上了，众人自然不可能说不好，于是纷纷答应。
<br/>　　李虎冷冷一笑，
<br/>　　“这货的意图也太明显了吧？分明就是想调查我们的背景！”
<br/>　　唐威也撇了撇嘴，
<br/>　　“真是无聊！”
<br/>　　另外一边，
<br/>　　在张连鹏的主持下，众人已经纷纷站起来，开始各自做起了自我介绍。
<br/>　　首先是张连鹏，以及坐在主桌上的一些学生会成员。
<br/>　　等到这些人挨个做完自我介绍之后，整个主桌之上，便只剩下陶婉清和杨之凡。
<br/>　　“之凡学弟，来！你先给大家打个样！”
<br/>　　张连鹏笑着开口道。
<br/>　　众人的目光，便齐刷刷的望向杨之凡。
<br/>　　杨之凡倒也丝毫不怯场，微微一笑，站起身来，缓缓开口道：
<br/>　　“大家好，我叫杨之凡，来着东江省临安市，我家是经商的，做的是生物科技这一块，而我本人对生物工程学，尤其是基因学，非常的感兴趣！”
<br/>　　“希望在以后的日子里，可以和大家互相学习，互相进步……”
<br/>　　杨之凡侃侃而谈，吐字清晰流畅，语气也不卑不亢，听的众人纷纷点头。
<br/>　　“不愧是大家风范！”
<br/>　　“没想到我男神，不光人长得帅，口才也这么好！”
<br/>　　“好想当他女朋友啊！”
<br/>　　“别做梦了！人家喜欢的是陶婉清！”
<br/>　　……
<br/>　　很快，
<br/>　　杨之凡介绍结束，张连鹏哈哈笑道：
<br/>　　“之凡学弟太谦虚了！谁不知道你的叔叔，那可是……”
<br/>　　还没等张连鹏说完，杨之凡一口打断道：
<br/>　　“张学长，我叔叔不希望家里人，四处打着他的旗号，影响不好！”
<br/>　　“是是是！”
<br/>　　张连鹏连忙住嘴，脸上露出一抹尴尬之色，不过旋即脸上，讨好的笑容更加浓郁，
<br/>　　“不过，令尊的大名，在整个东江省乃至全华夏，也是赫赫有名的啊！”
<br/>　　“我听说，令尊旗下的之凡生物科技公司，便是以你的名字命名，而且听说这个公司，已经取得紫金仙泉的代理权，不知道此事是真是假？”
<br/>　　众人听到这话，顿时一片哗然，
<br/>　　“紫金仙泉？”
<br/>　　“莫非就是那个，号称是生命之水的紫金仙泉吗？”
<br/>　　“听说这仙泉，出自那位华夏仙人之手，不但可以治愈各种疑难杂症，还能益寿年年，价格高的离谱不说，一般人就算有钱也买不到！”
<br/>　　……
<br/>　　云州紫金山泉，如今早已经名满华夏，这些人学的又是相关专业，自然都十分清楚。
<br/>　　听到众人的议论声，叶尘顿时也微微有些愕然。
<br/>　　这紫金仙泉，正是他那紫金山小区别墅中，所产出的神水。
<br/>　　他倒是没有想到，这杨之凡家的公司，居然还是苏叶集团的经销商！
<br/>　　原来，
<br/>　　最近这几个月，叶尘让楚飞烟，把主要重心转移到建设黑森林上面。
<br/>　　楚飞烟分心乏术，便挑选了一些有实力的生物制药公司，将神水经销权出让给了他们。
<br/>　　而这之凡生物科技公司，在东江省是实力最强的生物制药公司，所以自然也就被楚飞烟选中了。
<br/>　　杨之凡淡淡一笑，
<br/>　　“不错！我家的确已经取得紫金仙泉的经销权，而且我最近正打算，和我爸爸商量一下，捐出一瓶神水，供我们学院进行研究！”
<br/>　　轰！
<br/>　　杨之凡这话一出口，众人顿时再次哗然，一些人眼中流露出浓浓的崇拜之色。
<br/>　　要知道，这紫金仙泉的价格，如今已经炒到了十多万一克，而且还是有价无市！
<br/>　　“我滴乖乖！没想到这家伙，竟然还有这层关系，我倒是小瞧他了！”
<br/>　　李虎一阵咋舌道。
<br/>　　一旁的唐威满不在乎道：
<br/>　　“不就是紫金仙泉的经销权吗？再厉害也只是帮人跑腿的！”
<br/>　　李虎深吸了一口气道：
<br/>　　“老三，这你就不懂了吧！就算是跑腿的，你要看他是给谁跑腿！”
<br/>　　“紫金仙泉的幕后主人是谁，是那位华夏仙人，云州叶先生！”
<br/>　　“那可是连岛国和米国，这样的当世强国，都被他踩在脚下的存在！”
<br/>　　“杨家之前在东江省，只能勉强跻身一流家族而已，如今攀上了云州叶先生，等于一步登天！”
<br/>　　“恐怕要不了多久，整个东江省，都将会是杨家的天下了！”
<br/>　　听完李虎的分析，不光唐威和陈东一阵默然，就连叶尘也不由得对李虎刮目相看。
<br/>　　自己的这个室友，看问题倒是透彻！
<br/>　　只不过，他怎么也想到，他口中的那位华夏仙人，此刻正坐在他的旁边呢！
<br/>　　杨之凡做完自我介绍之后，便轮到了陶婉清，
<br/>　　“大家好，我叫陶婉清，来自京都，父母是公务员……”
<br/>　　随着陶婉清介绍完自己，再次引起了不少躁动。
<br/>　　虽然陶婉清的家世，不如杨之凡显赫，但也算是很不错的了，和杨之凡倒也算是门当户对。
<br/>　　接下来，
<br/>　　其他人的背景，明显就要比两人差多了，大多数人都只是普通家庭，偶尔有一两个，家中经商或从政的，都远远无法和杨之凡相提并论。
<br/>　　很快，轮到了叶尘这一桌，
<br/>　　陈东、唐威和李虎，也都相继做了自我介绍。
<br/>　　三人中，以李虎的家境最好，也算是富豪之家，可惜有杨之凡这座大山在前，所以并没有引起太多的关注。
<br/>　　最后，
<br/>　　终于轮到叶尘做自我介绍，所有人顿时纷纷望了过来……
<br/>(本章完)
