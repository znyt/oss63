　　第360章
<br/>　　以琴川关目前的规模，已到一个山寨可以到达的极限。再往下发展，便该建城。事实上，现在的琴川关已有一座小城的雏形。
<br/>　　该跨进下一个境界了。
<br/>　　可惜的是，程大雷没有收到晋级任务，系统提示是触发条件不足。
<br/>　　无论如何，只要开出火药，程大雷这次的目的已经达成。虽然没有触发晋级任务，程大雷却也并不着急。目前以休养生息为主，在财力不够雄厚时，一味大兴土木，却也未必是好事。
<br/>　　程大雷已经准备离开，可惜这次没有搞掂樊梨花，令程大雷有些遗憾。
<br/>　　他骑在黑牛上，肩系血披风，银眸骑马立在程大雷身后，小小的身材背着一柄硕大的斧头。
<br/>　　知道程大雷要走，落玉寨大部分人都送到门口，樊梨花却也在其中，冷着一张脸。相比之下，高力士却是分外舍不得程大雷离开。
<br/>　　“大当家，不多留几日么？我还想在大当家面前，多听几句教诲。”
<br/>　　“回去吧，回去吧，大家不要送了。”程大雷看了樊梨花一眼，发现樊梨花没有搭理他。
<br/>　　程大雷和银眸正准备离开，忽然见远处一人快马奔来。马蹄踏过之处，荡起一地烟尘。在人近了，程大雷才看清楚马上的人是赵子龙。
<br/>　　赵子龙勒马在程大雷面前停住，脸有焦急之色。
<br/>　　“琴川关出事了，请大当家速速回去处理。”
<br/>　　以赵子龙现在的身份，当是不用做传信这种小事。可既然是他亲自过来，便也能想到事情的紧急程度。
<br/>　　“出什么事了？”
<br/>　　赵子龙叹口气，道：“琴川关外三户人家，十三人被一夜之间杀尽，就是在昨夜发生的。”
<br/>　　程大雷沉默片刻，道：“走，回去看看。”
<br/>　　一行人踏上回程的路，但不知怎么会事，樊梨花也跟了上来。程大雷、赵子龙、银眸三人组成的队伍变成四人，但她跟上来也不说话，谁知她心中怎么想的。
<br/>　　不过，也不怪徐灵儿说樊梨花没脑子，高力士刚加入落玉寨，她就放手不管了，如今的落玉寨只有徐灵儿和高力士管理。
<br/>　　还好高力士是系统召唤出来的，忠诚方面不成问题，不然山寨很有可能就被人篡位了。
<br/>　　当然，樊梨花本身好像也不关心这些事。
<br/>　　四人回到琴川关。
<br/>　　程大雷刚换过衣服，便和樊梨花一起来到停尸房，十三具尸体便被摆在这里。如果不是等程大雷回来看一眼，怕尸体已经被埋了。
<br/>　　刘悲已等在门口，看到程大雷和樊梨花进来，略微顿了顿，道：“樊寨主还是不要进去了。”
<br/>　　“怎么了？”
<br/>　　“他们的死相有些惨，抬尸的人如今还在吐，樊寨主最好还是不要看了，怕以后会做噩梦。”
<br/>　　“呵，不就是死人么，还以为我真没见过。”
<br/>　　樊梨花冷哼一声，推门而入。
<br/>　　程大雷和刘悲对视一眼，无奈的耸耸肩，也踏入了房间。
<br/>　　房间摆着一具具尸体，身上都盖着白布，在程大雷面前摆成一排。
<br/>　　大概是环境使然，就连程大雷都觉得空气中有种阴森的味道。他用手遮住口鼻，以眼神示意刘悲。
<br/>　　“打开看看吧。”
<br/>　　“大当家仔细了。”
<br/>　　唰！
<br/>　　白布抖开，程大雷视线落在木板上，只是一瞬间，樊梨花一张脸便变得苍白，而程大雷脸上的肌肉也剧烈抖动了两下。
<br/>　　“其他的也都打开吧。”程大雷。
<br/>　　“只看一具就够了，其他情形也差不多。”刘悲。
<br/>　　“全部打开。”
<br/>　　一张张白布抖开，十三具尸体暴露在程大雷眼前，空气中弥漫着一股尸体特有的酸臭味，令人作呕。
<br/>　　程大雷也不是没有见过尸体，经他手杀死的人不少，可眼前这一幕还是给程大雷带来巨大的冲击力。
<br/>　　只见一具具尸体受伤的部位各有不同，有人是头部，有人是胸膛，有人是小腹，虽然已经被清理过，但惨状依旧令人不忍直视。
<br/>　　在回来的路上，赵子龙已经对程大雷简单说过情况。
<br/>　　事情是昨晚发生的，死的人住在琴川关外围。
<br/>　　琴川八百里，都是不见人烟的荒原，流民不断迁入，自然也不可能住满八百里的每个地方。
<br/>　　大多数是在琴川关附近安家，这样可以受到琴川关的保护，可同时也要缴纳一定的税费。所以便有些人住在远一些的地方，逃避缴税。
<br/>　　对于这些人，程大雷没有管，却是也管不过来，随他们去吧。
<br/>　　这次死的三户人家，一十三口就是这部分人。这些人里有老人一位，青壮年男人五位，女人四个，还有三个小孩。
<br/>　　死相都很惨，有人肚子里被填了石头，女人都受过糟蹋，下体插着木棍。
<br/>　　怪不得抬尸的会呕吐不止，就连程大雷今晚能不能睡着，自己都不清楚。
<br/>　　“他们用的是什么兵器？”程大雷必须镇定下来，身为琴川之主，他必须扛下去，发生事情解决便是。
<br/>　　刘悲目光凝重起来，道：“让山寨的大夫看过，他们用的是……”
<br/>　　刘悲用手敲了敲左手手背，程大雷一怔，眼中眯起一道线，杀机外露。
<br/>　　对方没有用兵器，用的是……双手。
<br/>　　意识到这点后，程大雷觉得不寒而栗。眼前浮现一幕幕情景：
<br/>　　一伙恶人潜入屋子，用双手撕开胸膛，掏出热气腾腾的心脏，扯开人的小腹，在里面填入石头……
<br/>　　程大雷还看到，一人是头骨碎裂而死，程大雷本以为是用石头之类的钝物砸的，此刻才知道，是被人生生用手力捏碎。
<br/>　　头骨算是人体最坚硬的骨头，需要怎样的力气才可以活活捏碎他，程大雷自问自己都做不到这一点。当然，他没有试验过，也不愿去做这方面的试验。
<br/>　　“看来琴川来了一伙狠人呐。”程大雷嘴角突然浮现一抹诡异的笑容。
<br/>　　“大当家，现在人心惶惶，许多人说是我们干的，是对他们逃税的惩罚。”刘悲道：“要不要封锁消息，别让他们议论。”
<br/>　　“封锁消息做什么？”程大雷冷冷道：“找到他们，杀死他们。”
<br/>
