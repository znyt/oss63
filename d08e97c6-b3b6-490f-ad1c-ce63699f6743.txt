<br/>　　苏洪立刻身形一震，不过他并没有任何异动，他完全相信自己的儿子。

<br/>　　不过，这战魂相当不弱，是他灭杀一个巫族小镇之时，吞噬得来。
<br/>　　而后，苏莫开始将父亲的神魂，和这虎头莽身的战魂融合，
<br/>　　这比苏莫自己凝练分身之时，融合战魂还要困难的多，因为不是自己的神魂，无法随心所欲的控制。
<br/>　　好在有吞噬战魂作为媒介，而虎头莽身的战魂又被苏莫完全炼化，不仅没有任何的意识，还收他的控制。
<br/>　　苏莫控制着战魂之力，小心翼翼的将父亲的神魂与虎头莽身的战魂融合，他丝毫不敢大意，因为苏洪的神魂实在太弱了。
<br/>　　一个不慎，很可能导致父亲的神魂崩溃。

<br/>　　进展的无比缓慢。
<br/>　　按照这个速度进展，估计想要融合成功，起码需要半个月的时间。
<br/>　　“父亲，努力的和这只兽类战魂沟通！”苏莫传出了一道意念，传进了苏洪的神魂之中。
<br/>　　果然，苏莫说出此言之后，融合的进度立刻加快数十倍。
<br/>　　因为苏洪的神魂，主动的和虎头莽身的战魂沟通、融合。
<br/>　　时间缓缓流逝，又过去了十几个时辰，苏洪的神魂和虎头莽身的战魂，逐渐的融合在了一起。
<br/>　　直到某一刻，两者彻底融合成功！
<br/>　　苏莫顿时大喜，果然，他的猜测没有错，果然可以融合，这简直有如神助啊！
<br/>　　虽然耗时比较长，但融合的无比完美，没出任何岔子。

<br/>　　闭目而坐的苏洪，闻言缓缓的睁开了双眼，眸中精光爆射。
<br/>　　“莫儿，这是真的吗？”苏洪激动的满脸潮红，因为他清楚的感觉到，自己体内拥有了强大的战魂。

<br/>　　苏洪深深的吸了口气，而后意念一动，立刻释放出了自己的战魂。
<br/>　　轰！
<br/>　　刹那之间，一只虎头蟒身的兽类战魂从苏洪背后升腾而起，硕大无比，狰狞恐怖，凶威盖世。
<br/>　　战魂乍一出现，便将这间房间直接撑爆了，木屑飞舞，瓦片激.射，化为了一片废墟

<br/>　　呼！

<br/>　　苏洪才真罡境修为，怎么可能有这么强大的战魂？
