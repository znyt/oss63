　　来人身材高大威猛，穿着一身皮质短衫，面目略显粗犷。
<br/>　　此人龙行虎步，一步千丈，转瞬之前，便来到了近前。
<br/>　　“拓跋流明？”苏莫眼眸微眯，目光紧紧的盯着此人，此人和拓跋青云有关系吗？
<br/>　　不过，此人的实力极度强大，修为应该是四品，但具体是低级还是中级，亦或者是高级，苏莫看不太清楚。
<br/>　　毕竟，修为差距太大了。
<br/>　　拓跋流明身形停留在了万道剑冢的上空，目光如电，俯瞰下方众人。
<br/>　　“哈哈，天才不少，不过，一个能打的都没有！”拓跋流明嗤笑一声，朗声说道。
<br/>　　“拓跋流明，你在轮回神朝猖狂也就罢了，天池没有你撒野的地方！”千幻邪君目光冷然，冷笑着说道。
<br/>　　“怎么？林千幻，你难道还想要和我切磋两招不成？”拓跋流明的目光，瞬间落在了千幻邪君的身上，眸中流露出强大的战意。
<br/>　　“难道还怕你不成！”千幻邪君冷喝一声，对方虽然修为很高，高达四品中级，但他也毅然不惧。
<br/>　　“这么自信？”
<br/>　　拓跋流明嘴角勾起一丝冷笑，眸中战意弥漫而出。
<br/>　　“够了！拓跋流明，你来此难道是挑事的吗？”
<br/>　　就在此刻，一声娇喝响起，只见云沐冷冷的望着拓跋流明，不待对方回答，继续道：“万道剑冢正在开启，你们若是想战，不要打扰我们！”
<br/>　　拓跋流明闻言，轻笑一声，揶揄道：“原来是自诩代表天道的云沐大小姐，久仰久仰！”
<br/>　　“你什么意思？”云沐见揶揄的神态，眉头忍不住一皱。
<br/>　　“在下久闻云沐大小姐的威名，心生仰慕而已！”拓跋流明笑着说道。
<br/>　　但是，他心中却是非常不屑，这个云沐天赋的确是高，人也长得美若天仙，但是，太过自傲了。
<br/>　　不过是拥有天罚神瞳元神，能施展天罚之力而已，却是一直以代表天道自诩，未免太过自大了。
<br/>　　“本小姐不稀罕！”云沐淡淡的说道，仰慕她的人千千万万，她根本不在乎，即便拓跋流明是万古天骄。
<br/>　　“呵呵！”拓跋流明闻言，顿时冷笑一声，不过，他也不再多言，更不再和千幻邪君争锋相对。
<br/>　　到了万道剑冢中之后，这些人都将被他踩在脚下，无人能与他争锋。
<br/>　　苏莫注视了拓跋流明片刻，便收回了目光，此人既然是来自轮回神朝，那必然是地王宫之人。
<br/>　　只是，他不清楚此人，和拓跋青云是什么关系。
<br/>　　但是，他能肯定，必然有关系。
<br/>　　那么此次，他更有小心行事了，牵扯到地王宫，不得不谨慎。
<br/>　　嗡嗡~~~
<br/>　　此时此刻，深坑之中，浓密的雾气翻滚的越来越剧烈。
<br/>　　片刻之后，浓密的雾气，汇聚在了一起，形成了一个巨大的雾球，飞速的沉入了深坑的底部，消失不见。
<br/>　　“雾气消散了！”
<br/>　　“可以进去了！”
<br/>　　霎时之间，朗喝声四起，深坑边聚集的数千人，纷纷行动了起来。
<br/>　　众人个个飞身而起，浩浩荡荡的向深坑之中冲了下去。
<br/>　　苏莫同样没有迟疑，和众人一起，飞进了深坑之中，不断的向下潜去。
<br/>　　下方一片漆黑，以他的修为，目力都不到百丈。
<br/>　　下潜了约莫十几个呼吸的时间，深坑终于到达了尽头，众人的脚步落在了实地上。
<br/>　　苏莫混沌之力涌入双眼，顿时能勉强看清周围的情况。
<br/>　　下方的空间，并没有上方那么大，约莫方圆两三里。
<br/>　　四周都是岩壁，光秃秃一片，枯寂无比。
<br/>　　在其中一个方向的岩壁之上，有着一个巨大的石门，高约三百丈，宽约两百丈。
<br/>　　这石门的门框，高高的突出，如同怒龙盘扎，显得苍劲有力。
<br/>　　而石门之上，雕刻着大量的宝剑，数量极多。
<br/>　　这些宝剑，形态各不相同，有巨大无锋的重剑，有、二尺寒锋的短剑，还有其它各类宝剑。
<br/>　　望着巨大石门上的宝剑，苏莫感觉斗转星移，天地变换。
<br/>　　周围的其它人都消失了，他的眼前，只剩下了巨大的石门。
<br/>　　那些石门上的宝剑，仿佛都活了过来，纷纷向他袭杀而来。
<br/>　　大量的宝剑，如同一阵剑雨，裹挟无匹的威势，势不可挡。
<br/>　　“这是……？”
<br/>　　苏莫心中一惊，面对这无法躲避的劈天盖地的剑雨，他毫不犹豫的出手抵挡。
<br/>　　长剑在手，剑光闪耀，剑气如虹。
<br/>　　苏莫连斩数十剑，强大的剑气，与铺天盖地而来的剑雨相击，将剑雨直接劈开了，无数宝剑迸射八方，随即纷纷返回到了石门之上。
<br/>　　下一刻，一切都恢复了平静，苏莫发现，他依旧站在深坑的底部，前方的石门亘古未变。

<br/>　　就在此刻，周围响起一声声惨叫，有的人吐血飞退，有的人甚至直接四分五裂。

<br/>　　这些人，基本上都是二品修为之人，也有寥寥几名三品低级修为之人。

<br/>　　而其他没有受伤之人，显然也是抵挡住了幻境中的攻击。

<br/>　　拥有千幻元神，幻境、幻术，对于他来说，都是雕虫小技。

<br/>　　因为，这山洞，从来没有人能走到尽头。
