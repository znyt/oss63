　　漫天火海，顷刻之间成为了天空之上唯一的主宰。
<br/>　　这一片火海，仿佛是在这一片魔域之中，硬生生的撕裂出了一道裂痕。
<br/>　　炙热的火浪，让那些五灵宗圣子，都只能退避。
<br/>　　“这，这是什么剑术？”
<br/>　　炎撼世见到这漫天火海，眼睛都要掉了下来，他本来就是控火之道的天才，所以才更可以看得出楚风眠这一剑的玄妙。
<br/>　　这种力量，简直不是人力可以做到的，超出了武者的极限，在这火焰之下，别说是三名魔族魔皇，就算是三十名，都未必可以抵挡得住。
<br/>　　在这火焰一出的一刻，仿佛就代表着一切的毁灭。
<br/>　　“这一剑是楚风眠打出来的?”
<br/>　　木麟还有一众五灵宗圣子如今都是长大这嘴巴看着这一幕，许久说不出去话来。
<br/>　　他们可是见到过曾经五灵宗之中的九劫古帝出手过的。
<br/>　　但是纵然是哪九劫古帝出手，都没有如今楚风眠的这般恐怖。
<br/>　　在实力上也许楚风眠还不足以跟九劫古帝抗衡，但是这施展出的一剑，已经是丝毫不逊色九劫古帝的全力一击了。
<br/>　　赤红大陆之上，无数人都是抬起了头，看着这漫天火海。
<br/>　　这火焰，要吞噬天地一切。
<br/>　　看到火海的逼近，三位魔皇如今的心中都是泛起了几分恐惧。
<br/>　　这一剑的力量，超出了他们的想象。
<br/>　　“魔意遮天！”
<br/>　　鬼魅皇咆哮一声，周围的无数魔族，顿时将一身力量凝聚到了极致，纷纷打了出来。
<br/>　　一时间魔意凝聚，群魔都是化身在魔意之中，除非是一招之下，毁灭所有魔意，不然的话，这些魔族便是不死不灭的存在。
<br/>　　“无谓的挣扎!”
<br/>　　楚风眠嘴角露出冰冷，剑锋一指，这漫天火海便是弥漫而去。
<br/>　　这火焰，可不是寻常之火，而是九重天火，纯阳天火。
<br/>　　当着火海沾染到那魔意的一刻，当即开始熊熊燃烧起来。
<br/>　　“啊!”
<br/>　　瞬息之间，在这魔意之中响彻而起无数惨叫声。
<br/>　　一尊尊魔族，落入在这火海之中，被全部焚毁，在这火海面前，没有任何挣扎出去的可能。
<br/>　　几乎是每时每刻，落入到了这火海之中的魔族，都在痛哭的咆哮，不断的陨落在火海之内。
<br/>　　而那些魔族的精血，全部都被吞天秘术所吞噬，化为了神树的给养。
<br/>　　大量精血的吞噬，让神树都是开始兴奋起来。
<br/>　　“这些魔族的精血，果然对神树来说，都是大补之物！”
<br/>　　楚风眠看着神树在吞噬了大量的魔族精血后，气息节节攀升，他的心中也兴奋无比。
<br/>　　这神树扎根于楚风眠的身躯之中，可以说现在这神树就是楚风眠力量的一部分。
<br/>　　神树的力量越强，代表的也就是楚风眠的力量越强。
<br/>　　楚风眠也发现了，对神树来说，吞噬魔族精血，比起吞噬人类武者，妖神的精血，得到的好处要更加巨大。
<br/>　　吞噬一名七劫古帝级别魔族的精血，几乎相当于同时十名同境界武者的精血。
<br/>　　看来要去多多猎杀魔族，吞噬了他们的精血才行！
<br/>　　吞噬！
<br/>　　楚风眠手心猛然一捏，突然对着地下的夜叉皇抓了过去。
<br/>　　“不！”
<br/>　　夜叉皇疯狂的咆哮道，他还打算是让鬼魅皇等人去救他。
<br/>　　可现在另外三名魔皇，已经是自身难保，谁能救他？
<br/>　　夜叉皇被楚风眠捏在手心之中，当即催动起吞天秘术来，将他的身躯拉人到漩涡之中。
<br/>　　漩涡转动，庞大的力量将夜叉皇的身躯当即撕裂开来，一身精血也喷洒而出，经过吞天秘术的萃取，最为精粹的精血，都被神树所吞噬。
<br/>　　“这精血之中蕴含的力量，真是恐怖！”
<br/>　　虽然这精血的力量被神树所吞噬，但是楚风眠也足以清晰的感觉得出，这精血的力量，到底是多么的恐怖。
<br/>　　魔族跟武者不同，人类武者的力量，跟武技，功法，灵器等，息息相关，如果抛去这一切，同境界之中武者几乎是最弱者。
<br/>　　可魔族的强大，凭借的就是自身的血脉之力，没有灵器，没有武技，他们的一切力量都是来源于血脉。
<br/>　　单单是论起血脉的力量，魔族要比起同境界的武者强横数十倍。
<br/>　　所以他们蕴含的血脉之力，才无比惊人，而这些血脉之力，都将会成为神树的力量。
<br/>　　在杀几名八劫古帝级别的魔皇，神树的力量将会再次蜕变！
<br/>　　楚风眠感觉得出，在吞噬了大量的精血后，现在神树的力量已经是在蜕变的边缘了。
<br/>　　神树的每一次蜕变，都足以给楚风眠带来无比巨大的好处。
<br/>　　楚风眠的目光看向那火海之中，如今七劫古帝级别的魔族，已经是没有一人活下，全部都是死在了火海之中了。
<br/>　　只剩下鬼魅皇，阎魔皇，独魔皇这三名八劫古帝的魔皇，还在苦苦支撑。
<br/>　　突然之间，在这火海之中，三道遁光携带这火焰，猛然间飞了出去。
<br/>　　这三名魔皇如今也明白，在火焰之中任何的抵抗都没有意义，终究会被这火焰弄到油灯枯尽的一刻。
<br/>　　不如趁着如今还有着几分力量的时候，拼死一搏。
<br/>　　“想逃？”
<br/>　　楚风眠看向这三道遁光，嘴角露出了几分不屑笑容。
<br/>　　这三名魔皇，想要从拥有始祖天龙血脉的楚风眠手中逃走？简直是在做梦。
<br/>　　“封锁！”
<br/>　　楚风眠运转起始祖天龙的力量，凝聚在指尖之上，手指点出的一刻。
<br/>　　周围的空间，被团团凝固住了，那三道遁光，也在空中，停滞了下来。
<br/>　　“一剑破苍穹！”
<br/>　　随即而来的，则是一道剑光，凌空斩落而下，斩杀在了那三名魔皇的身躯之上。
<br/>　　一剑之威，三名魔皇都没有任何反抗的能力，在这剑锋之下，被当即斩杀，爆炸开来，全部陨落。
<br/>　　一团团的精血，被楚风眠全部捏在手中，随即被神树所吞噬。
<br/>　　“这三名魔皇，就这样死了？”
<br/>　　“我们得救了？”
<br/>　　看着那三名魔皇的死，许多五灵宗的圣子都还是如梦初醒一般。
