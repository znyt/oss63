　　什么？黑火药改民用？这不可能吧？马周几乎不敢相信自己的耳朵。
<br/>　　那特么可是能炸死人的东西，改为民用……。
<br/>　　“殿下，您，您不再考虑一下么？这黑火药……”
<br/>　　“简化一下，先把配比改了，其次就是生产工艺的简化，不要那么细的颗粒了，大一点，再加上严格把控生产厂家的进出货清单，问题应该不大，就算是被一些异族弄去，估计没有百把十年他们也搞不清楚什么样的黑火药威力最大。”
<br/>　　马周已经无法用言语来形容李承乾的疯狂，因为他发现这位太子殿下似乎打算来真的。
<br/>　　可是黑火药那东西真的可以炸死人啊，万一落到一些有心人的手里，在路边埋上两个大号的，后果简直不堪想象。
<br/>　　而就在马周纠结是应该如何劝说固执的李承乾时，一行人已经离开了刚刚的位置，来到了一个奇怪的房间。
<br/>　　这里并没有多少人，一眼看去应该不到二十人，这些人分成两人组做着同样的事情，每组的第一个人都在摆弄着一只不大的天平，不同的是一人在称黑火药，一人在称所谓的无烟火药。
<br/>　　而那些被称好的火药则被他们身边的另外一人拿走，装进精致的铜制小瓶子里。
<br/>　　当然，这是因为马周并不知道如何称呼子弹这种东西，所以他坚定的认为那就是一个铜制的小瓶子。
<br/>　　接下来便是不断有人在那个小瓶子上摆弄着，有的人会给小瓶子安装上一个尖尖的盖子，有的人则给小瓶子安装一个底座，等到了最后一人手中的时候，则会在这些小瓶子上面作上标记，然后装箱。
<br/>　　“殿下，这是什么？”马周从头看到尾，直到离开那个房间之后，才问出了心中的疑惑。
<br/>　　“他们在制作子弹，就是左轮手枪发射出去的那个东西，只不过他们制作的这一种威力更大，可以在两百步之外命中敌人，击中要害可以一击致命。”李承乾解释道。
<br/>　　“两百步……”
<br/>　　今天这一天带给马周的惊讶简直太多了，从黑火药面临淘汰，到可以打到两百步之外的子弹，这些概念都在不断的刷新着马周的世界观。
<br/>　　他知道左轮手枪，可他不是军方之人，所以并没有装备那个东西，但多少还是见过几次，知道那东西如果准头儿好一些的，可以保证六十步之内一击必杀。
<br/>　　可是现在李承乾竟然说，这东西已经能在两百步之内一击必杀，这特么不是在玩笑玩么？
<br/>　　如果真有这东西，如果有人暗杀的话，岂不是隔着两百步就可以完成？那么一些重要人物的安全应该如何保证？
<br/>　　短短的一瞬间，马周想到了很多东西，他觉得自己很有必要与李承乾好好谈一谈了，否则再让他这样继续折腾下去，估计迟早要闹出大事儿来。
<br/>　　心动立即行动，想到此处，马周立刻追上李承乾的脚步：“殿下，殿下可否留步，臣有要事相商！”
<br/>　　“哦？什么事？”李承乾不解的问道。
<br/>　　“殿下，此物太过危险了，若是真的如您所说，两百步之外便可取人性命，这……”
<br/>　　“老马，我知道你在担心什么。”看到马周急切间有些词不太意的表现，李承乾脸上露出一丝笑意：“你是在担心一些不必要的暗杀是吧？”
<br/>　　“臣正是此意，殿下既然已经想到了，为何还要一意孤行呢？”
<br/>　　“那是因为我知道还有比这个更加厉害的东西存在。”李承乾抬头看了看头顶，可惜因为天花板的存在，无法看到天空。
<br/>　　“更厉害的东西？”
<br/>　　“父皇说过水能载舟，亦能覆舟，这句话的意思你应该明白，对吧？”
<br/>　　马周点点头。
<br/>　　“所以说民心很重要，不管是管理地方，还是管理国家，都要为百姓考虑，让面姓有生活下去的条件与机会，而只要你一心为百姓着想，又怎么可能会有人来刺杀？换句话说，如果当权者混到了需要时刻担心自己被刺杀的时候，那么他还有什么脸面活在这个世界上。“
<br/>　　歪理啊这是，马周明明知道李承乾说的这些就是歪理，可却找不出反驳的理由。
<br/>　　因为他知道仅仅从理论上理解，李承乾说的是对的，如果一个当权者混到了连百姓都会来刺杀他的时候，那就足以说明其失败的一生，也从另一个方面说明百姓其实并不拥护他，那么他自然就没有继续在那个位置上待下去的理由。
<br/>　　可是，这个世界上的事情真的那么简单么？如果真是那样的话，还要法律有什么用？而且谁又能保证刺杀一定是百姓发起的？政敌发起的刺杀不行么？
<br/>　　所以李承乾说的那些只不过是理想化的东西，和现实还是有区别的。
<br/>　　但马周并不知道，李承乾其实也是两世为人，在后世的时候比子弹威力大的东西不知道有多少，也没有看到谁拿着枪就把领导给崩了。
<br/>　　而且重要的一点是，这些装备如果出现了第一批准备的一定会是皇宫中的禁卫，等到普及全军的时候，大内里面已经不知道更新过多少代的武器了，防御力量比之以前只强不弱，又怎么可能会出意外。
<br/>　　当然了，脑血栓，心梗之类的意外还是有的，不过那属于天灾，在大唐非人力所能控制所以不在考虑范围之内。
<br/>　　……
<br/>　　靶场，类似于98K的新式步枪被李承乾端在手中，每次射击之后，巨大的撞击都让他的身体剧烈的抖动一下，而后便用有一股呛人的烟雾升起来，呛的人不住咳嗽。
<br/>　　马周手中同样有一只新式步枪，虽然他极力反对这东西的现出，但不可否认这东西还真是一个顶好的玩具，随着一声声枪响，让人乐不思蜀，尽管被呛的涕泪横流，却依旧不想把枪放下。
<br/>　　而就在马周玩的嗨皮时，边上的李承乾递过来另外一支枪：“再试试这个。”
<br/>　　“这个？这不是一样么？”看着一模一样的两只枝枪，马周疑惑的眨了眨眼睛。
<br/>　　不过随着他端起枪，射出第一颗子弹之后，便发现，这枝枪真的和刚刚那一枝不同，因为这一枝竟然没有那呛人的硝烟！
