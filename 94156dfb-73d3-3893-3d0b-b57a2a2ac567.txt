　　哗啦啦！
<br/>　　众人听到杨之凡的这话，看到眼前的这架势，顿时吓得纷纷朝教室外逃窜。
<br/>　　“啪！”
<br/>　　坐在叶尘旁边的唐威，一拍桌子，霍然站起身来，
<br/>　　“杨之凡！你还敢来找事！”
<br/>　　说着，唐威两腿一瞪，直接从座位后面窜到了走廊之上，就要上前和杨之凡理论。
<br/>　　唰！唰！唰！
<br/>　　保护在杨之凡周围的那些黑衣保镖，当即纷纷扑了上来，想要挡住唐威的去路。
<br/>　　可惜，以唐威如今的实力，对付这些寻常保镖，自然是绰绰有余。
<br/>　　砰砰砰！
<br/>　　唐威随手几下，顿时便将阻拦他的那些黑衣保镖，打的落花流水。
<br/>　　“好！打的好！”
<br/>　　李虎和陈东在后面，则纷纷叫好起来。
<br/>　　“哼！就凭你这群废物，也想找尘哥的麻烦，真是不知死活！”
<br/>　　唐威冷笑一声，脚下猛然一步迈出，直奔坐在轮椅上的杨之凡抓了过去。
<br/>　　杨之凡见状，顿时才有些慌了，连忙高呼了一声，
<br/>　　“华老！”
<br/>　　嗖！
<br/>　　声音落下的同时，一道人影在空中划过，瞬间便挡在了杨之凡的跟前。
<br/>　　赫然正是华云生！
<br/>　　“哈哈哈！不错不错！小小年纪就有如此实力，你也算是个人才了！”
<br/>　　华云生哈哈一笑，直接随手一抓，
<br/>　　刺啦！
<br/>　　空中陡然出现了一个灰色虚影巨掌，一把便擒住了唐威的身体，令他动弹不得。
<br/>　　“唐威！”
<br/>　　后方的李虎和陈东见状，顿时纷纷脸色大变，不由得一声惊呼。
<br/>　　“嗯？”
<br/>　　叶尘也眉头微微一皱，喃喃自语了一句，
<br/>　　“居然来请来神境高手，看来我倒是小瞧这杨家了……”
<br/>　　华夏国经过之前那次浩劫之后，几乎所有武林高手，都被神子卡斯特所杀，如今的华夏国，神境级别的高手，也不过寥寥数人而已。
<br/>　　叶尘原本以为，以杨家的能量，最多也就能请动圣境武者，没想到竟然来了一位神境！
<br/>　　当然了，若是在一年之前，对方或许还能和叶尘有一战之力，对于如今的他来说，就有些微不足道了。
<br/>　　“华老！杀了他！快给我杀了他！出了什么事，都由我杨家来担着！”
<br/>　　杨之凡看到华云生，轻而易举的将唐威擒住，顿时兴奋的大喊大叫起来。
<br/>　　他最恨的人虽然是叶尘，可是恨屋及乌，自然连唐威三人，也一并都恨上了。
<br/>　　不料，
<br/>　　华云生却仿佛没有听到他的话一般，反而好整以暇的望着唐威，笑呵呵道：
<br/>　　“小子，老夫念你也算是一个武道天才，不如你投你在老夫的门下，老夫便饶你一命，如何？”
<br/>　　杨之凡听到这话，顿时急了，
<br/>　　“华老！您这话是什么意思？我是请您来帮我报仇的，你收了我杨家的钱，怎么能……”
<br/>　　“闭嘴！”
<br/>　　还没等杨之凡说完，此时已经踏入教室的杨国强，当即一声暴喝，将他的话生生打断，
<br/>　　“孽障！华老是何等身份？岂容你如此无礼顶撞？”
<br/>　　杨之凡经自家老爸这么一提醒，这才恢复了几分理智，顿时吓出了一身的冷汗！
<br/>　　他刚刚一时情急之下，那番话便脱口而出了，竟然都忘了，站在自己面前的，那可是堂堂神境强者！
<br/>　　达到这种级别的强者，哪个不是心比天高，又岂会为金钱所驱使？
<br/>　　“哼！”
<br/>　　华云生回过头来，瞥了一眼轮椅上的杨之凡，冷冷道：
<br/>　　“小子，若非看在你父亲的面子上，就凭你刚刚那句话，老夫便非宰了你不可！”
<br/>　　杨之凡顿时吓得浑身一颤，连忙点头如捣蒜，
<br/>　　“是是是！是晚辈失言了，还请华老海涵！海涵！”
<br/>　　眼看对方自己起了内讧，而且华云生也没有对唐威痛下杀手，叶尘也没有着急出手搭救。
<br/>　　华云生缓缓转过身来，望着悬在半空中的唐威，继续又笑呵呵道：
<br/>　　“小子，你可考虑清楚了？愿不愿意做我的弟子？”
<br/>　　听到华云生再次开口询问，杨之凡顿时脸色变得无比难看。
<br/>　　倘若唐威真的拜了华云生为师，以唐威和叶尘的关系，他必然会为叶尘求情，到那个时候，华云生是会偏向自己？还是会偏向唐威？
<br/>　　答案不言而喻了。
<br/>　　当然了，看在杨家的面子上，华云生或许不会为难于他，可是他的大仇，恐怕也就报不了了！
<br/>　　然而，
<br/>　　等唐威一开口，杨之凡顿时愣住了，
<br/>　　“我呸！老家伙，就凭你这点功夫，也想让小爷拜你为师，你做梦去吧！”
<br/>　　唐威不但拒绝了华云生的收徒，竟然还对他破口大骂！
<br/>　　“哈哈哈！这个傻子，他竟然拒绝了？真是连老天都在帮我啊！”
<br/>　　后方轮椅上的杨之凡，顿时激动的差点笑出声来，心里欢喜的大喊大叫。
<br/>　　前一刻，他本以为报仇无望了，却怎么也没有想到，唐威居然拒绝了华云生的收徒，而且还把他痛骂了一顿。
<br/>　　这家伙简直就是在作死啊！
<br/>　　果不其然，
<br/>　　华云生听到这话，原本老脸上还洋溢着的笑容，瞬间消失的无影无踪，取而代之的是一脸寒霜，
<br/>　　“好小子！老夫念你是个人才，所以才给你这么一个机会，没想到你竟然敢如此践踏老夫的好意，既然如此，那你就给我死去吧！”
<br/>　　声音落下，华云生猛然抬起手臂，再次狠狠拍落下来。
<br/>　　轰隆隆！
<br/>　　强大的掌风气浪，瞬间将教室周围的门窗、玻璃、桌椅，全部冲的粉碎。
<br/>　　然而，就在华云生这强势一掌落下的同时，忽然一道冷漠的声音，陡然在教室里响起，
<br/>　　“想动我兄弟，你问过我了吗？”
<br/>　　轰隆！
<br/>　　声音落下的同时，那华云生猛然感觉到一股浩瀚的气息，朝他铺天盖地袭来！
<br/>　　气息霸道无比！
<br/>　　“什么！还有高手？”
<br/>　　原来，华云生先入为主，误以为唐威，便是将杨之凡打伤之人。
<br/>　　而且以唐威的年纪，能够具备接近化劲宗师的实力，已经算得上是武道天才。
<br/>　　所以他便临时动了收徒的念头。
<br/>　　可是他万万没有想到，在这小小的天海大学之中，竟然还有一位，实力比唐威还有强大许多的人物！
<br/>　　甚至连他，都隐隐感觉到了一丝莫名的压力和紧张！
<br/>　　这么怎么可能？！
