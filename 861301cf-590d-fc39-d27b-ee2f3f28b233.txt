<br/>　　刺啦！

<br/>　　极度类人的外貌下，满是凶残气息，正是蚁王直属三大护卫！

<br/>　　“她的使命是孕育本王，现在，她已经没用了。”初生的蚁王，心中没有丝毫感情。虽然有着人类的外表，却依然奉行着残酷的野兽法则。
<br/>　　族群之中，只能有一个王。有了他，蚁后就已经没用。即便那是孕育他的母亲。依然被决然抛弃，他甚至没有回头看一眼。
<br/>　　……
<br/>　　距离嵌合蚁巢穴数十千米外，密林之内一片寂静。

<br/>　　手指朝着四周随意划动，竟瞬间就杀死近百嵌合蚁！诺布把自己带入嵌合蚁的角色，却绝望的发现，他也躲不开致命攻击。他甚至连打开四次元公寓的机会都没有！
<br/>　　长得那么年轻，实力却如此恐怖，这是哪里来的妖孽！
<br/>　　嘟嘟嘟……
<br/>　　口袋中一阵震动，诺布被吓的一颤，好半晌才回过神来，这是自己的手机在震动。
<br/>　　“诺布，快回来，情况有些变化，需要会长出面做决断。”莫老五沉稳的声音透过电话传来。
<br/>　　“嗯，我知道了。”诺布含糊的应了一声，赶紧挂断电话，双眼再次紧盯苏重。心里反而松了一口气。
<br/>　　莫老五的电话给了他选择，他可以直接离开，而不用去冒险试探眼前这个不知深浅的家伙了。
<br/>　　“我要离开了，你要不要一起。”诺布试着开口道。
<br/>　　他不怕对方一起走，反而有些期待。他的空间里，可装着一个尼特罗呢。
<br/>　　诺布对尼特罗的实力很有信心，只要进了他的空间，对方敢闹幺蛾子，他就敢关门放会长！
<br/>　　苏重身影一闪，直接消失在密林深处，他可不想参与猎人协会的行动。他是来搞破坏的。身边跟着人，他嫌碍手。
<br/>　　“只要沿着嵌合蚁来的方向摸过去，肯定能到达嵌合蚁巢穴中心。”
<br/>　　轰的一声，苏重身周气魄爆发，在密林中横冲直撞。沿途所过之处，所有被碰到的物体，纷纷被侵蚀融化成灰。
<br/>　　一个圆形通道直直贯穿森林，黑色灰烬铺满地面，石头熔化的熔岩随处可见，简直像火山横流，黑烟熊熊！
<br/>　　噗噗噗！

<br/>　　砰！
